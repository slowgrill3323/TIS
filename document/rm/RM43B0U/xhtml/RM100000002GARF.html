Title: THEFT DETERRENT / KEYLESS ENTRY: SMART KEY SYSTEM (for Start Function, Gasoline Model): B27A613; Rear Floor Electrical Key Oscillator Circuit Open; 2024 - 2026 MY Tacoma [12/2023 -        ]

DTC

B27A613

Rear Floor Electrical Key Oscillator Circuit Open

DESCRIPTION

The certification ECU (smart key ECU assembly) generates a request signal and transmits the signal to the No. 2 indoor electrical key antenna assembly (for console box rear side). For the No. 2 indoor electrical key antenna assembly (for console box rear side) to detect when the electrical key transmitter sub-assembly is in the cabin, the signal from the certification ECU (smart key ECU assembly) requesting a response from the electrical key transmitter sub-assembly is transmitted inside the vehicle. DTC B27A613 is stored by the certification ECU (smart key ECU assembly) when an open is detected between the certification ECU (smart key ECU assembly) and No. 2 indoor electrical key antenna assembly (for console box rear side) (between terminals CLG6 and CLG, or terminals CG6B and CLGB).
