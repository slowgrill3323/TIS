Title: LIGHTING (INT): LIGHTING SYSTEM: Door Unlock Detection Switch Circuit; 2024 MY Tacoma Tacoma HV [12/2023 - 03/2024]

Door Unlock Detection Switch Circuit

DESCRIPTION

The main body ECU (multiplex network body ECU) detects the condition of each door unlock detection switch.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
