Stop and Start Cancel Switch Circuit

DESCRIPTION

Stop and start control can be disabled by pressing the stop and start system cancel switch (warning canceling switch assembly). The stop and start system cancel switch (warning canceling switch assembly) is a momentary-type switch that switches between on and off when the switch is pressed. While stop and start control is disabled, the stop and start cancel indicator is illuminated to inform the driver.
