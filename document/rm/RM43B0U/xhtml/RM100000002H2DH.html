DTC

B151187

Lost Communication with Meter Gigabit Video Interface Missing Message

DESCRIPTION

This DTC is stored when a GVIF communication malfunction occurs between the combination meter assembly and meter mirror sub-assembly.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

DTC Output from

Priority

B151187

Lost Communication with Meter Gigabit Video Interface Missing Message

Diagnosis Condition:

  • Ignition switch ON
  • IG power source voltage is 9.5 V or more

Malfunction Status:

  • No communication with meter mirror sub-assembly
  • Malfunction signal output to meter mirror sub-assembly

    (meter mirror sub-assembly receives the malfunction signal and notifies the combination meter assembly)

Malfunction Time:

  • 5.0 seconds or more
  • Meter mirror sub-assembly
  • Harness or connector
  • Combination meter assembly

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • When replacing the combination meter assembly, always replace it with a new one. If a combination meter assembly which was installed to another vehicle is used, the information stored in it will not match the information from the vehicle and a DTC may be stored.
  • When replacing the combination meter assembly, update the ECU security key.
