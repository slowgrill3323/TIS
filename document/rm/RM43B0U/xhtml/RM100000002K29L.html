A/T CODE REGISTRATION

NOTICE:

When the TCM is replaced, perform "A/T Code Registration".

HINT:

When the TCM (when the transmission compensation code can be read) is replaced, perform A/T Code Registration (Read Compensation Code) and perform K0 Compensation Code Registration (Read Compensation Code) before replacement.

After the TCM is replaced, perform A/T Code Registration (Set Compensation Code) and K0 Compensation Code Registration (Write Compensation Code) or K0 Compensation Code Restoration after performing K0 Compensation Code Reset.

(a) Make sure the shift lever is in P.

(b) Turn the ignition switch to ON.

NOTICE:

Do not turn the ignition switch to ON (READY).

(c) Enter the following menus: Powertrain / Transmission / Utility / A/T Code Registration.

Powertrain > Transmission > Utility

Tester Display

A/T Code Registration

(d) Select "Read Compensation Code".

(e) Perform A/T Code Registration (Read Compensation Code) according to the screen display.

NOTICE:

If the compensation code cannot be read, perform A/T Code Registration (Arbitrary Code).

(f) Turn the ignition switch off.

(g) Replace the TCM.

(h) Turn the ignition switch to ON.

NOTICE:

Do not turn the ignition switch to ON (READY).

(i) Enter the following menus: Powertrain / Transmission / Utility / A/T Code Registration.

Powertrain > Transmission > Utility

Tester Display

A/T Code Registration

(j) Select "Set Compensation Code".

(k) Perform A/T Code Registration (Set Compensation Code) according to the screen display.

(l) Turn the ignition switch off.

PERFORM ROAD TEST TO ALLOW TCM TO LEARN

NOTICE:

  • Do not accelerate or decelerate rapidly.
  • This test must be performed after checking and adjusting the engine.
  • Perform drive learning while the ATF temperature is within a range of 60 to 100°C (140 to 212°F).

(a) The Road Test procedure is as follows:

(1) According to the replaced parts, perform the appropriate road test shown below.

Replaced Parts

Road Test
  • Hybrid vehicle transmission assembly
    • Automatic transmission assembly
    • Torque converter assembly
    • Hybrid module assembly
    • Oil pump with motor assembly
  • Automatic transmission assembly and torque converter assembly
    • Automatic transmission assembly
    • Torque converter assembly
    • Oil pump with motor assembly
  • Automatic transmission assembly
    • Automatic transmission assembly
    • Oil pump with motor assembly
  • Torque converter assembly
  • Solenoid (SL1) valve
  • Solenoid (SL2) valve
  • Solenoid (SL3) valve
  • Solenoid (SL4) valve
  • Solenoid (SL5) valve
  • Solenoid (SL6) valve
  • Solenoid (SL) valve
  • Solenoid (SLT) valve
  • Solenoid (SLU) valve
  • Solenoid (SC3) valve
  • Transmission valve body assembly (for transmission side)
  • TCM
  • Hybrid module assembly
  • Transmission valve body assembly (for motor side)
  • Engine assembly

Perform Pattern 1 and Pattern 2.

Pre-work:

  • According to the display on the GTS, read the Data List.: Powertrain / Engine / Data List.

    With the shift position in P and the vehicle stopped, check the accelerator pedal depressed amount that results in a Data List item "Accelerator Position" between 15 and 20%, between 25 and 35%, and 10%.

Pattern 1:

  1. Pattern 1-1:

    With the shift lever in D, accelerate to approximately 60 km/h (37.28 mph) while "Accelerator Position" is between 15 and 20%, decelerate with the brake pedal released until the gear shifts down to 3rd gear, and then depress the brake pedal to stop the vehicle.

    If the shift shock is large, repeat this procedure until shift shock no longer occurs.

  2. Pattern 1-2:

    With the shift lever in D, accelerate to approximately 60 km/h (37.28 mph) while "Accelerator Position" is between 25 and 35%, release the accelerator pedal and check that the engine stops, and then depress the brake pedal to decelerate until the vehicle stops.

    If the shift shock is large, repeat this procedure until shift shock no longer occurs.

    HINT:

    • If the engine does not stop even after releasing the accelerator pedal, perform one of the following before performing Pattern 1-2.
      • If HV battery temperature is 45°C (113°F) or higher:

        Wait with the ignition switch ON (READY) and the vehicle stopped until the HV battery temperature drops.

      • If HV battery temperature is below 45°C (113°F):

        After accelerating as much as possible with the shift position in D, repeat coasting with the accelerator and brake pedals released until the engine stops.

    • When decelerating, perform with 2 patterns of deceleration speed: 0.7 m/s2 and 1.0 m/s2.
  3. Pattern 1-3:

    With the shift lever in D, accelerate to approximately 80 km/h (49.71 mph), and then decelerate with the brake pedal released to 60 km/h (37.28 mph).

    If the shift shock is large, repeat this procedure until shift shock no longer occurs.

Pattern 2:

  • Select the S2 range with the shift lever in S, and at a vehicle speed of approximately 5 km/h (3.11 mph) with the vehicle in 2nd gear, accelerate for 20 seconds or more by depressing the accelerator pedal so that the "Accelerator Position" changes from 0% to approximately 10% in approximately 3 seconds.

    Perform this drive learning 5 times.

(b) Repeat Road Test Pattern 1 until shift shock and engine revving subside.

NOTICE:

When repeating drive learning, always begin a new run with the vehicle stopped.

NOTICE:

  • Perform K0 Learning Promotion after replacing the hybrid vehicle transmission assembly (automatic transmission assembly, torque converter assembly, hybrid module assembly and oil pump with motor assembly), TCM, hybrid module assembly, transmission valve body assembly (for motor side) or engine assembly.
  • Perform K0 Learning Promotion while the ATF temperature is within a range of 85 to 100°C (185 to 212°F).
