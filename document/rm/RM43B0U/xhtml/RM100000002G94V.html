PROCEDURE

1. INSTALL OUTER REAR VIEW MIRROR ASSEMBLY LH

*A

w/ Panoramic View Monitor System

-

-

Install in this Direction

-

-

(1) Attach the claws to set the outer rear view mirror assembly LH.

NOTICE:

Hold the outer rear view mirror assembly LH with one hand so that it does not drop.

(2) Install the outer rear view mirror assembly LH with the 3 nuts in the order shown in the illustration.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(3) Connect the connector and attach the clamp.

(4) w/ Panoramic View Monitor System:

Connect the television camera connector.

3. INSTALL FRONT DOOR TRIM BOARD SUB-ASSEMBLY LH
