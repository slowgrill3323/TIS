DTC

C105F62

IGR Voltage / Actuator Supply Voltage Signal Compare Failure

DESCRIPTION

If a malfunction is detected in the power supply circuit, the trailer brake control ECU (brake control with bracket relay) stores this DTC.

With the ignition switch ON, the difference between the IGR terminal voltage and the +B terminal voltage continuously exceeds the threshold for 1 second or more
