PROCEDURE

1. INSTALL WINDSHIELD WIPER MOTOR ASSEMBLY

*a

13.8 to 17.8°

*b

Wiper Motor Center Line

Rotate in this Direction

-

-

(1) Install the windshield wiper motor assembly.

1. Using a T30 "TORX" socket wrench, install the windshield wiper motor assembly to the wiper link assembly with the "TORX" screw.

Torque:

13 N·m {133 kgf·cm, 10 ft·lbf}

2. Move the lever as shown in the illustration.

3. Using a T30 "TORX" socket wrench, install the windshield wiper motor assembly to the wiper link assembly with the 3 "TORX" screws.

Torque:

13 N·m {133 kgf·cm, 10 ft·lbf}

4. Rotate the lever 13.8 to 17.8° in the clockwise direction from the automatic stop position.

Install in this Direction

Apply MP Grease

(1) Connect the No. 1 wiper link rod.

1. Apply MP grease to the wiper motor pivot.

2. Connect the No. 1 wiper link rod to the wiper motor pivot.

2. INSTALL WINDSHIELD WIPER MOTOR AND LINK ASSEMBLY

Install in this Direction

-

-

(1) Install the windshield wiper motor and link assembly with the 3 bolts in the order shown in the illustration.

NOTICE:

Do not damage the vehicle with the front wiper motor and link.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

(2) Connect the connector.

3. INSTALL COWL TOP VENTILATOR LOUVER SUB-ASSEMBLY

NOTICE:

To prevent damage to the windshield glass, remove any foreign matter (sand, dust, etc.) from around the contacting surfaces of the cowl top ventilator louver sub-assembly and windshield glass.

4. INSTALL ROOF DRIP SIDE FINISH MOULDING SUB-ASSEMBLY RH

5. INSTALL ROOF DRIP SIDE FINISH MOULDING SUB-ASSEMBLY LH

6. INSTALL FRONT WIPER ARM AND BLADE ASSEMBLY RH

*a

Serration

*b

Wire Brush

*c

Ceramic Dot

*d

7.5 mm (0.295 in.)

(1) When reusing the front wiper arm and blade assembly RH:

1. Clean the wiper arm serrations to remove any burrs, dirt, etc.

NOTICE:

Do not grind down the wiper arm serrations.

(2) Set the automatic stop position by the below procedure.

1. Turn the ignition switch to ON.

2. Operate the windshield wipers while spraying washer fluid onto the windshield glass. Make sure that the windshield wipers function properly and the wipers do not contact the vehicle body.

3. Turn the ignition switch off.

(3) Temporarily install the front wiper arm and blade assembly RH with the nut to the position shown in the illustration.

(4) Install the front wiper arm and blade assembly RH with the nut to the position shown in the illustration.

Torque:

29 N·m {296 kgf·cm, 21 ft·lbf}

HINT:

Hold the arm hinge by hand when tightening the nut.
