DTC

B235787

Wiper Module Missing Message

DESCRIPTION

The main body ECU (multiplex network body ECU) and windshield wiper relay assembly communicate via CXPI communication. The main body ECU (multiplex network body ECU) stores this DTC if communication becomes abnormal.

Detection condition:

Malfunction Status:

  • Communication malfunction between the main body ECU (multiplex network body ECU) and semiconductor power integration ECU

Malfunction Time:

  • 10 seconds or more
  • Semiconductor power integration ECU
  • Main body ECU (multiplex network body ECU)
  • Harness or connector
  • Windshield washer relay (WIPER)
