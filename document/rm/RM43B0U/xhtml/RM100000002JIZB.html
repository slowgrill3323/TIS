DTC

P050031

Vehicle Speed Sensor "A" No Signal

DESCRIPTION

The skid control ECU (brake actuator assembly) converts the vehicle speed pulse signals output from the speed sensors installed to each wheel into vehicle speed signals and inputs these signals to the TCM.

DTC No.

Detection Item

DTC Detection Condition

Trouble Area

Note

P050031

Vehicle Speed Sensor "A" No Signal

Vehicle speed signal is not input from the skid control ECU for 5 seconds or more while driving with SP2 vehicle speed at 10 km/h (6.22 mph) or higher, NT speed at 1200 rpm or higher and shift position in D or R (gear change not in progress) (1-trip detection logic).
  • CAN communication system
  • Skid control ECU (Brake actuator assembly)

Comes on

Transmission

B

SAE: P0500

MONITOR DESCRIPTION

If there is no speed signal from the skid control ECU even though the TCM determines that the vehicle is being driven, the TCM interprets this as a malfunction in the speed signal circuit. The TCM then illuminates the MIL and stores this DTC.

    Click here

    HINT:

    [*1]: Normal judgment procedure.

  4. Enter the following menus: Powertrain / Transmission / Utility / All Readiness.
  5. Input the DTC: P050031.
  6. Check the DTC judgment result.

    GTS Display

    Description

    NORMAL
    • DTC judgment completed
    • System normal

    ABNORMAL
    • DTC judgment completed
    • System abnormal

    INCOMPLETE
    • DTC judgment not completed
    • Perform driving pattern after confirming DTC enabling conditions

    N/A
    • Unable to perform DTC judgment
    • Number of DTCs which do not fulfill DTC preconditions has reached ECU memory limit

    HINT:

    • If the judgment result shows NORMAL, the system is normal.
    • If the judgment result shows ABNORMAL, the system has a malfunction.
    • If the judgment result shows INCOMPLETE or N/A, perform the normal judgment procedure again.
