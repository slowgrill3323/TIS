PROCEDURE

PROCEDURE

1. INSTALL CLUTCH MASTER CYLINDER PUSH ROD CLEVIS BUSH

MP Grease

-

-

(1) Apply MP grease to the inside of clutch master cylinder push rod clevis bush.

(2) Install the clutch master cylinder push rod clevis bush to the clutch pedal sub-assembly.

2. INSTALL NO. 1 CLUTCH PEDAL CUSHION

3. INSTALL NO. 2 CLUTCH PEDAL CUSHION

4. INSTALL CLUTCH PEDAL TURNOVER BUSH

MP Grease

-

-

(1) Apply MP grease to the clutch pedal turnover bush.

(2) Install the clutch pedal turnover bush to the clutch pedal sub-assembly.

5. INSTALL CLUTCH PEDAL SHAFT COLLAR

6. INSTALL CLUTCH PEDAL BUSH

MP Grease

-

-

(1) Apply MP grease to the inner and outer surfaces of the 2 clutch pedal bushes.

(2) Install the 2 clutch pedal bushes to the clutch pedal sub-assembly.

7. INSTALL CLUTCH PEDAL PAD

8. INSTALL HEXAGON BOLT

Torque:

15.7 N·m {160 kgf·cm, 12 ft·lbf}

9. INSTALL CLUTCH PEDAL SPRING HOLDER

MP Grease

-

-

(1) Apply MP grease to the contact surface of the clutch pedal spring holder and clutch pedal mounting bracket sub-assembly.

(2) Install the clutch pedal spring holder to the clutch pedal sub-assembly.

10. INSTALL CLUTCH PEDAL SUB-ASSEMBLY

Torque:

34.3 N·m {350 kgf·cm, 25 ft·lbf}

11. INSTALL CLUTCH PEDAL SPRING

MP Grease

Install in this Direction

(1) Apply MP grease to the contact surfaces of the clutch pedal sub-assembly and the clutch pedal spring.

(2) Install the clutch pedal spring to the clutch pedal sub-assembly and the clutch pedal spring holder.

12. INSTALL CLUTCH PEDAL SUPPORT WAY

Torque:

4.5 N·m {46 kgf·cm, 40 in·lbf}

13. INSTALL CLUTCH PEDAL SUPPORT GROMMET

14. INSTALL NO. 1 CLUTCH RESERVOIR HOSE

15. INSTALL CLUTCH MASTER CYLINDER TO 2 WAY TUBE

*a

Torque Wrench Fulcrum Length

-

-

(1) Using a union nut wrench, tighten the clutch master cylinder to 2 way tube.

Torque:

Specified tightening torque :

15.2 N·m {155 kgf·cm, 11 ft·lbf}
