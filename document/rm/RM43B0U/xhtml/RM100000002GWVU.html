FREEZE FRAME DATA

FREEZE FRAME DATA

(a) When a DTC is stored, the trailer brake control ECU (brake control with bracket relay) stores the current vehicle state as Freeze Frame Data.

HINT:

Freeze Frame Data at the time a DTC is stored:

When the trailer brake control ECU (brake control with bracket relay) stores data at the time a DTC is stored, no updates will be performed until the data is cleared.

CHECK FREEZE FRAME DATA

(a) Read the Freeze Frame Data recorded when the DTCs were stored.
