15. INSTALL SEPARATE TYPE FRONT SEAT CUSHION COVER

HINT:

When installing a separate type front seat cushion cover, refer to Precaution in order to prevent wrinkles from forming.

*a

Hog Ring Pliers

*b

Band

Install in this Direction

-

-

(1) Using hog ring pliers, install the separate type front seat cushion cover with 14 new hog rings in the order shown in the illustration.

NOTICE:

  • Be careful not to dirty or damage the separate type front seat cushion cover and separate type front seat cushion pad.
  • Be sure to securely install the hog rings as shown in the illustration while preventing wrinkles from forming on the separate type front seat cushion cover.

(2) Attach the band as shown in the illustration.

22. INSTALL SEPARATE TYPE FRONT SEATBACK COVER

HINT:

When installing a separate type front seatback cover, refer to Precaution in order to prevent wrinkles from forming.

*a

Hog Ring Pliers

-

-

(1) Using hog ring pliers, install the separate type front seatback cover with 13 new hog rings in the order shown in the illustration.

NOTICE:

  • Be careful not to dirty or damage the separate type front seatback cover and separate type front seatback pad.
  • Be sure to securely install the hog rings as shown in the illustration while preventing wrinkles from forming on the separate type front seatback cover.

(2) Pass the wire harness of the front seatback heater assembly LH through the slit of the separate type front seatback cover.
