PROCEDURE

1. INSTALL INSTRUMENT PANEL PASSENGER W/O DOOR AIRBAG ASSEMBLY

(1) Engage the 3 hooks.

(2) Push in the instrument panel passenger w/o door airbag assembly to engage the 3 hooks.

(3) Install the 2 screws.

2. INSTALL NO. 3 INSTRUMENT PANEL WIRE

(1) Connect the 3 airbag connectors to install the No. 3 instrument panel wire.

NOTICE:

  • When connecting any airbag connector, take care not to damage the airbag wire harness.
  • Make sure the lock button is securely locked.
  • Be sure to connect the connectors to each corresponding color.
