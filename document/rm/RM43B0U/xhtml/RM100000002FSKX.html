PROCEDURE

PROCEDURE

1. INSTALL CLUTCH START SWITCH ASSEMBLY

(1) Using a 14 mm deep socket wrench, install the clutch start switch assembly.

NOTICE:

To prevent damage to the clutch start switch assembly, when removing the clutch start switch assembly, hold it so it does not rotate as you remove it.

Torque:

15.7 N·m {160 kgf·cm, 12 ft·lbf}

(2) Connect the connector.
