PROCEDURE

PROCEDURE

1. INSTALL REAR BODY TAIL GATE SUB-ASSEMBLY

*a

Approximately 15 degrees

*b

45 degrees

Install in this Direction (1)

Install in this Direction (2)

(1) Hold the tail gate at approximately 15 degrees from fully closed.

(2) Move the rear body tail gate sub-assembly in the direction of arrow 1 in the illustration to connect the tail gate hinge assembly LH.

(3) Rotate the tail gate open to 45degrees.

(4) Move the rear body tail gate sub-assembly in the direction of arrow 2 in the illustration to connect the tail gate hinge assembly RH and install rear body tail gate sub-assembly.

(1) Engage the claws and connect the connectors.

2. INSTALL TAIL GATE STAY ASSEMBLY LH

(1) Using a T40 "TORX" socket wrench, install the tail gate stay assembly LH to the body.

Torque:

26 N·m {265 kgf·cm, 19 ft·lbf}

3. INSTALL TAIL GATE STAY ASSEMBLY RH

(a) Use the same procedure described for the LH side.
