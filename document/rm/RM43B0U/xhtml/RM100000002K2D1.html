PROCEDURE

PROCEDURE

1. INSTALL TRANSMISSION CASE GASKET

(1) Coat 3 new transmission case gaskets and 3 new brake drum gaskets with ATF, and install them to the automatic transmission case sub-assembly.

2. INSTALL TRANSMISSION VALVE BODY ASSEMBLY

*1

Manual Valve Lever Sub-assembly

*2

Manual Valve

*3

Manual Detent Spring Cover

*4

Manual Detent Spring Sub-assembly

(1) Insert the manual valve lever sub-assembly into the groove on the end of the manual valve and install the transmission valve body assembly to the automatic transmission case sub-assembly with the 10 bolts.

Torque:

11 N·m {112 kgf·cm, 8 ft·lbf}

HINT:

Bolt length

  • Bolt A: 25 mm (0.984 in.)
  • Bolt B: 50 mm (1.97 in.)
  • Bolt C: 57 mm (2.24 in.)

(2) Install the detent spring sub-assembly and detent spring cover to the transmission valve body assembly with the bolt.

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}

3. INSTALL OIL PUMP WITH MOTOR ASSEMBLY

(1) Coat a new O-ring with ATF and install it to the oil pump with motor assembly.

NOTICE:

Be careful not to damage or twist the O-ring.

(2) Install the oil pump with motor assembly to the automatic transmission case sub-assembly with the 3 bolts.

Torque:

13.5 N·m {138 kgf·cm, 10 ft·lbf}

4. INSTALL OIL STRAINER SPACER

Torque:

9.5 N·m {97 kgf·cm, 84 in·lbf}

5. INSTALL TRANSMISSION OIL PIPE

(1) Coat a 2 new O-ring with ATF and install it to the automatic transmission case sub-assembly.

(2) Install the transmission oil pipe to the automatic transmission case sub-assembly with the 2 bolts.

Torque:

9.5 N·m {97 kgf·cm, 84 in·lbf}

6. INSTALL NO. 1 TRANSMISSION CASE PLATE

Torque:

9.8 N·m {100 kgf·cm, 87 in·lbf}

7. INSTALL TRANSMISSION WIRE
