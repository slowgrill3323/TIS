(1) Clamp the cooler compressor assembly in a vise between aluminum plates.

(1) Using SST, install the magnet clutch rotor and a new snap ring with the chamfered side facing outward.

09904-00011

NOTICE:

  • Do not damage the seal cover of the bearing when installing the snap ring.
  • When installing the snap ring, do not widen it excessively.

3. INSTALL MAGNET CLUTCH HUB

(1) Install the magnet clutch hub using the following procedure.

1. Install the magnet clutch washer(s) and magnet clutch hub.

NOTICE:

Do not change the combination of the magnet clutch washer(s) used before disassembly.

2. Using SST, hold the magnet clutch hub and install the bolt.

09985-00270

Torque:

18 N·m {184 kgf·cm, 13 ft·lbf}

NOTICE:

Make sure that there is no foreign matter or oil on the compressor shaft, bolt and magnet clutch hub.

4. INSPECT MAGNET CLUTCH CLEARANCE

(1) Inspect the magnet clutch clearance using the following procedure.

1. Set a dial indicator to the magnet clutch hub.

2. Connect an auxiliary battery positive lead to terminal 3 (MG+) of the magnet clutch connector and a negative lead to the ground wire. Turn the magnet clutch on and off and measure the clearance.

Standard Clearance:

0.30 to 0.55 mm (0.0119 to 0.0216 in.)

NOTICE:

Adjustment should be performed with 3 or fewer magnet clutch washers.

HINT:

If the measured value is not within the standard clearance, remove the magnet clutch hub and adjust the clearance using magnet clutch washers to obtain the standard clearance.

3. Remove the compressor with magnet clutch from the vise.
