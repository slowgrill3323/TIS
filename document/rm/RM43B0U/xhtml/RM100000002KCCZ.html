PROCEDURE

PROCEDURE

1. PRECAUTION

CAUTION:

  • Wear protective gloves. Sharp areas on the parts may injure your hands.
  • There is risk of injury.

2. INSTALL FRONT SEAT AIRBAG ASSEMBLY LH

(1) Install the front seat airbag assembly LH with 2 new nuts.

Torque:

5.5 N·m {56 kgf·cm, 49 in·lbf}

CAUTION:

  • Do not reuse the nut.
  • If the installation surface or surrounding area of the front seat airbag assembly LH is deformed or damaged, replace the front seatback frame sub-assembly LH, separate type front seatback pad and separate type front seatback cover.

(2) Engage the clamp.

3. INSTALL SEATBACK COVER WITH PAD

4. INSTALL FRONT SEATBACK ASSEMBLY LH

Torque:

45 N·m {459 kgf·cm, 33 ft·lbf}

5. INSTALL SEAT RECLINING ADJUSTING CABLE ASSEMBLY LH (for Walk in Seat Type)

6. INSTALL FRONT SEAT INNER CUSHION SHIELD RH (for Fold Down Seat Type)

7. INSTALL FRONT SEATBACK BOARD SUB-ASSEMBLY RH (for Fold Down Seat Type)

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

8. INSTALL FRONT SEAT INNER CUSHION SHIELD LH (except Fold Down Seat Type)

9. INSTALL FRONT SEAT CUSHION SHIELD LH

10. INSTALL FRONT SEAT VERTICAL ADJUSTER HANDLE LH

11. INSTALL RECLINING ADJUSTER RELEASE HANDLE LH
