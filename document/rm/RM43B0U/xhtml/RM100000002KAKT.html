PROCEDURE

1. INSTALL NO. 1 TRACTION BATTERY DEVICE BOX ASSEMBLY

CAUTION:

Wear insulated gloves.

(1) Install the No. 1 traction battery device box assembly with the 4 nuts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

NOTICE:

  • Do not drop the No. 1 traction battery device box assembly, strike it with tools or subject it to impact.
  • If the No. 1 traction battery device box assembly is subjected to an impact, replace it with a new one.

(2) Connect the 3 connectors.

NOTICE:

Make sure that the connector is connected securely.

CAUTION:

Wear insulated gloves.

8. PERFORM CURRENT SENSOR OFFSET LEARNING

HINT:

Perform this procedure when the No. 1 traction battery device box assembly has been replaced.

(a) Perform a road test.

NOTICE:

Accelerate and decelerate gently. Avoid rapid acceleration and deceleration.

(1) Drive the vehicle with the value of Data List item "Hybrid/EV Battery Current" between -50 A and 50 A.

Powertrain > Hybrid Control > Data List

Tester Display

Hybrid/EV Battery Current

HINT:

Distance and driving time are not specified.

(2) Turn the ignition switch off and leave the vehicle for 30 seconds or more.

(3) Check that the value of "Hybrid/EV Battery Current" is between -0.5 A and 0.5 A with the ignition switch on (IG).

Powertrain > Hybrid Control > Data List

Tester Display

Hybrid/EV Battery Current

NOTICE:

If the value is outside the specified range, perform the road test again.

HINT:

  • When the ignition switch is on (IG), if value of "Hybrid Battery Current" is between -0.5 A and 0.5 A, current sensor offset learning has been completed.
  • Even if the current sensor offset learning is not complete, the current sensor value will be corrected by repeating the road test a maximum of 7 times.
