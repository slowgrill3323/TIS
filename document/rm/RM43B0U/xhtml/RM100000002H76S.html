1. INSTALL FRONT UPPER BALL JOINT DUST COVER

*a

MP grease

-

-

(1) Pack the front upper arm ball joint with MP grease.

Grease capacity:

4.0 g (0.141 oz.)

(2) Apply MP grease to the locations shown in the illustration.

NOTICE:

Do not apply MP grease to the tapered or threaded parts of the ball joint.

(3) Install a new front upper ball joint dust cover to the front upper suspension arm assembly.

(4) Using snap ring expander, install a new dust cover set ring.

NOTICE:

Make sure that the set ring is securely installed in the groove.

2. INSTALL FRONT UPPER SUSPENSION ARM BUSHING

(1) Using SST and a press, press in a new front upper suspension arm bushing.
