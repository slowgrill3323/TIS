PROCEDURE

1. INSTALL KNOCK CONTROL SENSOR

*a

Top

*b

Right of Vehicle

*c

15°

*d

5°

(1) Install the knock control sensor to the cylinder block sub-assembly with the bolt so that the knock control sensor installation position is as shown in the illustration.

Torque:

20 N·m {204 kgf·cm, 15 ft·lbf}

NOTICE:

  • If the knock control sensor has been struck or dropped, replace it.
  • Make sure that the knock control sensor is in the correct position.

(2) Connect the knock control sensor connector.

2. CONNECT WIRE HARNESS

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}
