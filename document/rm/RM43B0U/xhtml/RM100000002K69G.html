UTILITY

ALL READINESS

HINT:

  • With "All Readiness", you can check whether or not the DTC judgment has been completed by using the GTS.
  • You should check "All Readiness" after simulating malfunction symptoms or for validation after finishing repairs.

(a) Connect the GTS to the DLC3.

(b) Turn the ignition switch to ON.

(c) Turn the GTS on.

(d) Clear the DTCs even if no DTCs are stored.

Powertrain > Hybrid Control > Clear DTCs

(e) Turn the ignition switch off and wait for 2 minutes or more.

(f) Turn the ignition switch to ON.

(g) Perform the DTC judgment driving pattern to run the DTC judgment.

(h) Enter the following menus: Powertrain / Hybrid Control / Utility / All Readiness.

Powertrain > Hybrid Control > Utility

Tester Display

All Readiness

(i) Input the DTC to be confirmed.

(j) Check the DTC judgment result.

GTS Display

Description
If the judgment result shows INCOMPLETE, perform the DTC confirmation driving pattern again.

NORMAL
  • DTC judgment completed
  • System normal

ABNORMAL
  • DTC judgment completed
  • System abnormal

INCOMPLETE
  • DTC judgment not completed
  • Perform the driving pattern

(k) Turn the ignition switch off.

DIAGNOSIS RELATED INFORMATION

(a) Check for diagnosis related information.

(1) Connect the GTS to the DLC3.

(2) Turn the ignition switch to ON.

(3) Turn the GTS on.

(4) Enter the following menus: Powertrain / Hybrid Control / Utility / Diagnosis Related Information.

Powertrain > Hybrid Control > Utility

Tester Display

Diagnosis Related Information

(5) Check the Diagnosis Related Information, and then write them down.

(b) Clear diagnosis related information.

NOTICE:

Clearing the DTCs will also clear the diagnosis related information.

(1) Connect the GTS to the DLC3.

(2) Turn the ignition switch to ON.

(3) Turn the GTS on.

(4) Enter the following menus: Powertrain / Hybrid Control / Trouble Codes.

(5) Clear the diagnosis related information and freeze frame data.

(6) Turn the ignition switch off.

BATTERY STATUS INFO UPDATE

HINT:

If the HV battery has been replaced, make sure to perform "Battery Status Info Update".

It is necessary to update the HV battery status information stored in the hybrid vehicle control ECU after the HV battery replacement.

(a) Connect the GTS to the DLC3.

(b) Turn the ignition switch to ON.

(c) Turn the GTS on.

(d) Enter the following menus: Powertrain / Hybrid Control / Utility / Battery Status Info Update.

Powertrain > Hybrid Control > Utility

Tester Display

Battery Status Info Update

(e) Turn the ignition switch off.

HIGH VOLTAGE FUSE ACCUMULATED LOAD HISTORY RESET

HINT:

  • If the hybrid battery terminal block has been replaced, make sure to perform "High Voltage Fuse Accumulated Load History Reset" and then clear the DTCs.
  • When "High Voltage Fuse Accumulated Load History Reset" is performed, the high voltage fuse accumulated load history stored in the hybrid vehicle control ECU will be initialized.
  • If "High Voltage Fuse Accumulated Load History Reset" is not performed after replacing the hybrid battery terminal block when troubleshooting DTC P0A9563, the same DTC will be output again.

(a) Connect the GTS to the DLC3.

(b) Turn the ignition switch to ON.

(c) Turn the GTS on.

(d) Enter the following menus: Powertrain / Hybrid Control / Utility / High Voltage Fuse Accumulated Load History Reset.

Powertrain > Hybrid Control > Utility

Tester Display

High Voltage Fuse Accumulated Load History Reset

(e) According to the display on the GTS, perform High Voltage Fuse Accumulated Load History Reset and check that "High Voltage Fuse Accumulated Load History Reset is complete." is displayed.

HINT:

If "High Voltage Fuse Accumulated Load History Reset has failed. Do you want to try again?" is displayed on the GTS, perform the procedure again.

(f) Turn the ignition switch off.
