PROCEDURE

PROCEDURE

1. INSTALL INTAKE CAMSHAFT TIMING GEAR BOLT ASSEMBLY

(1) Temporarily install the intake camshaft timing gear bolt assembly to the camshaft timing gear assembly.

NOTICE:

  • If the intake camshaft timing gear bolt assembly has been struck or dropped, replace it.
  • Make sure to fit the protrusion of the intake camshaft timing gear bolt assembly into the cutout of the camshaft timing gear assembly.

(2) Using a 5 mm hexagon socket wrench, install the intake camshaft timing gear bolt assembly with the bolt.

Torque:

19 N·m {194 kgf·cm, 14 ft·lbf}

(3) Pull the intake camshaft timing gear bolt assembly to confirm that is installed securely.

2. INSTALL EXHAUST CAMSHAFT TIMING GEAR BOLT ASSEMBLY

(1) Temporarily install the exhaust camshaft timing gear bolt assembly to the camshaft timing exhaust gear assembly.

NOTICE:

  • If the exhaust camshaft timing gear bolt assembly has been struck or dropped, replace it.
  • Make sure to fit the protrusion of the exhaust camshaft timing gear bolt assembly into the cutout of the camshaft timing exhaust gear assembly.

(2) Using a 5 mm hexagon socket wrench, install the exhaust camshaft timing gear bolt assembly with the bolt.

Torque:

19 N·m {194 kgf·cm, 14 ft·lbf}

(3) Pull the exhaust camshaft timing gear bolt assembly to confirm that is installed securely.
