Push Start Switch Illumination Circuit

DESCRIPTION

The illuminated entry system controls the engine switch illumination.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • Check that the "Start Indicator Function" customization setting is set to "ON".
  • Before replacing the certification ECU (smart key ECU assembly) or engine switch, refer to Registration.

    Click here

  • When replacing the engine switch, always replace it with a new one.
