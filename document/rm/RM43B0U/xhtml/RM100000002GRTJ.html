DTC

B232096

RL-Door P/W Motor Component Internal Failure

DESCRIPTION

The main body ECU (multiplex network body ECU) communicates with the power window regulator motor assembly LH (for rear door) via LIN communication.

This DTC is stored when a power window regulator motor assembly LH (for rear door) is malfunctioning, or the ECU built into the power window regulator motor assembly LH (for rear door) determines that the fully closed power window position has deviated approximately 20 mm (0.787 in.) or more from the normal position.
