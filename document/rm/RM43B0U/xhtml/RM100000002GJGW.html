1. INSTALL FRONT PROPELLER SHAFT UNIVERSAL JOINT SPIDER KIT

HINT:

Use the same procedure for both spiders.

*a

Turn

*b

Hold

(1) Install the spider to the flange yoke.

(2) Using SST, install 2 of the spider bearings to the spider.

09332-25010

HINT:

Before installing SST, sufficiently raise the part labeled A. If the part labeled A is too low, it may be difficult to install SST.

(3) Using SST, adjust both spider bearings so that the snap ring grooves are as wide as possible and equal in width.

09332-25010

(1) Install 2 new snap rings to the installed spider bearings. The snap rings must have equal thickness and allow 0 to 0.05 mm (0 to 0.00197 in.) of axial play.

NOTICE:

While the installed snap rings should have equal thickness, if the axial play is greater than 0.05 mm (0.00197 in.), select 2 snap rings that have as close to the same thickness as possible.

HINT:

Do not reuse the snap rings.
