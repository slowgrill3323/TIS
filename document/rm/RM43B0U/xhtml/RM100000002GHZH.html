PROCEDURE

PROCEDURE

1. INSTALL FUEL MAIN VALVE ASSEMBLY (for High Pressure)

(1) Apply gasoline to new O-rings, then install the O-rings to the fuel main valve assembly.

(2) Install the fuel main valve assembly to the fuel filter.

NOTICE:

Make sure that the O-rings are not cut or pinched during installation.

(3) Install the clip to the fuel filter.

2. INSTALL FUEL MAIN VALVE ASSEMBLY (for Low Pressure)

(1) Apply gasoline to new O-rings, then install the O-rings to the fuel main valve assembly.

(2) Install the fuel main valve assembly to the fuel filter.

NOTICE:

Make sure that the O-rings are not cut or pinched during installation.

(3) Install the clip to the fuel filter.
