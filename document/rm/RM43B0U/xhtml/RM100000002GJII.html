PROCEDURE

PROCEDURE

1. INSTALL REAR AXLE HUB BOLT

*a

Washer

*b

Nut

(1) Temporarily install a washer and nut to 6 new rear axle hub bolts as shown in the illustration.

(2) Install the rear axle hub bolts by tightening each nut.

(3) Remove the washer and nut from each hub bolt.

2. INSTALL REAR AXLE HUB AND BEARING ASSEMBLY

(1) Position the parking brake plate on a new rear axle hub and bearing.

NOTICE:

Make sure to securely fit the parking brake plate to the bearing.

(2) Using 2 socket wrenches and a press, press in the 4 parking brake plate to rear axle housing bolt.

3. INSTALL REAR AXLE SHAFT

*a

Tapered Surface

*b

Chamfered Surface

(1) Install the washer and a new rear axle bearing inner retainer to the axle hub as shown in the illustration.

NOTICE:

  • Install the washer with its tapered surface facing downward.
  • Install the retainer with its chamfered surface facing downward.

(2) Using SST and a press, press in the rear axle shaft.

NOTICE:

Make sure to securely fit the bearing to the rear axle shaft.
