16. INSTALL CHAIN SUB-ASSEMBLY

*a

Cutout

*b

*c

*d

Mark Plate (Orange)

Turn

-

-

(1) Align the mark plate (yellow) and timing mark and install the chain sub-assembly to the crankshaft timing gear or sprocket as shown in the illustration.

NOTICE:

Make sure to install with the mark plate (yellow) facing the front side of the engine.

(2) Install the crankshaft timing gear or sprocket and chain sub-assembly to the crankshaft.

(3) Temporarily install the crankshaft pulley bolt.

(4) Rotate the crankshaft to align the timing mark of the crankshaft timing gear or sprocket and mark plate (yellow) of the chain sub-assembly with the cutout as shown in the illustration.

(5) Align the mark plate (orange) and timing mark as shown in the illustration and install the chain sub-assembly to the camshaft timing gear assembly.

NOTICE:

Make sure to install with the mark plate (orange) facing the front side of the engine.

HINT:

Place the chain sub-assembly on top of the camshaft timing exhaust gear assembly without installing it to the camshaft timing exhaust gear assembly.

(6) Using a wrench, hold the hexagonal portion of the intake camshaft sub-assembly, rotate the camshaft timing gear assembly counterclockwise and align the mark plate (orange) with the timing mark as shown in the illustration, and install the chain sub-assembly to the camshaft timing exhaust gear assembly.

NOTICE:

  • Be careful not to damage the camshaft housing sub-assembly, cylinder head sub-assembly, fuel injector tube and spark plug tube with the wrench.
  • Make sure to install with the mark plate (orange) facing the front side of the engine.
