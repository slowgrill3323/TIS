PROCEDURE

PROCEDURE

1. INSTALL FUEL WITH FILTER PUMP ASSEMBLY

(1) Attach the claw and install the fuel pump harness.

NOTICE:

  • Securely attach the claw.
  • Make sure that there is no foreign matter between the fuel pump and the ground terminal of the fuel pump harness.
  • Make sure that the ground terminal of the fuel pump harness is securely connected to the fuel pump body.

(2) Check the connection state of the fuel pump harness.

Measure the resistance according to the value(s) in the table below.

Standard Resistance:

Tester Connection

Condition

Specified Condition

Fuel pump body - Ground terminal of the fuel pump harness

Always

Below 50 Ω

If the result is not as specified, reconnect the fuel pump harness to the fuel pump so that the ground terminal of the fuel pump harness is securely connected to the fuel pump body.

*b

Fuel Pump Spacer

(1) Apply gasoline to a new O-ring, then install the O-ring and fuel pump spacer to the fuel pump.

(1) Connect the fuel pump harness connector.

(2) Install the fuel pump.

NOTICE:

  • Make sure that the O-ring is not cut or pinched during the installation.
  • Securely attach the claw.

3. INSTALL NO. 1 FUEL SUB-TANK

(1) Attach the claws to install the No. 1 fuel sub-tank to the No. 1 fuel suction support.

NOTICE:

(2) Attach the claws to install the No. 1 fuel sub-tank to the fuel suction plate sub-assembly.

NOTICE:

Securely attach the claw.

(3) Install the jet pump to the No. 1 fuel sub-tank as shown in the illustration.

(4) Attach the claw to install the fuel pump filter hose.

NOTICE:

Securely attach the claw.

4. CONNECT FUEL PUMP HARNESS

(1) Attach the 2 clamps to install the fuel pump harness.

NOTICE:

  • Do not damage the wire harness.
  • When engaging each wire harness to the clamp, attach one wire at a time.

(2) Connect the 2 fuel pump harness connectors.
