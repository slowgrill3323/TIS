PROCEDURE

PROCEDURE

1. INSTALL REAR DECK TRIM COVER (for LH Side without Side Rail)

(a) for Short Deck:

(1) Attach the guide and temporarily install the rear deck trim cover.

(2) Using a T30 "TORX" socket wrench, install the rear deck trim cover with the 4 "TORX"screws.

Torque:

9.0 N·m {92 kgf·cm, 80 in·lbf}

(b) for Long Deck:

(1) Attach the guide and temporarily install the rear deck trim cover.

(2) Using a T30 "TORX" socket wrench, install the rear deck trim cover with the 5 "TORX"screws.

Torque:

9.0 N·m {92 kgf·cm, 80 in·lbf}

2. INSTALL REAR DECK TRIM COVER (for RH Side without Side Rail)

(a) Use the same procedure described for the LH side.

4. INSTALL TOP SIDE PANEL RAIL B RH (for LH Side with Side Rail)

(a) for Short Deck:

(1) Attach the guide and temporarily install the top side panel rail B RH.

(2) Using a T30 "TORX" socket wrench, install the top side panel rail B RH with the 4 "TORX" screws.

Torque:

9.0 N·m {92 kgf·cm, 80 in·lbf}

(b) for Long Deck:

(1) Attach the guide and temporarily install the top side panel rail B RH.

(2) Using a T30 "TORX" socket wrench, install the top side panel rail B RH with the 5 "TORX" screws.

Torque:

9.0 N·m {92 kgf·cm, 80 in·lbf}

5. INSTALL TOP SIDE PANEL RAIL B RH (for RH Side with Side Rail)

(a) Use the same procedure described for the LH side.
