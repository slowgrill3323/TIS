2. INSTALL IGNITION COIL ASSEMBLY

(1) Using an 8 mm socket wrench, install the 4 ignition coil assemblies to the cylinder head cover sub-assembly with the 4 bolts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

NOTICE:

If an ignition coil assembly has been struck or dropped, replace it.
