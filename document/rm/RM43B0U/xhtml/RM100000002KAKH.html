PROCEDURE

1. INSTALL BATTERY VOLTAGE SENSOR

CAUTION:

Wear insulated gloves.

*a

Arrow Mark (Battery Voltage Sensor)

*b

Arrow Mark (Lower Case)

(1) Install the battery voltage sensor with the 2 nuts.

Torque:

7.5 N·m {76 kgf·cm, 66 in·lbf}

NOTICE:

  • Install the battery voltage sensor with the arrow mark of the battery voltage sensor and the arrow mark of the lower case facing each other.
  • Check identification code of the label.

    *a

    Identification Code AA
  • If the battery voltage sensor has been struck or dropped, replace it.

(2) Connect the battery voltage sensor 3 connectors.

NOTICE:

Make sure that the connector is connected securely.
