Brake Pressure Sensor / Brake Pedal Position Sensor Signal Compare Failure (X21D2)

DESCRIPTION

Code

Tester Display

Measurement Item

Trouble Area

Output ECU

X21D2

Brake Pressure Sensor / Brake Pedal Position Sensor Signal Compare Failure

History of mismatch between brake pedal stroke sensor assembly and reaction force pressure sensor
  • Brake booster with master cylinder assembly
  • Brake pedal stroke sensor assembly

No. 1 skid control ECU (brake booster with master cylinder assembly)

(a) Turn the ignition switch off.

(b) Mount a pedal effort gauge.

(c) Turn the ignition switch to ON (READY) and depress the brake pedal once.

HINT:

Depress the brake pedal after turning the ignition switch to ON (READY) to set the vehicle to linear mode (switching solenoid (SSA) on).

(d) Depress the brake pedal slowly and check the brake pedal stroke sensor assembly output voltages with respect to brake pedal depression force.
