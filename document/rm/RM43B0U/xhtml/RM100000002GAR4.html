Title: THEFT DETERRENT / KEYLESS ENTRY: SMART KEY SYSTEM (for Start Function, Gasoline Model): B228562; Steering Lock Position Signal Compare Failure; 2024 - 2026 MY Tacoma [12/2023 -        ]

DTC

B228562

Steering Lock Position Signal Compare Failure

DESCRIPTION

except Manual Transmission:

This DTC is stored when there is no continuity between terminal SLP of the certification ECU (smart key ECU assembly) and ground.

for Manual Transmission:

This DTC is stored when the steering lock position signal sent by the steering lock ECU (steering lock actuator assembly) via direct line and the steering lock position signal sent via LIN communication do not match.
