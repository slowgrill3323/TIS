PROCEDURE

PROCEDURE

1. INSTALL TRANSMISSION CASE GASKET

(1) Coat 3 new transmission case gaskets with ATF, and install them to the automatic transmission case sub-assembly.

2. INSTALL TRANSMISSION VALVE BODY ASSEMBLY

*1

Manual Valve Lever Sub-assembly

*2

Manual Valve

*3

Detent Spring Cover

*4

Detent Spring

(1) Insert the manual valve lever sub-assembly into the groove on the end of the manual valve and install the transmission valve body assembly to the automatic transmission case sub-assembly with the 10 bolts.

Torque:

11 N·m {112 kgf·cm, 8 ft·lbf}

NOTICE:

Take care that the transmission wire does not become trapped.

HINT:

Bolt length

  • Bolt A: 25 mm (0.984 in.)
  • Bolt B: 50 mm (1.97 in.)
  • Bolt C: 57 mm (2.24 in.)

(2) Install the detent spring and detent spring cover to the transmission valve body assembly with the bolt.

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}
