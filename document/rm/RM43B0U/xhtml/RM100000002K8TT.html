PROCEDURE

PROCEDURE

1. INSTALL AUXILIARY BATTERY

2. INSTALL NO. 3 BATTERY CLAMP

(1) Set the No. 3 battery clamp.

NOTICE:

Make sure that the auxiliary battery is pushed frontward and inboard as much as possible before fastening.

(2) Install the bolt.

Torque:

15.4 N·m {157 kgf·cm, 11 ft·lbf}

3. CONNECT BATTERY HOSE

NOTICE:

After replacing the auxiliary battery, install the ventilation hole plug supplied with the new auxiliary battery or the ventilation hole plug from the old auxiliary battery to the ventilation hole on the side to which the battery hose is not connected.

HINT:

  • Push the battery hose onto the auxiliary battery until a click sound is heard.
  • Do not twist the battery hose.

4. CONNECT CABLE TO POSITIVE AUXILIARY BATTERY TERMINAL

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

5. CONNECT CABLE TO NEGATIVE AUXILIARY BATTERY TERMINAL

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

6. PERFORM SYSTEM INITIALIZATION AND CALIBRATION
