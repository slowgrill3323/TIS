SYSTEM DESCRIPTION

SYSTEM DESCRIPTION

(a) The Electronic Controlled Automatic Transmission (ECT) is an automatic transmission that electronically controls shift timing using the TCM. The TCM detects electrical signals that indicate engine and driving conditions, and controls shift points based on driver habits and road conditions.

Shift shock is reduced by controlling the engine, MG and transmission simultaneously.

In addition, the ECT has the following features:

  • Diagnostic function.
  • Fail-safe function when a malfunction occurs.
