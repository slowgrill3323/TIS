DTC

B15C464

Airbag Signal Signal Plausibility Failure

DESCRIPTION

If the DCM (telematics transceiver) detects an error in communication between the DCM (telematics transceiver) and the airbag ECU assembly as a result of the DCM (telematics transceiver) self check, this DTC will be stored.
