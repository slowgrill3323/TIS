PROCEDURE

PROCEDURE

1. INSTALL E.F.I. ENGINE COOLANT TEMPERATURE SENSOR (for Cylinder Head Side)

(1) Install a new gasket to the E.F.I engine coolant temperature sensor.

(2) Using SST, install the E.F.I engine coolant temperature sensor to the cylinder head sub-assembly.

09817-33191

Torque:

19.6 N·m {200 kgf·cm, 14 ft·lbf}

NOTICE:

If the E.F.I engine coolant temperature sensor has been struck or dropped, replace it.

(3) Connect the E.F.I engine coolant temperature sensor connector.

2. INSTALL E.F.I. ENGINE COOLANT TEMPERATURE SENSOR (for Water Inlet Housing Side)

(1) Install a new gasket to the E.F.I engine coolant temperature sensor.

(2) Using SST, install the E.F.I engine coolant temperature sensor to the water inlet housing.

09817-33191

Torque:

Specified tightening torque :

19.6 N·m {200 kgf·cm, 14 ft·lbf}

NOTICE:

If the E.F.I engine coolant temperature sensor has been struck or dropped, replace it.

(3) Connect the E.F.I engine coolant temperature sensor connector.

3. INSTALL INTAKE MANIFOLD WITH THROTTLE BODY ASSEMBLY

HINT:

Perform this procedure only when replacement of the E.F.I engine coolant temperature sensor (for Cylinder Head Side) is necessary.

Click here

4. INSTALL ENGINE COOLANT TEMPERATURE SENSOR (for Outlet Side)

(1) Apply a light coat of engine coolant to the O-ring of the engine coolant temperature sensor.

(2) Install the engine coolant temperature sensor to the front water by-pass joint with a new clip.

NOTICE:

  • If the engine coolant temperature sensor has been struck or dropped, replace it.
  • If reusing the engine coolant temperature sensor, be sure to inspect the O-ring.
  • Make sure that the O-ring is not cracked or moved out of place when installing the engine coolant temperature sensor.

(3) Connect the engine coolant temperature sensor connector.

5. INSTALL ENGINE COOLANT TEMPERATURE SENSOR (for Radiator Side)

(1) Apply a light coat of engine coolant to the O-ring of the engine coolant temperature sensor.

(2) Install the engine coolant temperature sensor to the radiator pipe assembly with a new clip.

NOTICE:

  • If the engine coolant temperature sensor has been struck or dropped, replace it.
  • If reusing the engine coolant temperature sensor, be sure to inspect the O-ring.
  • Make sure that the O-ring is not cracked or moved out of place when installing the engine coolant temperature sensor.

(3) Connect the engine coolant temperature sensor connector.
