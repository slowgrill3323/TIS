Jam Protection Function Activates During Power Tail Gate Operation

DESCRIPTION

The power tail gate is controlled by the multiplex network door ECU and the tail gate motor is driven. During motor operation, the power tail gate (touch sensor) used for jam protection checks for obstruction by foreign objects, etc. by detecting changes in resistance due to contact.

Furthermore, the speed of the tail gate is monitored using the pulse sensor (power tail gate unit assembly RH) and if a change in speed is detected, it is judged that there is an obstruction by a foreign object and the tail gate is stopped.

Once the tail gate is stopped, the operation of the power tail gate is prohibited until it is fully closed by hand.
