PROCEDURE

PROCEDURE

1. INSTALL STABILIZER CONTROL ECU

NOTICE:

  • Avoid any impact to the stabilizer control ECU.
  • Do not drop the stabilizer control ECU. If it is dropped, replace it with a new one.

Torque:

7.0 N·m {71 kgf·cm, 62 in·lbf}

2. INSTALL LOWER QUARTER TRIM PANEL ASSEMBLY LH

3. INSTALL REAR SEAT OUTER BELT ASSEMBLY LH (FLOOR ANCHOR)

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

4. INSTALL REAR NO. 1 SEAT OUTER LAP BELT ANCHOR COVER

5. CONNECT REAR DOOR OPENING TRIM WEATHERSTRIP LH

Click here

6. INSTALL REAR UNDER SIDE COVER LH

7. INSTALL REAR DOOR SCUFF PLATE LH

8. INSTALL SEAT FLOOR BOX (w/o Luggage Compartment Box)

9. INSTALL PACKAGE TRAY TRIM STRAP (w/o Luggage Compartment Box)

10. INSTALL REAR LUGGAGE COMPARTMENT TRIM COVER SUB-ASSEMBLY (w/o Luggage Compartment Box)

11. INSTALL LUGGAGE COMPARTMENT BOX LH (w/ Luggage Compartment Box)

12. CONNECT CABLE TO NEGATIVE AUXILIARY BATTERY TERMINAL

  • except Gasoline Model: Click here
