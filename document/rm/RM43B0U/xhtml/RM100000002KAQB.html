PROCEDURE

PROCEDURE

1. INSTALL REAR ENGINE OIL SEAL

*a

Oil Seal Protrusion Height

-

-

(1) Clean the oil seal retainer oil seal contact surface of the crankshaft.

(2) Apply MP grease to the lip of a new rear engine oil seal.

NOTICE:

  • Do not allow foreign matter to contact the lip of the timing chain case oil seal.
  • Do not allow MP grease to contact the dust seal.

(3) Using SST and a hammer, tap in the rear engine oil seal.

09223-15030 09950-70010 09951-07150

Standard Depth:

-0.9 to 1.1 mm (-0.0354 to 0.0433 in.) (From the edge of the cylinder block sub-assembly and crankshaft bearing cap)

NOTICE:

Do not tap in the rear engine oil seal at an angle.

2. INSTALL NO. 1 CRANKSHAFT POSITION SENSOR PLATE

*a

Pin

*b

Pin Hole

(1) Align the pin hole of the No. 1 crankshaft position sensor plate with the pin of the crankshaft, and install the No. 1 crankshaft position sensor plate.

3. INSTALL FLYWHEEL SUB-ASSEMBLY

(1) Using SST, hold the crankshaft pulley.

09213-54015 91651-60855 09330-00021

NOTICE:

Never turn the crankshaft pulley counterclockwise (when viwed from the front of the engine).

(2) Install and uniformly tighten with the 8 new bolts in several steps in the sequence shown in the illustration.

Torque:

160 N·m {1632 kgf·cm, 118 ft·lbf}

4. INSTALL TRANSMISSION INPUT DAMPER COVER ASSEMBLY

(1) Using SST, hold the crankshaft pulley.

09213-54015 91651-60855 09330-00021

NOTICE:

Never turn the crankshaft pulley counterclockwise (when viwed from the front of the engine).

(2) Install the transmission input damper cover assmebly to the flywheel with the 9 bolts.

Torque:

19.1 N·m {195 kgf·cm, 14 ft·lbf}

NOTICE:

  • Make sure to install the transmission input damper cover assembly in the correct direction.
