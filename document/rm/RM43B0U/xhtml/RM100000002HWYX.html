Electric Parking Brake AUTO OFF Indicator Light Does Not Illuminate/turn off When AUTO Function Set to ON/OFF

DESCRIPTION

The AUTO OFF indicator light in the combination meter assembly turns on/off when the AUTO function is set to ON/OFF.
