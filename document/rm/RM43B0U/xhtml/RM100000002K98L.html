DTC

B124300

Impact Detection Sensor Circuit Malfunction

DESCRIPTION

If the collision door lock release function does not operate normally, or an open or short in the GSW input circuit of the main body ECU (multiplex network body ECU) is detected, DTC B124300 will be stored.

HINT:

If DTC B124300 is stored, the automatic door lock function and collision door lock release function will be prohibited.
