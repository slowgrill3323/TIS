PROCEDURE

1. INSTALL ASSIST STRAP ASSEMBLY LH (for Gasoline Model)

2. INSTALL LOCK SUPPORT BAR BRACKET (except Gasoline Model)

3. INSTALL REAR SEAT CUSHION HINGE BRACKET RH

(a) Use the same procedure described for the RH side.

4. INSTALL REAR SEAT CUSHION HINGE BRACKET LH

Torque:

53 N·m {540 kgf·cm, 39 ft·lbf}

5. INSTALL NO. 1 SEAT CUSHION COVER SUB-ASSEMBLY LH

NOTICE:

Be careful not to dirty or damage the No. 1 seat cushion cover sub-assembly LH and rear seat cushion pad LH.

HINT:

When installing a No. 1 seat cushion cover sub-assembly LH, refer to Precaution in order to prevent wrinkles from forming.

*a

Hog Ring Pliers

-

-

(1) Using hog ring pliers, install the No. 1 seat cushion cover sub-assembly LH with 14 new hog rings as shown in the illustration.

NOTICE:

  • Be careful not to dirty or damage the No. 1 seat cushion cover sub-assembly LH and rear seat cushion pad LH.
  • Be sure to securely install the hog rings as shown in the illustration while preventing wrinkles from forming on the No. 1 seat cushion cover sub-assembly LH.

13. INSTALL REAR NO. 1 SEATBACK COVER SUB-ASSEMBLY LH

HINT:

When installing a rear No. 1 seatback cover sub-assembly LH, refer to Precaution in order to prevent wrinkles from forming.

*a

Hog Ring Pliers

-

-

NOTICE:

  • Be careful not to dirty or damage the rear No. 1 seatback cover sub-assembly LH and rear seatback pad LH.
  • Be sure to securely install the hog rings as shown in the illustration while preventing wrinkles from forming on the rear No. 1 seatback cover sub-assembly LH.

Hog Ring Pliers
