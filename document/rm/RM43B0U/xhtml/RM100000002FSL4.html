PROCEDURE

1. INSTALL CLUTCH PEDAL STROKE SENSOR ASSEMBLY

NOTICE:

  • Do not drop the clutch pedal stroke sensor assembly.
  • If the clutch pedal stroke sensor assembly has been dropped, replace the clutch pedal stroke sensor assembly with a new one.

(a) When installing a new clutch pedal stroke sensor assembly:

*a

Clutch Pedal Stroke Sensor Assembly Lever

*b

Clutch Pedal Groove

*c

Clutch Pedal Stroke Sensor Assembly Lever Set Pin

-

-

NOTICE:

Do not break the clutch pedal stroke sensor assembly lever set pin before installing the clutch pedal stroke sensor assembly with the 2 nuts.

(1) Install a new clutch pedal stroke sensor assembly to the clutch pedal support assembly with the 2 nuts.

Torque:

8.5 N·m {87 kgf·cm, 75 in·lbf}

NOTICE:

  • Engage the clutch pedal stroke sensor assembly lever with the clutch pedal groove.
  • Check that there is no foreign matter attached to the contact surface of the clutch pedal stroke sensor assembly.
  • Check that the tip of the clutch pedal stroke sensor assembly lever is protruding from the clutch pedal groove.

(2) Connect the connector.

(3) Firmly depress the clutch pedal to break the clutch pedal stroke sensor assembly lever set pin.

(4) Remove the broken lever set pin.

(b) When reusing the clutch pedal stroke sensor assembly:

*a

Clutch Pedal Stroke Sensor Assembly Lever

*b

Clutch Pedal Groove

(1) Install the clutch pedal stroke sensor assembly to the clutch pedal support assembly and temporarily tighten the 2 nuts.

NOTICE:

  • Engage the clutch pedal stroke sensor assembly lever with the clutch pedal groove.
  • Check that there is no foreign matter attached to the contact surface of the clutch pedal stroke sensor assembly.
  • Check that the tip of the clutch pedal stroke sensor assembly lever is protruding from the clutch pedal groove.

(2) Connect the connector.

3. ADJUST CLUTCH PEDAL STROKE SENSOR ASSEMBLY

NOTICE:

When the clutch pedal stroke sensor assembly is being reused, perform the following procedure to adjust it.

(a) Connect the GTS to the DLC3 with the ignition switch off.

(b) Turn the ignition switch to ON.

(c) Turn the GTS on.

(d) Enter the following menus: Powertrain / Engine / Data List / Clutch Stroke Sensor Voltage.

Powertrain > Engine > Data List

Tester Display

Clutch Stroke Sensor Voltage

(e) Read the stroke sensor value in the Data List, and turn the clutch pedal stroke sensor assembly slowly to the right or left to adjust the output voltage so that it is within the following range.

Standard Voltage (clutch pedal fully released):

0.96 to 1.04 V

(f) Tighten the 2 nuts.

Torque:

8.5 N·m {87 kgf·cm, 75 in·lbf}

NOTICE:

Do not depress the clutch pedal after turning the ignition switch ON.

(g) Turn the GTS off and turn the ignition switch off.

(h) Disconnect the GTS from the DLC3.
