PROCEDURE

PROCEDURE

1. INSTALL INPUT SHAFT FRONT BEARING

(1) Using SST and a hammer, tap in a new input shaft front bearing.

09304-12012

HINT:

  • Tap in the input shaft front bearing as far as possible.
  • After installing the bearing to the crankshaft, make sure that it rotates smoothly.

(2) Apply adhesive to 2 or 3 threads of the bolts.

Adhesive:

Toyota Genuine Adhesive 1324, Three Bond 1324 or equivalent

(3) Install the 2 bolts.

Torque:

160 N·m {1632 kgf·cm, 118 ft·lbf}

NOTICE:

Do not forget to install the bolts when installing the input shaft front bearing.

2. INSTALL CLUTCH DISC ASSEMBLY

*a

Flywheel Side

-

-

(1) Insert SST into the clutch disc assembly. Then insert SST (together with the clutch disc) into the flywheel.

09301-00110

NOTICE:

Be sure to install the clutch disc so that it is facing in the correct direction.

3. INSTALL CLUTCH COVER ASSEMBLY

*a

Matchmark

-

-

(1) Align the matchmarks on the clutch cover assembly and flywheel.

(2) Tighten the 6 bolts as described below.

  1. Determine the first bolt to be tightened by choosing the bolt closest to the knock pin.
  2. Uniformly tighten the 6 bolts in diametrically opposite pairs relative to the position of the first bolt. Use the illustration as a reference.

(3) Lightly move SST up and down, and right and left.

09301-00110

(4) Check that the disc is in the center, and then tighten the bolts.

Torque:

19.1 N·m {195 kgf·cm, 14 ft·lbf}

4. INSPECT AND ADJUST CLUTCH COVER ASSEMBLY

(1) Using a dial indicator with a roller instrument, measure the diaphragm spring tip alignment.

Maximum misalignment:

0.5 mm (0.020 in.)

If the misalignment is more than the maximum, use SST to adjust the diaphragm spring tip alignment.

09333-00013

5. INSTALL RELEASE FORK SUPPORT

Torque:

49 N·m {500 kgf·cm, 36 ft·lbf}

6. INSTALL CLUTCH RELEASE FORK DUST SEAL

7. INSTALL CLUTCH RELEASE FORK COLLAR

8. INSTALL RELEASE BEARING HUB CLIP

9. INSTALL CLUTCH RELEASE BEARING ASSEMBLY

Toyota Genuine Clutch Release Hub Grease or equivalent (mineral ingredients should not be included)

-

-

(1) Apply grease to the clutch release bearing assembly.

Grease:

Toyota Genuine Clutch Release Hub Grease or equivalent (mineral ingredients should not be included)

Amount of Grease:

0.3 to 0.5 g (0.011 to 0.018 oz)

HINT:

When replacing the clutch release bearing assembly with a new one, do not apply grease.

(2) Install the clutch release bearing assembly to the release fork sub-assembly.

10. INSTALL CLUTCH RELEASE FORK SUB-ASSEMBLY

*a

The surfaces of the release fork that contact the release bearing

*b

The surfaces of the release fork that contact the release fork support

*c

The surfaces of the release fork that contact the push rod of the release cylinder

*d

Contact surface of the bearing

Toyota Genuine Clutch Release Hub Grease or equivalent (mineral ingredients should not be included)

Toyota Genuine Clutch Spline Grease or equivalent

(1) Apply grease or equivalent to the following areas.

Grease:

Toyota Genuine Clutch Release Hub Grease or equivalent (mineral ingredients should not be included)

Amount of Grease:

The surfaces of the release fork that contact the release bearing

1.0 to 1.3 g (0.04 to 0.05 oz)

The surfaces of the release fork that contact the release fork support

1.3 to 1.8 g (0.05 to 0.06 oz)

The surfaces of the release fork that contact the push rod of the release cylinder

0.5 to 0.9 g (0.02 to 0.03 oz)

(2) Install the clutch release bearing assembly together with the clutch release fork sub-assembly to the clutch housing.

NOTICE:

After the installation, move the fork forward and backward to check that the release bearing slides smoothly.

(3) Apply the clutch spline grease to the spline of the input shaft.

Grease:

Toyota Genuine Clutch Spline Grease or equivalent

NOTICE:

Make sure that no grease adheres to the contact surface of the bearing.
