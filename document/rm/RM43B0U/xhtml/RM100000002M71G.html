PROCEDURE

PROCEDURE

1. INSTALL POWER STEERING SENSOR WIRE

2. INSTALL RACK AND PINION POWER STEERING GEAR ASSEMBLY

Install in this Direction

-

-

(1) Install the rack and pinion power steering gear assembly to the front frame assembly with the 3 bolts, 3 nuts and 3 washers.

Torque:

120 N·m {1224 kgf·cm, 89 ft·lbf}

(1) Install the wire harness to the rack and pinion power steering gear assembly with the 2 bolts.

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}

(2) Connect the wire harness connectors to the rack and pinion power steering gear assembly.

HINT:

Make sure that the connector is fully inserted before rotating the lock lever to engage the lock.

3. TEMPORARILY INSTALL FRONT SHOCK ABSORBER WITH COIL SPRING

Click here

4. CONNECT TIE ROD END SUB-ASSEMBLY LH

(1) Connect the tie rod end sub-assembly LH to the steering knuckle LH with the nut.

Torque:

120 N·m {1224 kgf·cm, 89 ft·lbf}

(2) Install a new cotter pin.

NOTICE:

Further tighten the nut up to 60° if the holes for the cotter pin are not aligned.

5. CONNECT TIE ROD END SUB-ASSEMBLY RH

(a) Perform the same procedure as for the LH side.

6. CONNECT NO. 2 STEERING INTERMEDIATE SHAFT SUB-ASSEMBLY

*a

Matchmark

-

-

(1) Align the matchmarks on the No. 2 steering intermediate shaft sub-assembly and rack and pinion power steering gear assembly.

(2) Connect the No. 2 steering intermediate shaft sub-assembly to the rack and pinion power steering gear assembly.

(3) Install the bolt.

Torque:

35 N·m {357 kgf·cm, 26 ft·lbf}
