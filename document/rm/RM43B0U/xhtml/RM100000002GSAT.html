FREEZE FRAME DATA

DESCRIPTION

(a) When an occupant classification system DTC is stored, the airbag ECU assembly stores the current vehicle (ECU or sensor) state as freeze frame data.

CHECK FREEZE FRAME DATA

(a) Select a DTC to display the freeze frame data.

Body Electrical > SRS Airbag > Trouble Codes

(b) Check the freeze frame data for the output DTC.

CHECK FREEZE FRAME DATA

(a) Read the freeze frame data recorded when the DTC was stored.

Body Electrical > SRS Airbag
