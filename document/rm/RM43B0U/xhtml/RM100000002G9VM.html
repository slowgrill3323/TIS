FREEZE FRAME DATA

HINT:

  • If DTCs or vehicle control history (RoB) is stored, record the freeze frame data at the time the DTC was detected.
  • Once freeze frame data of a DTC is recorded, the data will not be updated or cleared until the DTC is cleared.
  • Each item in the contents of the freeze frame data is similar to that of the Data List.

FREEZE FRAME DATA

(a) Using the DTC check screen, display the DTCs.

Chassis > Stabilizer with Disconnection Mechanism > Trouble Codes

(b) Select DTCs which have displayed freeze frame data and check the freeze frame data.
