SYSTEM DESCRIPTION

ACCESSORY POWER OUTLET SYSTEM

(a) Accessory Power Outlet System Description

(1) The voltage inverter converts voltage from the HV battery to a standard AC120 V (2400 W max) output.

(2) The accessory power outlet system can be operated when the power outlet switch is turned on with the ignition switch ON (READY).
