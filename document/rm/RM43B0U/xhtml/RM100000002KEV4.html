Power Tail Gate does not Operate Using Outside Switch

DESCRIPTION

The power tail gate elbow switch LH (luggage electrical key switch), power tail gate elbow switch RH (luggage electrical key switch) and tail gate handle switch (tail gate opener switch assembly) sends a signal to the multiplex network door ECU using CAN communication via the certification ECU (smart key ECU assembly).

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • First perform the communication function inspections in How to Proceed with Troubleshooting to confirm that there are no CAN communication malfunctions before troubleshooting this problem.

    Click here

  • If the multiplex network door ECU has been replaced, or if any of the connectors has been disconnected, initialize the power tail gate system.

    Click here

  • If the certification ECU (smart key ECU assembly) is replaced, refer to registration.

    for Gasoline Model: Click here

    except Gasoline Model: Click here
