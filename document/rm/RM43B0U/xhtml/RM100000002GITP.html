INITIALIZATION

INITIALIZE MULTIPLEX NETWORK DOOR ECU

NOTICE:

Initialization of the multiplex network door ECU (tail gate initial position learning) is necessary if the following work procedures are performed while the tail gate is open.

  • The cable to the negative (-) auxiliary battery terminal is connected.
  • A power source system fuse of the power tail gate system is disconnected and reconnected.
  • The multiplex network door ECU is replaced or the connectors are disconnected.
  • The power tail gate unit assembly RH is removed and reinstalled, or replaced.

The power tail gate does not operate unless the multiplex network door ECU is initialized.

HINT:

  • Initialization is not necessary if the above procedures are performed while the tail gate is closed.
  • When disconnecting the negative (-) auxiliary battery terminal, some systems need to be initialized after the negative (-) auxiliary battery terminal is reconnected.
