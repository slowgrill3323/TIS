(3) Using a snap ring expander, install the snap ring to the input shaft.

2. INSTALL 5TH GEAR NEEDLE ROLLER BEARING

NOTICE:

Although either type may be installed, make sure to follow the installation procedure for each type.

(1) Coat the 5th gear needle roller bearing with gear oil and install it to the input shaft.

NOTICE:

  • Install the 5th gear needle roller bearing in the axial direction of the input shaft.
  • Do not expand the 5th gear needle roller bearing 22 mm (0.866 in.) or more.

*1

5th Gear Needle Roller Bearing

*a

4. INSTALL NO. 3 SYNCHRONIZER RING

(1) Coat the No. 3 synchronizer ring with gear oil, and then install it to the input shaft.
