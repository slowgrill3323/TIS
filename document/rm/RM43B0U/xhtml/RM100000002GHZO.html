8. INSTALL FUEL TANK ASSEMBLY

CAUTION:

The fuel tank assembly is very heavy. Be sure to follow the procedure described in the repair manual, or the fuel tank assembly may fall off the engine lifter.

(1) Set the fuel tank assembly on an engine lifter and raise the fuel tank assembly.

(2) Install the 3 fuel tank bands with the 3 pins and 3 clips.

(3) Install the 3 fuel tank bands with the 3 bolts.
