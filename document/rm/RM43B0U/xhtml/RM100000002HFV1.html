UPDATE ECU SECURITY KEY

UPDATE ECU SECURITY KEY

NOTICE:

  • It is necessary to log in to the Toyota server to perform an ECU security key update.
  • Users other than Toyota dealers can log in with a guest account issued when an ECU is ordered.

(a) In accordance with the display on the GTS, perform Health Check.

(b) Check the "RoB" column of the Health Check result screen and if any result is displayed, clear it.

HINT:

If RoB is stored, it is necessary to clear it in advance, as ECU security key update cannot be performed with RoB stored.

(c) Check whether "ECU Security Key: Necessary" is displayed in the upper corner of the GTS screen.

(d) If "ECU Security Key: Necessary" is displayed, follow the prompts on the GTS screen to update the ECU security key.

(e) After ECU security key update is completed, perform Health Check in accordance with the display on the GTS.

(f) Check that "ECU security key: -" is displayed.

NOTICE:

If "ECU Security Key: Necessary" is displayed in the upper corner of the GTS screen, perform the ECU security key update again.

HINT:

If RoB is output, clear RoB.
