DTC

U032257

Software Incompatibility with Body Control Module Invalid/Incompatible Software Component

DESCRIPTION

The vehicle approaching speaker controller receives vehicle information from the main body ECU (multiplex network body ECU) via CAN communication.

This DTC is stored when the vehicle information from the main body ECU (multiplex network body ECU) and that stored in the vehicle approaching speaker controller do not match.

CAUTION / NOTICE / HINT

NOTICE:

  • If a vehicle approaching speaker controller which was installed to another vehicle is used, the information stored in the vehicle approaching speaker controller will not match the information from the vehicle. As a result, a DTC may be stored.
  • Before replacing the main body ECU (multiplex network body ECU), refer to Registration.
