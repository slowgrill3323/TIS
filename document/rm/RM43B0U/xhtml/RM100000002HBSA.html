PROCEDURE

1. ECU EXCHANGE

NOTICE:

  • When replacing the engine stop and start ECU, make sure to download the number of starter operations (step 1). After replacing the engine stop and start ECU (step 2), write the number of starter operations to the new engine stop and start ECU (step 3).
  • After replacing the engine stop and start ECU, make sure to perform external backup boost converter (eco run vehicle converter assembly) existence learning (step 4).
  • Even if the number of starter operations could not be downloaded, after replacing the engine stop and start ECU with a new one, make sure to perform external backup boost converter (eco run vehicle converter assembly) existence learning (step 4).

(a) Download the engine stop and start ECU data. (Step 1)

(1) Enter the following menus: Powertrain / Stop and Start / Utility / ECU Exchange.

Powertrain > Stop and Start > Utility

Tester Display

ECU Exchange

(2) According to the display on the GTS, select "Step 1 Save" and then "Next" to download the ECU data from the engine stop and start ECU.

(b) Replace the engine stop and start ECU. (step 2)

(1) Replace the engine stop and start ECU.

Click here

(c) Write the engine stop and start ECU data to the new engine stop and start ECU. (Step 3)

(1) Enter the following menus: Powertrain / Stop and Start / Utility / ECU Exchange.

Powertrain > Stop and Start > Utility

Tester Display

ECU Exchange

(2) According to the display on the GTS, select "Step 3 Write" and then "Next" to write the ECU data to the new engine stop and start ECU.

(d) Perform external backup boost converter (eco run vehicle converter assembly) existence learning. (Step 4)

(1) According to the display on the GTS, select "Step 4 Learning" and then "Next" to perform external backup boost converter (eco run vehicle converter assembly) existence learning.

HINT:

  • If the external backup boost converter (eco run vehicle converter assembly) existence value of the old engine stop and start ECU differs from the current learned value or if data could not be downloaded from the old engine stop and start ECU, the learning result will be displayed.
  • If the external backup boost converter (eco run vehicle converter assembly) existence value of the old engine stop and start ECU was "available" but the value after replacement is "not available", the external backup boost converter (eco run vehicle converter assembly) may not be connected securely. Check the connector connection condition and perform learning again.

2. CLEAR NUMBER OF STARTER OPERATIONS

NOTICE:

  • After the starter assembly is replaced, be sure to clear the number of starter operations stored in the engine stop and start ECU.
  • When the starter assembly is replaced, "ST relay" must be also replaced.

(a) Enter the following menus: Powertrain / Stop and Start / Utility / Starter Exchange.

Powertrain > Stop and Start > Utility

Tester Display

Starter Exchange

(b) Following the prompts on the GTS display, clear the number of starter operations.

(c) Check the Vehicle Control History and clear X0877 if it is output.

Click here

3. TOTAL NUMBER OF ENGINE STARTS RESET

NOTICE:

After the drive plate and ring gear sub-assembly is replaced, be sure to clear the number of total number of engine starts stored in the engine stop and start ECU.

(a) Enter the following menus: Powertrain / Stop and Start / Utility / Total Number of Engine Starts Reset.

Powertrain > Stop and Start > Utility

Tester Display

Total Number of Engine Starts Reset

(b) Check the Vehicle Control History and clear X0878 if it is output.

Click here

4. RESET AIR CONDITIONING INFORMATION (for Mexico Models)

NOTICE:

After replacing the engine stop and start ECU or air conditioning amplifier assembly, initialization of the air conditioning information in the engine stop and start ECU is necessary.

(a) Enter the following menus: Powertrain / Stop and Start / Utility / Reset A/C Information.

Powertrain > Stop and Start > Utility

Tester Display

Reset A/C Information

(b) Following the prompts on the GTS display, clear the A/C information.

5. LEARNING AIR CONDITIONING INFORMATION (for Mexico Models)

HINT:

When initialization of the air conditioning information is complete, air conditioning information learning is automatically performed. If the learning cannot be performed properly, inspect the CAN communication system.

6. CLEAR THE INTEGRATED CURRENT VALUE

HINT:

If the integrated current value does not exceed the threshold*1 even though the battery has been charged until the battery current decreases to 5 A or less, perform an integrated current value initialization.

  • *1: Refer to System Description for details on the integrated current threshold.

7. SWITCH BATTERY TYPE

(a) Check whether the battery is specified or not specified.

(b) Connect the GTS to the DLC3.

(c) Turn the ignition switch to ON.

HINT:

Do not start the engine.

(d) Turn the GTS on.

(e) Enter the following menus: Powertrain / Stop and Start / Data List / Battery Type.

Powertrain > Stop and Start > Data List

Tester Display

Battery Type

HINT:

Check the battery type (Specified/Other) stored by the engine stop and start ECU.

(f) If the battery installed does not match the battery stored by the ECM, switch to the battery type for the installed battery.

(1) Enter the following menus: Powertrain / Engine / Utility / Switch Battery Type.

Powertrain > Engine > Utility

Tester Display

Switch Battery Type

8. CHECK OIL PUMP WITH SOLENOID ASSEMBLY

NOTICE:

When the oil pump with solenoid assembly or engine stop and start ECU is replaced, perform the "Electromagnetic Oil Pump" Active Test.

(a) Connect the GTS to the DLC3.

(b) Turn the ignition switch to ON.

HINT:

Do not start the engine.

(c) Move the shift lever to P.

(d) Turn the GTS on.

(e) Perform the "Electromagnetic Oil Pump" Active Test for 2 seconds.

Enter the following menus: Powertrain / Stop and Start / Active Test / Electromagnetic Oil Pump.

Powertrain > Stop and Start > Active Test

Tester Display

Electromagnetic Oil Pump

(f) Check that the "Stop&Start Operating Permission (Electromagnetic Oil Pump)" Data List item is listed as "OK".

(g) Enter the following menus: Powertrain / Stop and Start / Data List / Stop&Start Operating Permission (Electromagnetic Oil Pump).

Powertrain > Stop and Start > Data List

Tester Display

Stop&Start Operating Permission (Electromagnetic Oil Pump)
