7. K0 COMPENSATION CODE RESET

NOTICE:

Perform K0 Compensation Code Reset when replacing the TCM.

(a) Connect the GTS to the DLC3.

(b) Turn the ignition switch to ON.

NOTICE:

Do not turn the ignition switch to ON.

(c) Turn the GTS on.

(d) Enter the following menus: Powertrain / Transmission / Utility / K0 Compensation Code Reset.
