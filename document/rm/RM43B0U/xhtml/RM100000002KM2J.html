2. INSTALL SEAT POSITION AIRBAG SENSOR

HINT:

The illustrations are representative examples, and details may differ.

*a

Feeler Gauge

*b

1.0 mm (0.0394 in.)

*c

Seat Rail

-

-

(1) Using a 1.0 mm (0.0394 in.) feeler gauge, temporarily install the seat position airbag sensor.

NOTICE:

  • If the seat position airbag sensor has been dropped, or there are any cracks, dents or other defects in the case or connector, replace the seat position airbag sensor with a new one.
  • When installing the seat position airbag sensor, make sure that the SRS wiring does not interfere with or is pinched between other parts.

HINT:

Maintain a clearance between the seat position airbag sensor and the seat rail of between 0.6 mm (0.0236 in.) and 2 mm (0.0787 in.)

(b) for Manual Seat:

(1) Install the seat position airbag sensor with the bolt.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(2) Connect the connector and engage the clamp.

(3) Check that there is no looseness in the installed parts of the seat position airbag sensor.

(c) for Power Seat:

(1) Install the seat position airbag sensor with the bolt.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(2) Connect the connector.

(3) Check that there is no looseness in the installed parts of the seat position airbag sensor.
