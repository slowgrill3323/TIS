FREEZE FRAME DATA

FREEZE FRAME DATA

HINT:

  • When a DTC is output, freeze frame data at the time the DTC is output is stored.
  • Once freeze frame data at the time a DTC is output is stored, data is not updated or cleared until the stored data is cleared (Clear DTCs).

(a) Using the GTS, select the DTC that is displayed as having freeze frame data on the DTC check screen, and check the freeze frame data.
