PROCEDURE

PROCEDURE

1. TEMPORARILY TIGHTEN FRONT DISC BRAKE BLEEDER PLUG

HINT:

Fully tighten the front disc brake bleeder plug after bleeding the system.

2. INSTALL FRONT DISC BRAKE BLEEDER PLUG CAP

3. INSTALL PISTON SEAL

Lithium soap base glycol grease

-

-

(1) Apply lithium soap base glycol grease to the entire circumference of 4 new piston seals.

(2) Install the 4 piston seals to the disc brake cylinder assembly.

NOTICE:

Securely install the piston seal into the groove of the disc brake cylinder assembly.

4. INSTALL FRONT DISC BRAKE PISTON

*1

Cylinder Boot

*2

Front Disc Brake Piston

Lithium soap base glycol grease

-

-

(1) Apply a light layer of lithium soap base glycol grease to the entire circumference of 4 new cylinder boots.

(2) Install the 4 cylinder boots to the front disc brake pistons.

NOTICE:

Securely install the cylinder boot into the groove of the front disc brake piston.

(3) Apply a light layer of lithium soap base glycol grease to the contact surfaces of the front disc brake pistons, and install it to the disc brake cylinder assembly.

NOTICE:

Do not forcibly install the front disc brake piston into the disc brake cylinder assembly.

5. INSTALL CYLINDER BOOT

*1

Cylinder Boot

*2

Front Disc Brake Piston

*3

Disc Brake Cylinder Assembly

*4

Set Ring

(1) Install the 4 cylinder boots to the disc brake cylinder assembly as shown in the illustration.

NOTICE:

  • Securely install the cylinder boot into the groove of the disc brake cylinder assembly and front disc brake piston.
  • Do not damage the cylinder boot.

(2) Using a screwdriver with its tip wrapped with protective tape, install the 4 new set rings.

NOTICE:

Do not damage the cylinder boot.
