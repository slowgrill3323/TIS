Push Start Switch Illumination Circuit

DESCRIPTION

for Gasoline Model:

  • The illuminated entry system controls the engine switch illumination.

except Gasoline Model:

  • The illuminated entry system controls the power switch illumination.

WIRING DIAGRAM

for Gasoline Model:
except Gasoline Model:

CAUTION / NOTICE / HINT

NOTICE:

  • Check that the "Start Indicator Function" customization setting is set to "ON".
  • for Gasoline Model:

    Before replacing the certification ECU (smart key ECU assembly) or engine switch, refer to Registration.

    Click here

  • except Gasoline Model:

    Before replacing the certification ECU (smart key ECU assembly) or power switch, refer to Registration.

    Click here

  • for Gasoline Model:

    When replacing the engine switch, always replace it with a new one.

  • except Gasoline Model:

    When replacing the power switch, always replace it with a new one.
