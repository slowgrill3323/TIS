PROCEDURE

1. INSTALL 2ND GEAR NEEDLE ROLLER BEARING

(1) Coat the 2nd gear needle roller bearing with gear oil, and then install it to the counter gear.

2. INSTALL COUNTERSHAFT 2ND SPEED GEAR

(1) Coat the countershaft 2nd speed gear with gear oil, and then install it to the counter gear.

3. INSTALL NO. 2 SYNCHRONIZER RING SET

*b

(1) Coat the No. 2 synchronizer ring with gear oil.

(2) Align the 3 synchronizer rings as shown in the illustration.

NOTICE:

Align the protrusion of the No. 2 synchronizer inner ring with the cutout of the No. 2 synchronizer outer ring.

(3) Install the No. 2 synchronizer ring set to the countershaft 2nd speed gear.

NOTICE:

Align the protrusion of the No. 2 synchronizer middle ring with the groove of the 2nd gear.

4. INSTALL NO. 1 TRANSMISSION CLUTCH HUB

*a

Groove

*b

Protrusion

(1) Using SST and a press, install the No. 1 transmission clutch hub to the counter gear.

09316-60012

NOTICE:

  • Pay attention to installation direction of the No. 1 transmission clutch hub.
  • Align the groove of the No. 1 transmission clutch hub with the protrusion of the No. 2 synchronizer ring set.

(2) Check that the countershaft 2nd speed gear and No. 2 synchronizer ring set move smoothly.
