PROCEDURE

PROCEDURE

1. INSTALL TRANSFER ASSEMBLY

(1) Install the transfer assembly to the transmission assembly with the 8 bolts.

Torque:

24 N·m {245 kgf·cm, 18 ft·lbf}

NOTICE:

When installing the transfer assembly to the transmission assembly, make sure that the transmission assembly and transmission case oil seal do not become damaged.

2. INSTALL TRANSFER BREATHER HOSE SUB-ASSEMBLY (for TRD Off Road Package)

3. INSTALL TRANSFER BREATHER HOSE SUB-ASSEMBLY (except TRD Off Road Package)

4. INSTALL WIRE HARNESS CLAMP BRACKET

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}

5. CONNECT WIRE HARNESS

6. INSTALL REAR ENGINE MOUNTING INSULATOR

Torque:

51.8 N·m {528 kgf·cm, 38 ft·lbf}

7. INSTALL NO. 3 FRAME CROSSMEMBER ASSEMBLY

Bolts

Nuts

(1) Install the No. 3 frame crossmember assembly with the 4 nuts and 4 bolts.

Torque:

42 N·m {428 kgf·cm, 31 ft·lbf}

NOTICE:

Make sure to install the nuts so that they are facing the rear of the vehicle.

HINT:

Hold the bolts to tighten the nuts.

(2) Connect the rear engine mounting insulator to the No. 3 frame crossmember assembly with the 4 bolts.

Torque:

41 N·m {418 kgf·cm, 30 ft·lbf}

Torque:

12 N·m {122 kgf·cm, 9 ft·lbf}
