PROCEDURE

PROCEDURE

1. INSTALL SEAT TRACK UPPER RAIL COVER (for Rear Side)

Install in this Direction

-

-

(1) Connect the No. 3 terminal of the slide motor to the positive (+) auxiliary battery terminal and the No. 4 terminal to the negative (-) auxiliary battery terminal to move the front seat adjuster assembly LH backward so that the claw of the seat track upper rail cover can be attached.

(2) Attach the claw and install the seat track upper rail cover.

2. INSTALL SEAT TRACK UPPER RAIL COVER LH (for Rear Side)

(a) Use the same procedure described for the seat track upper rail cover.

3. INSTALL FRONT SEAT LOWER CUSHION SHIELD RH (for Driver Seat)

Remove in this Direction

-

-

(1) Connect the No. 3 terminal of the slide motor to the positive (+) auxiliary battery terminal and the No. 4 terminal to the negative (-) auxiliary battery terminal to move the front seat adjuster assembly LH backward so that the claw of the seat track upper rail cover can be detached.

(2) Using a thin-bladed screwdriver, attach the claw and install the seat track upper rail cover.

HINT:

Tape the thin-bladed screwdriver tip before use.

4. INSTALL SEAT TRACK UPPER RAIL COVER LH (for Front Side)

Install in this Direction

-

-

(1) Connect the No. 4 terminal of the slide motor to the positive (+) auxiliary battery terminal and the No. 3 terminal to the negative (-) auxiliary battery terminal to move the front seat adjuster assembly LH forward so that the claw of the seat track upper rail cover LH can be attached.

(2) Attach the claw and install the seat track upper rail cover LH.

(3) Connect the No. 3 terminal of the slide motor to the positive (+) auxiliary battery terminal and the No. 4 terminal to the negative (-) auxiliary battery terminal to move the front seat adjuster assembly LH to the center position.

5. INSTALL SEAT TRACK UPPER RAIL COVER (for Front Side)

(a) Use the same procedure described for the LH side.

-
