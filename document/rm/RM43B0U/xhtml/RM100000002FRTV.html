PROCEDURE

1. INSTALL ENGINE STOP AND START SOLENOID FILTER

(1) Install the engine stop and start solenoid filter with the nut.

Torque:

10 N·m {102 kgf·cm, 7 ft·lbf}

NOTICE:

If the engine stop and start solenoid filter has been struck or dropped, replace it.

(2) Connect the engine stop and start solenoid filter connector.

NOTICE:

Be sure to securely connect the engine stop and start solenoid filter connector.
