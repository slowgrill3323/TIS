A/T CODE REGISTRATION

NOTICE:

When the TCM is replaced, perform "A/T Code Registration".

HINT:

When the TCM (when the transmission compensation code can be read) is replaced, perform A/T Code Registration (Read Compensation Code) before replacement.

After the TCM is replaced, perform A/T Code Registration (Set Compensation Code).

(a) Make sure the shift lever is in P.

(b) Turn the ignition switch to ON.

NOTICE:

Do not start the engine.

(c) Enter the following menus: Powertrain / Transmission / Utility / A/T Code Registration.

Powertrain > Transmission > Utility

Tester Display

A/T Code Registration

(d) Select "Read Compensation Code".

(e) Perform A/T Code Registration (Read Compensation Code) according to the screen display.

NOTICE:

If the compensation code cannot be read, perform A/T Code Registration (Arbitrary Code).

(f) Turn the ignition switch off.

(g) Replace the TCM.

(h) Turn the ignition switch to ON.

NOTICE:

Do not start the engine.

(i) Enter the following menus: Powertrain / Transmission / Utility / A/T Code Registration.

Powertrain > Transmission > Utility

Tester Display

A/T Code Registration

(j) Select "Set Compensation Code".

(k) Perform A/T Code Registration (Set Compensation Code) according to the screen display.

(l) Turn the ignition switch off.

A/T CODE REGISTRATION (ARBITRARY CODE)

NOTICE:

Perform A/T Code Registration (Arbitrary Code) after replacing the automatic transmission assembly and torque converter assembly, automatic transmission assembly or TCM.

(a) Record the compensation code on the transmission compensation code label of the automatic transmission assembly.

NOTICE:

  • When replacing the automatic transmission assembly and torque converter assembly or automatic transmission assembly, record the compensation code of the automatic transmission assembly service part.
  • When replacing the TCM (when the transmission compensation code cannot be read), record the compensation code of the automatic transmission assembly installed in the vehicle.

*2

Compensation Code

*3

  • The compensation code is a 64-digit alphanumeric value imprinted on the transmission compensation code label.
  • This information can also be scanned from the service QR code.

(b) Replace the relevant part.

(c) Make sure the shift lever is in P.

(d) Turn the ignition switch to ON.

NOTICE:

Do not start the engine.

(e) Enter the following menus: Powertrain / Transmission / Utility / A/T Code Registration (Arbitrary Code).

Powertrain > Transmission > Utility

ATF THERMAL DEGRADATION ESTIMATE RESET

NOTICE:

If either of the following conditions is met, perform ATF Thermal Degradation Estimate Reset.

  • The ATF has been replaced.

    HINT:

    If 50000 or more is displayed for the Data List item "ATF Thermal Degradation Estimate", thermal degradation of the ATF is suspected. Perform ATF Thermal Degradation Estimate Reset after replacing the ATF.

  • Approximately 50% or more of the ATF has been replaced during a repair of the transmission or a similar operation.

(a) Turn the ignition switch to ON.

NOTICE:

Do not start the engine.

(b) Enter the following menus: Powertrain / Transmission / Utility / ATF Thermal Degradation Estimate Reset.

Powertrain > Transmission > Utility

ATF Thermal Degradation Estimate Reset

PERFORM ROAD TEST TO ALLOW TCM TO LEARN

CAUTION:

  • Perform the following procedures while strictly observing all traffic laws and speed limits.
  • Do not operate the GTS while driving the vehicle.
  • Operating the GTS while driving the vehicle will prevent you from paying sufficient attention to vehicle surroundings, and could result in a serious accident.

NOTICE:

  • Do not accelerate or decelerate rapidly.
  • This test must be performed after checking and adjusting the engine.
  • Perform drive learning while the ATF temperature is within a range of 60 to 100°C (140 to 212°F).

(a) The Road Test procedure is as follows:

(1) According to the replaced parts, perform the appropriate road test shown below.

Replaced Parts

Road Test
  • Automatic transmission assembly and torque converter assembly
  • Automatic transmission assembly
  • Torque converter assembly
  • Solenoid (SL1) valve
  • Solenoid (SL2) valve
  • Solenoid (SL3) valve
  • Solenoid (SL4) valve
  • Solenoid (SL5) valve
  • Solenoid (SL6) valve
  • Solenoid (SL) valve
  • Solenoid (SLT) valve
  • Solenoid (SLU) valve
  • Solenoid (SC3) valve
  • Transmission valve body assembly
  • TCM
  • Engine assembly

Perform Pattern 1 and Pattern 2.

Pre-work:

  • According to the display on the GTS, read the Data List.: Powertrain / Engine / Data List.

    With the shift position in P and the vehicle stopped, check the accelerator pedal depressed amount that results in a Data List item "Accelerator Position" between 15 and 35%, and 10%.

Pattern 1:

  • With the shift lever in D, accelerate to approximately 60 km/h (37.28 mph) while "Accelerator Position" is between 15 and 35%, decelerate with the brake pedal released until the gear shifts down to 3rd gear, and then depress the brake pedal to stop the vehicle.

    If the shift shock is large, repeat this procedure until shift shock no longer occurs.

Pattern 2:

  • Select the S2 range with the shift lever in S. Then, from the condition where the vehicle is traveling at a vehicle speed of approximately 5 km/h (3.11 mph) and in 2nd gear, depress the accelerator pedal so that the "Accelerator Position" becomes about 0% to 10% in approximately 3 seconds, and accelerate for 20 seconds or more.

    Perform this drive learning 5 times.

(b) Repeat Road Test Pattern until shift shock and engine revving subside.

NOTICE:

When repeating drive learning, always begin a new run with the vehicle stopped.
