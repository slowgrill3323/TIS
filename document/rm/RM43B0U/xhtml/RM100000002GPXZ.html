PROCEDURE

PROCEDURE

1. REMOVE STEERING LOCK ACTUATOR WITH ECU ASSEMBLY (for Manual Transmission)

NOTICE:

  • Do not drop the steering lock actuator with ECU assembly, strike it with tools or subject it to impacts.
  • If the steering lock actuator with ECU assembly is subjected to an impact, replace it with a new one.
  • Before replacing the steering lock actuator with ECU assembly, refer to Registration.

*a

Wooden Block

*b

Aluminum Plates

(1) Secure the steering column assembly in a vise using aluminum plates and wooden blocks.

NOTICE:

  • Do not overtighten the vise.
  • Do not clamp the steering lock actuator with ECU assembly in the vise.

(2) Using SST, strike a punch mark for drilling in the center of the steering lock set bolt.

09622-00010

NOTICE:

When striking the punch mark, in order to prevent impact force from being applied to the steering lock actuator with ECU assembly, do not use a normal center punch.

(3) Using a drill, drill a hole in the 2 steering lock set bolts and insert a screw extractor.

(4) Using the screw extractor, remove the 2 steering lock set bolts and steering lock actuator with ECU assembly.

NOTICE:

Push in the screw extractor by hand, and do not strike it with tools, etc.

2. REMOVE STEERING SHAKE DUMPER (except Manual Transmission)

*a

Wooden Block

*b

Aluminum Plates

(1) Secure the steering column assembly in a vise using aluminum plates and wooden blocks.

NOTICE:

  • Do not overtighten the vise.
  • Do not clamp the upper steering column clamp in the vise.

(2) Using SST, strike a punch mark for drilling in the center of the steering lock set bolt.

09622-00010

(3) Using a drill, drill a hole in the 2 steering lock set bolts and insert a screw extractor.

(4) Using the screw extractor, remove the 2 steering lock set bolts and steering shake dumper.

NOTICE:

Push in the screw extractor by hand, and do not strike it with tools, etc.
