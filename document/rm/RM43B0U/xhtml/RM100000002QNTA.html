PROCEDURE

PROCEDURE

1. INSTALL NO. 2 POWER OUTLET SOCKET

(1) Connect the connector to the No. 2 power outlet socket.

(2) Attach the claws.

(3) Using a T30 "TORX" socket wrench, install the No. 2 power outlet socket with the 4 screws.

Torque:

8.5 N·m {87 kgf·cm, 75 in·lbf}

2. INSTALL POWER OUTLET SOCKET ASSEMBLY

(1) Connect the connector to the power outlet socket assembly.

(2) Using a T30 "TORX" socket wrench, install the power outlet socket assembly with the 4 bolts.

Torque:

5.0 N·m {51 kgf·cm, 44 in·lbf}
