1. INSTALL INNER REAR VIEW MIRROR ASSEMBLY

Install in this Direction

-

-

(1) Slide the inner rear view mirror assembly as shown in the illustration to temporarily install it.

(2) Using a T20 "TORX" socket wrench, install the inner rear view mirror assembly with the "TORX" screw.

Torque:

1.5 N·m {15 kgf·cm, 13 in·lbf}

(3) Check that there is no play in the inner rear view mirror assembly.

(4) Attach the clamp and connect the 2 connectors.
