PROCEDURE

PROCEDURE

1. INSTALL PARK/NEUTRAL POSITION SWITCH ASSEMBLY

HINT:

Make sure that the manual valve lever shaft has not been rotated prior to installing the park/neutral position switch assembly as the detent spring may become detached from the manual valve lever shaft.

*a

Neutral Basic Line

*b

Groove

*c

Torque Wrench Fulcrum Length

-

-

(1) Apply adhesive to threads of the bolt.

Adhesive:

Toyota Genuine Adhesive 1344, Three Bond 1344 or equivalent

(2) Using SST, temporarily install the park/neutral position switch assembly to the automatic transmission assembly with the bolt.

09729-00120

HINT:

Tighten the bolt to the specified torque when adjusting the park/neutral position switch assembly.

(3) Install the lock washer and nut to the park/neutral position switch assembly.

Torque:

6.9 N·m {70 kgf·cm, 61 in·lbf}

(4) Turn the transmission control shaft lever RH clockwise until it stops, and then return the transmission control shaft lever RH counterclockwise 2 notches to N.

(5) Align the neutral basic line with the groove as shown in the illustration.

(6) Using SST, tighten the bolt.

09729-00120

Torque:

Specified tightening torque

12.8 N*m (131 kgf*cm, 9 ft.*lbf)

HINT:

  • When using SST, make sure that SST is parallel to the torque wrench.
  • Calculate the torque wrench reading when changing the fulcrum length of the torque wrench.

    Click here

  • When using a SST (fulcrum length of 200 mm (7.874 in.)) + torque wrench (fulcrum length of 162 mm (6.3779 in.)):5.7 N*m (58 kgf*cm, 50 in.*lbf)

(7) Using a screwdriver, bend the tabs of the lock washer.

NOTICE:

  • Be careful not to damage the park/neutral position switch assembly.
  • Bend at least 2 of the lock washer tabs.

HINT:

Tape the screwdriver tip before use.

(8) Connect the park/neutral position switch assembly connector.
