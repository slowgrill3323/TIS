CUSTOMIZE PARAMETERS

CUSTOMIZE PARKING ASSIST MONITOR SYSTEM

(a) Customize using multi-information display.

NOTICE:

  • When the customer requests a change in a function, first make sure that the function can be customized.
  • Be sure to make a note of the current settings before customizing.
  • When troubleshooting a function, first make sure that the function is set to the default setting.
  • When a trailer is connected, a screen for checking the connection of the trailer appears on the multi-information display; selecting OK on this screen turns off the setting of "RCD" automatically.

    However, if a problem has occurred with the forward recognition camera, then the screen for confirming the connection of the trailer may not appear.
