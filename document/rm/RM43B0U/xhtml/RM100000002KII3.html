PROCEDURE

1. INSTALL POWER WINDOW REGULATOR MOTOR ASSEMBLY LH

(1) Install the power window regulator motor assembly LH to the rear door window regulator assembly with the 3 screws.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

HINT:

A new rear window regulator uses self-tapping screws to thread new installation holes when the self-tapping screws are inserted.
