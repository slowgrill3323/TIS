SOFTWARE VERSION CONFIRMATION

PROCEDURE

PROCEDURE

1. NECESSITY OF SOFTWARE UPDATE CONFIRMATION

NOTICE:

If the GTS is connected and an operation such as a health check is performed when OTA reprogramming is being performed, the software update is suspended. In this case, the software update guidance will be displayed again on the navigation screen the next day or later and it is necessary to perform the software update again by operating the screen. To avoid this situation, check if OTA reprogramming is being performed and only connect the GTS after the OTA reprogramming has completed.

HINT:

When performing reprogramming of the forward recognition camera via a wired connection, the wired connection has priority and it is not necessary to wait for OTA reprogramming to complete.

(a) Connect the GTS to the DLC3.

(b) In accordance with the display on the GTS, select and perform Health Check on the menu screen.

(c) After completing the health check, perform "ECU SEARCH" to display if there is an update available for ECU.

HINT:

By performing "ECU SEARCH", the software version check can complete.

(d) Perform the software update as necessary.
