(1) Press the main body ECU (multiplex network body ECU) to the housing sidewall as shown in the illustration.

HINT:

Make sure the angle between the power distribution box assembly and the main body ECU (multiplex network body ECU) is 20° or more.

(2) Slide the main body ECU (multiplex network body ECU) along the housing sidewall in the direction of the arrow (power distribution box assembly side A) and insert the guides as shown in the illustration.

NOTICE:

Do not press against the main body ECU (multiplex network body ECU) sidewall too hard.

(3) Rotate the main body ECU (multiplex network body ECU) about its edge which contacts with the power distribution box assembly side A as shown in the illustration.

Install in this Direction

Pressing Area

-

-

(1) Rotate it until the main body ECU (multiplex network body ECU) enters the lock of the power distribution box assembly.

(2) Press the push area, attach the claw, and install the main body ECU (multiplex network body ECU).

NOTICE:

  • Be sure to press only the push area.
  • Do not insert with a force of 98 N (10 kgf, 22.0 lbf) or higher to prevent it from being damaged.
  • Check the engagement of the main body ECU (multiplex network body ECU) and the power distribution box assembly by the locking sound.
  • Do not put your weight on the main body ECU (multiplex network body ECU) when engaging the main body ECU (multiplex network body ECU) to prevent it from being damaged.

HINT:

If a locking sound cannot be heard, visually check the lock part engagement. Or check that the height of the main body ECU (multiplex network body ECU) and the power distribution box assembly are the same.

(3) Press it in one of the ways as shown in the illustration.
