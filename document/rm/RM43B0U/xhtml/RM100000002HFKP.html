INSTALLATION

CAUTION / NOTICE / HINT

NOTICE:

  • Immediately after installing the brake pads, the braking performance may be reduced. Always perform a road test in a safe place while paying attention to the surroundings.
  • After replacing the front disc brake pads, the brake pedal may feel soft due to clearance between the front disc brake pads and front disc. Depress the brake pedal several times until the brake pedal feels firm.
  • After replacing the front disc brake pads, always perform a road test to check the braking performance and check for vibrations.

HINT:

  • Use the same procedure for the RH side and LH side.
  • The following procedure is for the LH side.
  • Refer to the illustration to confirm the type of brake caliper.

1. INSTALL FRONT ANTI SQUEAL SHIM KIT

NOTICE:

  • When replacing the front disc brake pads with new ones, make sure to replace the front anti-squeal shim kit at the same time.
  • Do not apply lubricants to the lining surface of the brake pad.
  • Install the front disc brake anti-squeal shims in the correct position and direction.

3. INSTALL FRONT DISC BRAKE FITTING KIT (for Type A)

(1) Temporarily install the front disc brake anti-rattle spring to the front disc brake pads as shown in the illustration.

(1) Install the front disc brake anti-rattle with hole pin (upper side).

(2) Install the front disc brake anti-rattle with hole pin (lower side) and set the front disc brake anti-rattle spring to the front disc brake pads.

(3) Install the pin hold clip to the front disc brake anti-rattle with hole pin and engage the hook to the disc brake cylinder assembly.

4. INSTALL FRONT DISC BRAKE FITTING KIT (for Type B)

Front of the Vehicle

(1) Temporarily install the front disc brake anti-rattle spring to the front disc brake pads as shown in the illustration.

(1) Install the front disc brake anti-rattle with hole pin (upper side) and set the front disc brake anti-rattle spring to the front disc brake pads.

NOTICE:

Set the front disc brake anti-rattle spring so that it does not overlap and ride up onto the disc brake cylinder assembly.

(2) While pressing the front disc brake anti-rattle spring, install the front disc brake anti-rattle with hole pin (lower side).

(3) Install the pin hold clip to the front disc brake anti-rattle with hole pin and engage the hook to the disc brake cylinder assembly.
