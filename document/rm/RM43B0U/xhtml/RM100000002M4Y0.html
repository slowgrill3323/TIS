B179316

Load Sensor for Occupant Detection System Circuit Voltage Below Threshold

When the detected voltage is below specified value before supplying power to the sensor.
  • Harness or connector
  • Front occupant classification sensor LH (front seat frame with adjuster assembly RH)*1, *3 or front occupant classification sensor LH (front seat adjuster assembly RH)*2, *4
  • Rear occupant classification sensor LH (front seat frame with adjuster assembly RH)*1, *3 or rear occupant classification sensor LH (front seat adjuster assembly RH)*2, *4
  • Airbag ECU assembly

Comes on

SRS Airbag

A

Does not apply to test/check mode
  • *1: The front occupant classification sensor LH and rear occupant classification sensor LH are built into the front seat frame with adjuster assembly RH.
  • *2: The front occupant classification sensor LH and rear occupant classification sensor LH are built into the front seat adjuster assembly RH.
  • *3: except 6-way Power + 2-way Manual Seat
  • *4: for 6-way Power + 2-way Manual Seat

  • After replacing the airbag ECU assembly, refer to work procedure.

    HINT:

    When disconnecting and reconnecting the auxiliary battery, there is an automatic learning function that completes learning when the respective system is used.

    Click here

  • Inspect the fuses for circuits related to this system before performing the following procedure.

HINT:

  • If it is difficult to perform troubleshooting (wire harness inspection), remove the front passenger seat installation bolts to see under the seat cushion.
  • In the above case, lift and hold the seat so that it does not fall down. Hold the seat only as necessary because holding the seat for a long period of time may cause seat rail deformation.
