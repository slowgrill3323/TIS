DESCRIPTION

The transmission revolution sensor (SP2) detects the output shaft rotation speed and sends it to the TCM.

Based on the transmission revolution sensor signal (NT) and the transmission revolution sensor (SP2) signal, the TCM controls engine torque and shift timing.

MONITOR DESCRIPTION

The transmission revolution sensor (SP2) monitors the output shaft speed.

The TCM calculates gear shifts comparing the transmission revolution sensor (NT) with the transmission revolution sensor (SP2).

If the TCM detects a malfunction the transmission revolution sensor (SP2), illuminates the MIL and stores a DTC.
