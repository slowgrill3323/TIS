PROCEDURE

PROCEDURE

1. TEMPORARILY INSTALL REAR SLIDING ROOF HOUSING MOUNTING BRACKET RH

(1) Temporarily install the rear sliding roof housing mounting bracket RH.

1. Install the rear sliding roof housing mounting bracket RH with the bolt.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

2. After tightening the bolts, loosen them 180 to 270 degrees.

2. TEMPORARILY INSTALL REAR SLIDING ROOF HOUSING MOUNTING BRACKET LH

(a) Use the same procedure for the RH and LH sides.

3. TEMPORARILY INSTALL CENTER SLIDING ROOF HOUSING MOUNTING BRACKET RH

(1) Temporarily install the center sliding roof housing mounting bracket LH.

1. Install the center sliding roof housing mounting bracket LH with the 3 bolts.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

2. After tightening the bolts, loosen them 450 to 540 degrees.

4. TEMPORARILY INSTALL CENTER SLIDING ROOF HOUSING MOUNTING BRACKET LH

(a) Use the same procedure for the LH side.

5. TEMPORARILY INSTALL FRONT SLIDING ROOF HOUSING MOUNTING BRACKET RH

(1) Temporarily install the front sliding roof housing mounting bracket RH.

1. Install the front sliding roof housing mounting bracket RH with the bolt.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

2. After tightening the bolts, loosen them 180 to 270 degrees.

6. TEMPORARILY INSTALL FRONT SLIDING ROOF HOUSING MOUNTING BRACKET LH

(a) Use the same procedure for the RH and LH sides.

7. INSTALL SUNSHADE TRIM SUB-ASSEMBLY

8. INSTALL NO. 1 SLIDING ROOF PIECE SUB-ASSEMBLY RH

Torque:

1.5 N·m {15 kgf·cm, 13 in·lbf}

9. INSTALL NO. 1 SLIDING ROOF PIECE SUB-ASSEMBLY LH

10. INSTALL SLIDING ROOF DRIVE GEAR ASSEMBLY

MP grease

-

-

(1) Apply MP grease to the gear of the sliding roof drive gear assembly.

(2) Install the sliding roof drive gear assembly with the 2 bolts.

Torque:

5.4 N·m {55 kgf·cm, 48 in·lbf}

NOTICE:

Evenly tighten the left and right bolts.
