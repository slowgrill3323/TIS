Title: BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS: ELECTRONICALLY CONTROLLED BRAKE SYSTEM (for Gasoline Model): C051D00; Multi-axis Acceleration Sensor Module "A"; 2024 - 2026 MY Tacoma [12/2023 -        ]

DTC

C051D00

Multi-axis Acceleration Sensor Module "A"

DESCRIPTION

The airbag ECU assembly has a built-in yaw rate and acceleration sensor.

The skid control ECU (brake actuator assembly) receives signals from the yaw rate and acceleration sensor (airbag ECU assembly) via CAN communication.

This DTC is also output when the calibration has not been completed.
