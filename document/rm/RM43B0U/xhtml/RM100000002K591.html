PROCEDURE

PROCEDURE

1. INSTALL FRONT AXLE HUB DUST SEAL

(1) Using a brass bar and hammer, tap on the front drive outboard joint assembly until a new front axle hub dust seal contacts the edge of the front drive outboard joint assembly.

NOTICE:

  • Do not damage the front axle hub dust seal.
  • The front axle hub dust seal should be installed completely.
  • Make sure to install the front axle hub dust seal in the correct direction.

2. INSTALL FRONT DRIVE SHAFT DUST COVER

(1) Using SST and a press, press in a new front drive shaft dust cover until the inboard joint contacts the edge of the inboard joint.

09527-10011 09316-20011

NOTICE:

  • Do not damage the front drive shaft dust cover.
  • The front drive shaft dust cover should be installed completely.
  • Make sure to install the front drive shaft dust cover in the correct direction.

3. INSTALL OUTBOARD JOINT BOOT

Outboard Joint Side

Inboard Joint Side

(1) Before installing the outboard joint boot, wrap the splines of the front drive outboard joint assembly with protective tape to prevent the outboard joint boot from being damaged.

(2) Install new parts to the front drive outboard joint assembly in the following order.

  1. Front No. 2 axle outboard joint boot clamp
  2. Outboard joint boot
  3. Front axle outboard joint boot clamp

(3) Install the outboard joint boot so that it is between the front No. 2 axle outboard joint boot clamp and front drive outboard joint assembly.

(4) Temporarily install the front axle outboard joint boot clamp to the outboard joint boot.

(5) Pack the front drive outboard joint assembly and outboard joint boot with grease from the boot kit.

Standard grease capacity:

for Gasoline Model TRD Package or 1Motor-HEV Model

203.0 to 213.0 g (7.16 to 7.51 oz)

for Gasoline Models Other Than TRD Package

165.0 to 175.0 g (5.82 to 6.17 oz)

(6) Install the outboard joint boot to the groove of the front drive outboard joint assembly.

NOTICE:

  • Do not apply grease to the part of the outboard joint boot that contacts the groove.
  • Do not allow foreign matter to enter the outboard joint boot.

4. INSTALL FRONT NO. 2 AXLE OUTBOARD JOINT BOOT CLAMP

*a

Turn

*b

Hold

(1) Install SST to the front No. 2 axle outboard joint boot clamp, and then while pressing the outboard joint boot, slightly tighten the SST bolt.

09521-24010

NOTICE:

  • Correctly set the front No. 2 axle outboard joint boot clamp to the guide groove.
  • Do not damage the outboard joint boot.

(2) Tighten SST so that the front No. 2 axle outboard joint boot clamp is pinched.

Standard clearance:

1.5 mm (0.0591 in.) or less

NOTICE:

  • When tightening SST, make sure the clearance of the front No. 2 axle outboard joint boot clamp is within the standard clearance.
  • Do not damage the outboard joint boot.

(3) Remove SST from the front No. 2 axle outboard joint boot clamp.

(4) Using SST, measure the clearance of the front No. 2 axle outboard joint boot clamp.

09240-00021

Standard clearance:

1.5 mm (0.0591 in.) or less

NOTICE:

If the measured value exceeds the specified value, retighten the front No. 2 axle outboard joint boot clamp.

5. INSTALL FRONT AXLE OUTBOARD JOINT BOOT CLAMP

(a) Use the same procedure described for the front NO. 2 axle outboard joint boot clamp.

7. INSTALL INBOARD JOINT BOOT

(1) Install the inboard joint boot to the inboard joint.

NOTICE:

  • Do not apply grease to the part of the inboard joint boot that contacts the groove.
  • Do not allow foreign matter to enter the inboard joint boot.

(2) Check that the inboard joint boot and outboard joint boot are not stretched or contracted when the front drive shaft assembly is at the standard length.

Dimension (A) (for Gasoline Models Other Than TRD Package):

for LH Side

588.15 mm (1.93 ft.)

for RH Side

666.65 mm (2.19 ft.)

Dimension (A) (for Gasoline Model TRD Package or 1Motor-HEV Model):

for LH Side

594.27 mm (1.95 ft.)

for RH Side

671.27 mm (2.20 ft.)

NOTICE:

Keep the front drive shaft assembly level during inspection.

If the inboard joint boot and outboard joint boot are stretched or contracted, correct them.

8. INSTALL FRONT AXLE INBOARD JOINT BOOT CLAMP

(a) Use the same procedure described for the front axle outboard joint boot clamp.
