FAIL-SAFE CHART

FAIL-SAFE CHART

(a) When the skid control ECU (brake actuator assembly) detects a solenoid valve malfunction or abnormal signal from a sensor, the brake actuator assembly cuts power and sends a signal indicating a malfunction of the electronically controlled brake system to the ECM.

HINT:

Based on signals sent from the skid control ECU (brake actuator assembly), the brake actuator assembly prevents the solenoids from operating, and the ECM no longer accepts fuel-cut request signals from electronically controlled brake system.

(b) If a malfunction of ABS, brake assist, TRAC, AUTO LSD* or VSC is detected, operation of the function is prohibited and the related warning lights illuminate.

The EBD function can still operate even if operation of the ABS function is prohibited. However, if the EBD function is also affected, operation of the EBD system will be prohibited and the brake system warning light (red indicator) illuminates.

  • *: for 2WD

HINT:

When a component of the electronically controlled brake system is in fail-safe mode, operation is prohibited. However, if the system enters fail-safe mode while braking is being controlled, control is gradually suspended to prevent sudden changes in vehicle behavior.

(c) If a malfunction in the electronically controlled brake system or electric parking brake system is detected, control by the brake hold function is prohibited to prevent undesired operation and to protect the system.

HINT:

When control by the brake hold system is prohibited, the brake hold switch is ON and the following conditions are met, the brake hold indicator will blink.

  • Vehicle conditions:
    • The driver door is closed.
    • The driver seat belt is fastened.

(d) If the skid control ECU (brake actuator assembly) detects a malfunction in the SRS system, secondary collision brake control is prohibited to prevent undesired operation and to protect the system.

HINT:

When control is prohibited, a message is displayed on the multi-information display in the combination meter assembly.
