PROCEDURE

1. VIN (VEHICLE IDENTIFICATION NUMBER)

NOTICE:

The Vehicle Identification Number (VIN) must be written to a replacement ECM.

HINT:

The VIN is a 17-digit alphanumeric vehicle identification number. The GTS is required to register the VIN.

(a) DESCRIPTION

HINT:

This registration section consists of 2 parts: Read VIN and Write VIN.

(1) Read VIN: This procedure allows the VIN stored in the ECM to be read in order to confirm that the 2 VINs, provided with the vehicle and stored in the vehicle ECM, are the same.

(2) Write VIN: This procedure allows the VIN to be written into the ECM. If the ECM is replaced, or the ECM VIN and vehicle VIN do not match, the VIN can be registered, or overwritten in the ECM by following this procedure.

(b) READ VIN

(1) Confirm the vehicle VIN.

(2) Enter the following menus: Powertrain / Engine / Utility / VIN / VIN Read.

Powertrain > Engine > Utility

Tester Display

VIN

(3) According to the display on the GTS, read the VIN stored in the ECM.

(c) WRITE VIN

(1) Confirm the vehicle VIN.

(2) Enter the following menus: Powertrain / Engine / Utility / VIN / VIN Write.

Powertrain > Engine > Utility

Tester Display

VIN

(3) According to the display on the GTS, write the VIN to the ECM.

2. HEAVY KNOCK HISTORY

NOTICE:

When replacing the ECM, use the GTS to save "Heavy Knock History" and then upload it to the new ECM.

HINT:

This function allows the stored history data to be saved or written when the ECM is replaced.

(a) SAVE HISTORY

HINT:

This function saves the history data stored in the ECM to the GTS.

(1) Enter the following menus: Powertrain / Engine / Utility / Heavy Knock History / Save History.

Powertrain > Engine > Utility

Tester Display

Heavy Knock History

(2) Save the history data to the GTS according to the instructions on the GTS screen.

(b) WRITE HISTORY

HINT:

This function writes the history data saved to the GTS to the ECM.

(1) Enter the following menus: Powertrain / Engine / Utility / Heavy Knock History / Write History.

Powertrain > Engine > Utility

Tester Display

Heavy Knock History

(2) Write the history data to the ECM according to the instructions on the GTS screen.
