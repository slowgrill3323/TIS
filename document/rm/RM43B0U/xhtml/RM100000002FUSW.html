2. REMOVE CLUTCH RELEASE FORK BOOT

3. REMOVE CLUTCH RELEASE FORK SUB-ASSEMBLY

4. REMOVE CLUTCH RELEASE BEARING ASSEMBLY

(1) Remove the clutch release fork sub-assembly from the clutch release bearing assembly.

NOTICE:

If the clutch release fork collar comes off from the clutch release fork sub-assembly, replace the clutch release fork collar and clutch release fork dust seal with new ones.

5. REMOVE RELEASE BEARING HUB CLIP

6. REMOVE RELEASE FORK SUPPORT

7. REMOVE CLUTCH COVER ASSEMBLY

*a

Matchmark

-

-

(1) Place matchmarks on the clutch cover assembly and flywheel.

(2) Loosen each set bolt one turn at a time until spring tension is released.

(3) Remove the 6 set bolts and pull off the clutch cover assembly.

NOTICE:

  • Do not drop the clutch disc assembly.
  • Keep the lining part of the clutch disc assembly, pressure plate and surface of the flywheel sub-assembly free from oil and foreign matter.

8. REMOVE CLUTCH RELEASE FORK COLLAR

9. REMOVE CLUTCH RELEASE FORK DUST SEAL

10. REMOVE INPUT SHAFT FRONT BEARING

(1) Remove any 2 diametrically opposite bolts.

(2) Using SST, remove the input shaft front bearing.
