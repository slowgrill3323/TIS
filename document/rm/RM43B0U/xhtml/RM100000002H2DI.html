DTC

B151287

Lost Communication with Steering Switch Control Module Missing Message

DESCRIPTION

The multiplex network steering ECU is connected to the combination meter assembly via CXPI communication line. If a CXPI communication malfunction is detected, the combination meter assembly stores DTC B151287.

Diagnosis Condition:

  • Ignition switch ON
  • IG power source voltage is 9.5 V or more

Malfunction Status:

  • No communication with multiplex network steering ECU

Malfunction Time:

  • 5 seconds or more
  • Spiral cable sub-assembly
  • Multiplex network steering ECU
  • Harness or connector
  • Combination meter assembly

CAUTION / NOTICE / HINT

NOTICE:

  • When replacing the combination meter assembly, always replace it with a new one. If a combination meter assembly which was installed to another vehicle is used, the information stored in it will not match the information from the vehicle and a DTC may be stored.
  • When replacing the combination meter assembly, update the ECU security key.
