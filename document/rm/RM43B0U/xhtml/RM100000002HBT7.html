DESCRIPTION

  • Perform this procedure if the engine restarts after an abnormally short period of time after being stopped by the stop and start system.

Engine Start Request (Battery Condition)

Action to be Taken and Suspected Area

Conditions
  • As charging is prioritized over the stop and start system, the engine is restarted. (Run the engine until battery charging priority ends and check that the stop and start system operates.)
  • Battery (deteriorated)
  • Data List / Battery Voltage

    (Standard: 11.8 V or less)

  • Data List / Integrated Current

    (Integrated current value fluctuates 2765 A-sec or more when engine is stopped)

  • Data List / Battery Fluid Temperature

    -10°C or lower, or 60°C or higher (14°F or lower, or 140°F or higher)(for Mexico Models: -10°C or lower, or 55°C or higher (14°F or lower, or 131°F or higher ))

    except for Mexico Models:

    When the intake air temperature sensor is abnormal

  • Data List / Integrated Current, Status of Battery Charge Control

    -6451 A-sec or less: "Status of Battery Charge Control" is Charge Control Coordination Mode, Stop and Start Standalone Mode or Low Temperature Mode

    -1152 A-sec or less: "Status of Battery Charge Control" is Stop and Start Restriction Mode or Temperature High/Low Mode

    When any of the criteria are met (except for Mexico Models):

    - When the battery sensor current is abnormal

    - When LIN communication is abnormal

9.

CHECK VEHICLE CONDITIONS (AIR CONDITIONING SYSTEM)

(a) Based on the respective Data List item, ensure the engine restart request conditions are not met.

HINT:

Depending on the vehicle, some of the items are not displayed.

Engine Start Request (A/C)

Action to be Taken and Suspected Area

Conditions
  • Set the air conditioning system temperature to be approximately equivalent to the ambient temperature.
  • Check that the A/C switch or defogger switch is not turned on during engine stop by stop and start control.
  • Check that the stop and start operation conditions related to the air conditioning system are met.
  • If engine start is requested even though all stop and start operation conditions related to the air conditioning system are met, inspect the following items:
    • - Air conditioning system
  • Timer in the air conditioning amplifier assembly runs out
  • Both of the following conditions are met:
    • Blower Switch: ON
    • Evaporator temperature exceeds threshold value (threshold value varies with temperature, humidity and amount of solar radiation)
  • Any of the following conditions are met:
    • A/C switch, blower switch: ON
    • Defogger switch: OFF to ON
    • Blower switch turned off during auto control or the blower mode changed to manual
    • Changes from stop and start system priority control to normal system control

Engine Start Request (Brake Release)

Action to be Taken and Suspected Area

Conditions

When the engine is stopped by the stop and start system and an engine start request is generated even though the brake pedal is not depressed, inspect the following items:

  • Stop Light switch assembly (ECM)
  • Stop Light switch assembly (ABS/VSC)
  • Master cylinder pressure sensor (brake control system)
  • Airbag sensor assembly (G sensor)
  • Stop light switch circuit

HINT:

Depending on the vehicle, the deceleration sensor may not be built into the airbag sensor assembly, but installed separately.
  • Data List / Stop Light SW (ECM)

    (OFF: Brake pedal released)

  • Data List / Stop Light SW (ABS/VSC)

    (OFF: Brake pedal released)

  • Data List / Master Cylinder Pressure*

    (Brake pedal released)

HINT:

  • *: w/ Vacuum Brake Booster
  • w/o Vacuum Brake Booster: Refer to the threshold of "Driver Operation Control Torque".

Engine Start Request (Brake Negative Pressure Low)

Action to be Taken and Suspected Area

Conditions
  • Check that the brake pedal is not further depressed and the brake pedal is not excessively pumped.
  • Drive the vehicle and wait for the vacuum in the brake booster to restore and check that the stop and start system operates.
  • Brake booster vacuum insufficient
  • Vacuum sensor assembly
  • Brake booster assembly
  • Master cylinder pressure sensor (brake control system)
  • Engine stop and start ECU

Data List / Brake Boost Pressure

(Brake booster vacuum insufficient (Under 37.5 kPa (0.4 kg/cm2, 5 psi)) (Brake booster vacuum changes in accordance with brake fluid pressure (brake pedal effort))

Conditions

    (ON: Brake pedal depressed → OFF: Brake pedal released → ON: Brake pedal depressed)

    (0.04 MPa or more)

Conditions
