PROCEDURE

PROCEDURE

1. INSTALL STEERING LOCK ACTUATOR WITH ECU ASSEMBLY (for Manual Transmission)

NOTICE:

  • Do not drop the steering lock actuator with ECU assembly, strike it with tools or subject it to impacts.
  • If the steering lock actuator with ECU assembly is subjected to an impact, replace it with a new one.
  • When replacing the steering lock actuator with ECU assembly, perform registration.

*a

Wooden Block

*b

Aluminum Plates

(1) Secure the steering column assembly in a vise using aluminum plates and wooden blocks.

NOTICE:

  • Do not overtighten the vise.
  • Do not clamp the steering lock actuator with ECU assembly in the vise.

(2) Temporarily install the steering lock actuator with ECU assembly with 2 new steering lock set bolts.

(3) Tighten the 2 steering lock set bolts until the bolt head breaks off.

2. INSTALL STEERING SHAKE DUMPER (except Manual Transmission)

*a

Wooden Block

*b

Aluminum Plates

(1) Secure the steering column assembly in a vise using aluminum plates and wooden blocks.

NOTICE:

  • Do not overtighten the vise.
  • Do not clamp the steering shake dumper in the vise.

(2) Temporarily install the steering shake dumper with 2 new steering lock set bolts.

(3) Tighten the 2 steering lock set bolts until the bolt head breaks off.
