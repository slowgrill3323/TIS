Digital Key cannot be Canceled

DESCRIPTION

After the ignition switch was turned ON using the electrical key transmitter sub-assembly, it is possible to cancel the digital key system. If it is not possible to cancel the digital key system after the ignition switch was turned ON using the electrical key transmitter sub-assembly, there may be a malfunction in the communication system between the certification ECU (smart key ECU assembly) and the digital key ECU assembly.
