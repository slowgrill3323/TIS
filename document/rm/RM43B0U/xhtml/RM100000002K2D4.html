PROCEDURE

PROCEDURE

1. INSTALL TRANSMISSION VALVE BODY ASSEMBLY

(1) Coat 2 new O-rings with ATF and install it to the hybrid module assembly.

(2) Install the wire harness clamp to the hybrid module assembly with the bolt.

Torque:

11 N·m {112 kgf·cm, 8 ft·lbf}

(3) Install the transmission valve body assembly to the hybrid module assembly with the 4 bolts.

Torque:

11 N·m {112 kgf·cm, 8 ft·lbf}

(4) Connect the 2 solenoid valve connectors.

2. INSTALL TRANSAXLE OIL PAN SUB-ASSEMBLY

*a

Torque Wrench Fulcrum Length

-

-

(1) Install the transmission oil cleaner magnet to the transmission oil pan sub-assembly.

(2) Install a new transmission oil pan gasket to the transmission oil pan sub-assembly.

(3) Using SST, install the transmission oil pan sub-assembly to the hybrid module assembly with the 4 bolts.

09961-00950
