PROCEDURE

PROCEDURE

1. INSTALL TAIL GATE LOCK ACTUATOR ASSEMBLY (w/ Tail Gate Lock Actuator)

2. INSTALL TAIL GATE LOCK ASSEMBLY LH (w/ Power Tail Gate)

(1) Connect the connecor.

(2) Install the tail gate lock assembly LH with the screw.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(3) Using a T40 "TORX" socket wrench, install the tail gate lock assembly LH with the screw.

Torque:

29 N·m {296 kgf·cm, 21 ft·lbf}

3. INSTALL POWER TAIL GATE SENSOR ASSEMBLY LH (w/ Power Tail Gate)

(1) Connect the connecor.

(2) Install the tail gate sensor assembly LH with the 2 screws.

(1) Using a T40 "TORX" socket wrench, install the tail gate stay assembly LH.

Torque:

29 N·m {296 kgf·cm, 21 ft·lbf}

4. INSTALL TAIL GATE LOCK ASSEMBLY RH (w/ Power Tail Gate)

(a) Use the same procedure described for the LH side

5. INSTALL POWER TAIL GATE SENSOR ASSEMBLY RH (w/ Power Tail Gate)

(a) Use the same procedure described for the LH side

6. INSTALL TAIL GATE LOCK CYLINDER ASSEMBLY

7. INSTALL TAIL GATE LOCK ASSEMBLY LH (w/o Power Tail Gate)

(1) Using a T40 "TORX" socket wrench, install the tail gate lock assembly LH with the 2 screws.

Torque:

29 N·m {296 kgf·cm, 21 ft·lbf}

8. INSTALL TAIL GATE LOCK ASSEMBLY RH (w/o Power Tail Gate)

(1) Connect the connector.

(2) Using a T40 "TORX" socket wrench, install the tail gate lock assembly LH with the 2 screws.

Torque:

29 N·m {296 kgf·cm, 21 ft·lbf}

9. INSTALL TAIL GATE LOCK CONTROL LINK LH (w/o Power Tail Gate)

10. INSTALL TAIL GATE LOCK CONTROL LINK RH (w/o Power Tail Gate)
