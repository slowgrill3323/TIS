PROCEDURE

PROCEDURE

1. INSTALL FRONT NO. 1 SPEAKER ASSEMBLY

NOTICE:

Do not touch the cone of the front No. 1 speaker assembly

(1) Attach the guide and claw to temporarily install the front No. 1 speaker assembly.

(2) Install the front No. 1 speaker assembly with the 4 screws in the order shown in the illustration.

(3) Connect the connector.

2. INSTALL FRONT DOOR TRIM BOARD SUB-ASSEMBLY LH

3. INSTALL FRONT DOOR TRIM ORNAMENT SUB-ASSEMBLY LH

4. INSTALL FRONT DOOR LOWER FRAME BRACKET GARNISH LH

5. INSTALL POWER WINDOW REGULATOR SWITCH ASSEMBLY WITH FRONT DOOR ARMREST BASE PANEL (for Front Passenger Side)

6. INSTALL POWER WINDOW REGULATOR MASTER SWITCH ASSEMBLY WITH FRONT DOOR ARMREST BASE PANEL (for Driver Side)

7. INSTALL FRONT DOOR INSIDE HANDLE BEZEL PLUG LH

8. INSTALL REFLEX REFLECTOR (w/ Reflex Reflector Assembly)

9. INSTALL COURTESY LIGHT ASSEMBLY (w/ Courtesy Light)
