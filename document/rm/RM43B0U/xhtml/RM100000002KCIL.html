8. INSTALL FRONT DOOR CHECK ASSEMBLY LH

(1) Clean the bolt hole in the vehicle body.

(2) When reusing a bolt:

1. Clean the threads of the bolt with non-residue solvent.

2. Apply adhesive to the threads of the bolt.

Adhesive:

Toyota Genuine Adhesive 1324, Three Bond 1324 or equivalent.

(3) Install the front door check assembly with the bolt and 2 nuts.

Torque:

Bolt :

29 N·m {296 kgf·cm, 21 ft·lbf}

Nut :

8.5 N·m {87 kgf·cm, 75 in·lbf}

9. INSTALL FRONT DOOR OUTSIDE HANDLE FRAME SUB-ASSEMBLY LH

Install in this Direction

MP grease

(1) Apply MP grease to the sliding parts on the front door outside handle frame sub-assembly.

(2) Engage the guides to install the front door outside handle frame sub-assembly as shown in the illustration.

(3) Using a T30 "TORX" socket wrench, tighten the screw.

Torque:

4.0 N·m {41 kgf·cm, 35 in·lbf}

(4) Engage the clamp to install the wire harness.

15. INSTALL FRONT DOOR OUTSIDE HANDLE COVER (for Front Passenger Side)

(1) Engage the claws to install the front door outside handle cover.

(2) Using a T30 "TORX" socket wrench, tighten the screw.

Torque:

4.0 N·m {41 kgf·cm, 35 in·lbf}

(3) Install the hole plug.

16. INSTALL FRONT DOOR LOCK CYLINDER ASSEMBLY (for Driver Side)

(1) Install the front door lock cylinder assembly.

HINT:

Check that the front door lock cylinder assembly is inserted into the front door lock with motor assembly.

(2) Using a T30 "TORX" socket wrench, tighten the screw.

Torque:

4.0 N·m {41 kgf·cm, 35 in·lbf}

(3) Install the hole plug.

22. INSTALL FRONT DOOR GLASS SUB-ASSEMBLY LH

*A

for Driver Side

*B

for Front Passenger Side

(1) for Driver Side:

1. Connect the multiplex network master switch assembly connector.

(2) for Front Passenger Side:

1.Connect the power window regulator switch assembly connector.

(3) Connect the cable to the negative (-) auxiliary battery terminal.

  • for Gasoline Model :

    Click here

  • except Gasoline Model :

    Click here

(4) Turn the ignition switch to ON.

(5) Move the front door window regulator sub-assembly until the installation bolts can be seen.

(6) Temporarily install the front door glass sub-assembly to the front door window regulator assembly.

NOTICE:

Be careful not to damage the front door glass sub-assembly .

(7) Install the front door glass sub-assembly with the 2 bolts.

Torque:

8.0 N·m {82 kgf·cm, 71 in·lbf}

(8) Move the front door glass sub-assembly until the open position.

(9) Turn the ignition switch off.

(10) Disconnect the cable from the negative (-) auxiliary battery terminal.

  • for Gasoline Model :

    Click here

  • except Gasoline Model :

    Click here

CAUTION:

  • Wait at least 60 seconds after disconnecting the cable from the negative (-) auxiliary battery terminal to disable the SRS system.
  • If the airbag deploys for any reason, it may cause a serious accident.

(11) for Driver Side:

1. Disconnect the multiplex network master switch assembly connector.

(12) for Front Passenger Side:

1.Disconnect the power window regulator switch assembly connector.

(13) Install the hole plug.
