Power Tail Gate cannot be Operated Using Any Switch

DESCRIPTION

The power tail gate is controlled by the multiplex network door ECU and drives the tail gate motor.

If the power tail gate does not operate using any of the operations, a malfunction related to the power tail gate operation conditions or multiplex network door ECU are possible causes.

WIRING DIAGRAM

CAUTION / NOTICE / HINT

NOTICE:

  • First perform the communication function inspections in How to Proceed with Troubleshooting to confirm that there are no CAN communication malfunctions before troubleshooting this problem.

    Click here

  • Inspect fuses for circuits related to this system before performing the following procedure.
  • If the multiplex network door ECU has been replaced, or if any of the connectors has been disconnected, initialize the power tail gate system.

    Click here

  • If the main body ECU (multiplex network body ECU) is replaced, refer to registration.

    for Gasoline Model: Click here

    except Gasoline Model: Click here

  • If the 2-step unlock function is enabled via customize settings, only the driver door unlocks when the first unlock operation is performed and the power tail gate does not operate as all the doors are not unlocked.
