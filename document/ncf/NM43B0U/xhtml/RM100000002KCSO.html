SYSTEM CONTROL

(a) The welcome light illumination system operates as follows:

Condition

Details

System Activation

When all of the following conditions are met, the welcome light illumination system operates.

  • The ignition switch is off.
  • The light control switch is in the AUTO position.
  • The automatic light control sensor outputs the night mode signal.

Light Turn-on

The turn signal lights flash twice and then the parking lights, front side marker lights, taillights, rear side marker lights, and license plate lights turns on for 15 seconds when the door unlock operation is being used.

Light Turn-off

After the light is illuminated, the parking lights, front side marker lights, taillights, rear side marker lights, and license plate lights are turned off when either of the following conditions is met before 15 seconds have elapsed.

  • The ignition switch is ON.
  • The light control switch is in any position except the AUTO position.
  • Door lock is performed.

FUNCTION

(a) The welcome light illumination system operates the turn signal lights, parking lights, front side marker lights, taillights, rear side marker lights, and license plate lights in conjunction with the locking and unlocking of the doors, ensuring nighttime usability.

(b) The welcome light illumination control function can be turned on and off using the customization function. For details, refer to the Repair Manual.
