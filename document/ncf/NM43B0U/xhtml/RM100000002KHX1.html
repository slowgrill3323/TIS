MULTI-INFORMATION DISPLAY

CONSTRUCTION

  • The multi-information displays located on the center, left and right can be customized by operating the steering pad switch assembly.

    Multi-information Display

    Outline
    *1: Models with navigation system

    *2: 4WD models

    *3: HEV models

    LEFT/RIGHT

    The following information can be displayed:

    • Blank
    • Fuel Economy
    • Eco Score
    • Driving Support
    • Navigation*1
    • Audio
    • Drive Info
    • TRIP A
    • TRIP B
    • Tire Pressure
    • 4WD*2
    • Widget (when the meter layout is set to 0-dial or 1-dial)
    • Boost Gauge*3
    • MG Gauge*3
    • Traction Monitor Display*3

    HINT:

    The display of the various items can be turned off.

    CENTER

    The following information can be displayed:

    • Blank
    • Driving Support
    • Navigation*1
    • Settings
    • Message

(a) Fuel Economy Screen

  • Displays the fuel economy information.

    Current Fuel Economy

    Display Range

    0.0 to 30.0 km/L (0.0 to 30.0 L/100 km) (0.0 to 60.0 MPG)

    Minimum Unit

    0.1 km/L (L/100 km) (MPG)

    Function
    • Displays the instantaneous fuel economy calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal.
    • The displayed value is updated every 10 seconds.
    • When the value cannot be calculated, "－－.－" is displayed.

    Average Fuel Economy (After Start)

    Display Range

    0.0 to 99.9 km/L (L/100 km) (MPG)

    Minimum Unit

    0.1 km/L (L/100 km) (MPG)

    Function
    • Displays the average fuel consumption calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal after the engine system starts.
    • The displayed value is updated every 10 seconds.
    • When the value cannot be calculated, the display changes to blank.

    Average Fuel Economy (After Reset)

    Display Range

    0.0 to 99.9 km/L (L/100 km) (MPG)

    Minimum Unit

    0.1 km/L (L/100 km) (MPG)

    Function
    • Displays the average fuel consumption calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal.
    • The displayed value is updated every 10 seconds.
    • When the value cannot be calculated, the display changes to blank.

(b) Eco Score Screen

  • Displays the score of driving conditions (start, cruise and stop) rated out of 5 level.

    HINT:

    Evaluate each driving from start to stop.

    Start

    Bar graph to display Eco driving score when accelerating from stopped.

    Cruise

    Bar graph to display Eco driving score when cruising.

    Stop

    Bar graph to display Eco driving score when decelerating to stopped.
  • The eco accelerator guidance displays a bar graph to indicate the ideal acceleration for driving in an eco-friendly manner.

(c) Driving Support Screen

  • Displays the operating status of the following systems:
    • Dynamic Radar Cruise Control
    • Lane Departure Alert (LDA)
    • Lane Tracing Assist System (LTA)
    • Emergency Driving Stop System (EDSS)*1
    • Proactive Driving Assist (PDA)*2
      • *1: Models with EDSS (Emergency Driving Stop System)

        *2: Models with Proactive Driving Assist (PDA)

(d) Drive Info Screen

  • Displays the average vehicle speed and travel time after reset.

    Average Vehicle Speed (After Start)

    *1: HEV models

    *2: Gasoline models

    Display Range

    0 to 999 km/h (MPH)

    Minimum Unit

    1 km/h (MPH)

    Function
    • The average vehicle speed after starting the hybrid system*1 or engine*2 is calculated from vehicle speed signals and displayed.
    • When the value cannot be calculated, the display changes to blank.

    Travel Distance (After Start)

    *1: HEV models

    *2: Gasoline models

    Display Range

    0.0 to 6553.3 km (miles)

    Minimum Unit

    0.1 km (miles)

    Function
    • The travel distance is calculated from vehicle speed signals, and the distance traveled after starting the hybrid system*1 or engine*2 is displayed.
    • When the value cannot be calculated, the display changes to blank.

    Average Vehicle Speed (After Reset)

    Display Range

    0 to 999 km/h (MPH)

    Minimum Unit

    1 km/h (MPH)

    Function
    • The average vehicle speed after reset is calculated from vehicle speed signals and displayed.
    • When the value cannot be calculated, the display changes to blank.

    Travel Time (After Start)

    Display Range

    00:00 to 99:59

    Function
    • The time elapsed since starting the engine system is measured, and then integrated and displayed.
    • When the value cannot be calculated, the display changes to blank.

(e) TRIP A/B Screen

  • Displays the average vehicle speed, travel distance and travel time after reset.

    Average Vehicle Speed (Between Resets (Total))

    Display Range

    0 to 999 km/h (MPH)

    Minimum Unit

    1 km/h (MPH)

    Function
    • The average vehicle speed is calculated from vehicle speed signals and displayed.
    • When the value cannot be calculated, the display changes to blank.

    Travel Distance (Between Resets (Total))

    Display Range

    0.0 to 6553.3 km (miles)

    Minimum Unit

    0.1 km (miles)

    Function
    • The travel distance is calculated from vehicle speed signals and displayed.
    • When the value cannot be calculated, the display changes to blank.

    Travel Time (Between Resets (Total))

    *1: HEV models

    *2: Gasoline models

    Display Range

    00:00 to 99:59

    Function
    • The time elapsed since starting the hybrid system*1 or engine system*2 is measured and displayed.
    • When the value cannot be calculated, the display changes to blank.

(f) 4WD Screen (4WD Models)

  • Displays the 4WD driving force distribution in real time.

(g) Navigation Screen (Models with Navigation System)

  • Displays map and destination information linked with the navigation system.

(h) Widget

  • Displays the fuel economy information and audio status information when the meter layout is set to 0-dial or 1-dial.
