GENERAL

OUTLINE

(a) The active noise control system which reduces vehicle cabin noise and generates engine sounds has been adopted. The active noise control system consists of the Active Noise Control (ANC) and Engine Sound Enhancement (ESE) functions.

(b) The stereo component equalizer assembly controls the active noise control system and generates the sound of ESE function. The stereo component equalizer assembly outputs the generated sound to the audio system speakers.

HINT:

As the active noise control system is not directly linked to the audio system, sound will still be output even when the audio system is set to off or mute.

MAIN FEATURES

(a) ANC Function

(1) The ANC function uses microphones to detect the noise output from the engine, and generate and output an inverted phase audio signal (noise reducing sound) from the audio system speakers to reduce noise in the cabin.

(2) The ANC function uses microphones to detect the noise output from the engine, and generate and output an inverted phase audio signal (noise reducing sound) from the audio system speakers to reduce noise in the cabin.

(b) ESE Function

(1) The ESE function generates engine order component sound*1 based on the accelerator pedal depression amount, vehicle speed and cabin conditions, and outputs the sound via the audio system speakers to enhance the engine sounds*2.

HINT:

*1: Engine order component sound is the primary composition of frequencies of real engine sound and consists of the engine speed as the fundamental frequency, and multiple orders of harmonics. For example, if the engine speed creates a fundamental frequency of 100Hz, then the actual engine sound that is heard will consist of 100 Hz + 2 x 100 Hz + 3 x 100 Hz + ... + n x 100 Hz. These harmonics are called the 1st order, 2nd order, 3rd order... nth order harmonics.

*2: The ESE function calculates the engine order component sound of nth order, and enhances those harmonics with a synthesized sound.
