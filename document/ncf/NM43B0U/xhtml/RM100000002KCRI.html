SYSTEM CONTROL

(a) Operating Condition

(1) The daytime running lights illuminate when the following conditions are all met.

  • The ignition switch is ON.
  • The engine speed signal is input (engine is running).*1
  • The READY signal is input.*2
  • The light control switch is in the TAIL or AUTO position (and the taillights are not turned on via the automatic light control system).
  • The parking brake is released.
    • *1: Gasoline models
    • *2: HEV models

FUNCTION

(a) The daytime running light system is designed to automatically illuminate the Light Emitting Diode (LED) lights during the daytime to keep the vehicle highly visible to other vehicles.
