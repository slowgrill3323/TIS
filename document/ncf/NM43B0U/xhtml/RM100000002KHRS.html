Main Body ECU (Multiplex Network Body ECU)
  • Controls the theft deterrent system in accordance with the signal received from the switches, sensors and ECUs.
  • Outputs lock/unlock signals to each door lock assembly to operate the door lock motor and double lock motor.
  • Outputs horn sounding request signals and outputs exterior lights*1, interior lights*1, hazard warning light flash request signals when an attempted theft is detected.
  • When an unauthorized intrusion is detected at the time of the automatic alarm watch, outputs hazard warning light operation signals to the combination meter assembly.
  • When an unauthorized intrusion is detected at the time of the automatic alarm watch, outputs sounding signals to the, theft warning siren assembly, high pitched horn assembly and low pitched horn assembly.

Certification ECU (Smart Key ECU Assembly)
  • Judges and certifies the electrical key transmitter sub-assembly.
  • When a smart key system operation by the user is detected, sends vehicle exterior certification result signals to the main body ECU via CAN communication.
  • Sends wireless remote control signals input from the electrical key and TPMS receiver assembly to the main body ECU and power tail gate ECU*2 via CAN communication.
  • Outputs immobilizer set/unset commands.
  • Controls the security indicator light operation, power source change condition display (ACC, IG ON) and smart key indicator display.
  • Operates parts such as the IGR relay and ACC relay and controls the power source change condition.
  • Sends engine start permission signals to the ECM.*5
  • Sends hybrid system READY permission signals to the hybrid vehicle control ECU assembly.*6
  • Outputs lock/unlock commands for the steering lock ECU.*7
  • Receives unlock signals from the tail gate opener switch assembly and elbow switch (luggage electrical key switch), and sends them to the body ECU.*2

Power Tail Gate ECU (Multiplex Network Door ECU)*2
  • Receives power tail gate operation request signals from the certification ECU to operate the tail gate lock motor.
  • Determines the operation condition of the power tail gate system from the ON/OFF signals of each switch built into the tail gate lock with courtesy light switch assembly and sends signals to the main body ECU via CAN communication.

FUNCTION

(a) The theft deterrent system has the following functions:

Function

Operation
*1: Models for U.S.A.

*2: Except models for U.S.A.

Alarm

When the theft deterrent system switches to the alarm state, the warning items are operated and the alarm sounds for approximately 57 seconds*1 or 27.5 seconds*2.

Alarm Indicator

When the theft deterrent system switches to the armed preparation state or alarm state, the security indicator light is illuminated to inform the user. This indicator also serves as the immobiliser indicator, which blinks when the immobiliser is in the set state.

Forced Door Lock

The main body ECU transmits a door lock signal to all the doors when all of the following conditions are met:

  • The theft deterrent system is in the alarm state.
  • One of the doors has been unlocked.
  • There is no electrical key transmitter sub-assembly in the actuation area.

The function stops when one of the following conditions is met:

  • All doors are locked.
  • The warning ends after approximately 57 seconds*1 or 27.5 seconds*2.
  • The electrical key transmitter sub-assembly is in the actuation area.

Panic Control

When the panic button of the electrical key transmitter sub-assembly is pressed, the main body ECU operates the alarm function.

Security indicator light

Informs driver of theft deterrent system status.

SYSTEM CONTROL

(a) Models for U.S.A.

(1) The theft deterrent system automatically determines the vehicle condition based on factors such as the condition of the electrical key transmitter sub-assembly, engine hood, side doors and tail gate. As shown in the figure below, theft deterrent system setting/releasing and warnings are controlled based on various conditions. The control status is indicated by flashing or turning the security indicator light on or off.

State

Description

Security Indicator Light

Disarmed State

The theft deterrent system has not been set by the user.

Immobiliser System (Set): Flash

Immobiliser System (Not Set): Off

Arming Preparation State

The user has locked all doors using the wireless remote door lock control system, etc., and the theft deterrent system is preparing to set. If the vehicle is not operated (such as unlocking a door), the system will switch automatically to the armed state after approximately 30 seconds.

Illuminate

Armed State

The theft deterrent system is active. (Theft can be detected.)

Flash

Pre Alarm Sounding State

The door is unlocked with a mechanical key and a small warning sound is being emitted.

Illuminate

Alarm Sounding State

The system has determined that the vehicle may have been stolen, and intermittently sounds the theft warning siren, intermittently sounds the vehicle horn, flashes the hazard warning lights, flashes the exterior lights and illuminates the interior lights for approximately 57 seconds.

Illuminate

Transition Condition

Condition

Item

[1]

One of the following conditions have been met:

  • All doors are locked using the wireless door lock control system or smart key system.
  • The mechanical key is inserted into the driver seat door key cylinder and turned, and all doors are locked.
  • With any door open, all doors are closed after all doors are locked.

[2]

One of the following conditions have been met:

  • All doors are unlocked using the wireless door lock control system or smart key system.

  • The engine switch*1 or power switch*2 is pressed.
  • The auxiliary battery is reconnected.

[4]

  • The closed engine hood is opened.*3
  • With all doors closed, any door is opened.
  • With all doors locked, any door is unlocked.
  • Either the engine hood*3 or a door is opened, or the auxiliary battery is reconnected with any door unlocked.
  • The power source mode switches from off to ON
  • The engine is started.*1*4
  • The hybrid system is started.*2*5

[8]

The mechanical key is inserted into the driver seat door key cylinder and turned, and the door is unlocked.

[9]

The pre alarm sounding state time elapses (approximately 10 seconds).

[10]

  • All doors are locked using the wireless door lock control system or smart key system.
  • The mechanical key is inserted into the driver seat door key cylinder and turned, and all doors are locked.
  • With any door open, all doors are closed after all doors are locked.

HINT:

The theft deterrent system will begin arming and will provide a warning for theft conditions other than the engine hood, even if the wireless door lock control system, etc. is used to lock a door with the engine hood open.
