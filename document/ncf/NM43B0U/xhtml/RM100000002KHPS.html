Title: THEFT DETERRENT / KEYLESS ENTRY: DIGITAL KEY SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [03/2024 -        ]

GENERAL

OUTLINE

HINT:

  • A contract is required before the digital key system can be used. Furthermore, various functions may be limited depending on the model of smartphone.
  • Make sure to operate the digital key system while carrying a smart device.

(a) By using a smartphone with the dedicated application installed, the vehicle can be locked or unlocked remotely via "Bluetooth" communication. Also, if the smartphone is placed near the console box after entering the vehicle, the engine*1 or hybrid system*2 can be started using the push start button. Furthermore, a key used by the smartphone application can be shared with family, friends, colleagues, etc.

  • *1: Gasoline models
  • *2: HEV models

(b) The vehicle can be used with just a smartphone, without carrying the electrical key transmitter sub-assembly. Also, a key can be provided or deleted without lending a physical key.

NOTICE:

  • Before using the digital key system, refer to the owner's manual for the method of use and make sure to observe all precautions.
  • The digital key system may not be able to be used in situations such as the following. (For details, refer to https://toyota.jp/digital_key/) In this case, it is recommended to carry the electrical key transmitter sub-assembly.
    • When the smartphone is malfunctioning
    • When the smartphone and server cannot communicate
    • When the smartphone battery is depleted
    • When the service is temporarily stopped due to maintenance, etc.

(c) It is possible to perform operations when the smartphone and vehicle are connected via BLE ("Bluetooth" low energy). The BLE connection status can be checked using the application screen.

(d) The smart device (owner key) which has no restrictions is for the owner of the vehicle, and only one smart device (owner key) can be registered to and used for each vehicle. Furthermore, the owner key can issue a maximum of 7 share keys.

(e) When using a second smart device for the owner or user other than the owner, a share key can be registered.

(f) Anybody other than the owner of the vehicle can use a share key.

(g) When the vehicle is to be serviced, it is possible for the owner key to register a share key for the service employee.

PRECAUTION

(a) Make sure to use this system in an area where the smartphone has an internet connection. A notification is provided to the user's smartphone if the system is used offline.

(b) When handling a smart device, make sure to follow the user manual for each device maker.

(c) As the cabin temperature may become extremely high, it is dangerous to leave a smart device in the vehicle.

(d) Take sufficient care not to leave the smart device in the vehicle due to the risk of theft.

(e) The digital key system cannot be used when the battery of the smart device is depleted. Make sure to check the battery charge level of the smart device when using the digital key system. When going to an area where charging is not convenient, make sure to also carry the electrical key transmitter sub-assembly.

(f) Never operate a smart device when driving the vehicle.

(g) When taking the vehicle to a dealer due to a malfunction of the digital key system, etc., make sure to bring the electrical key transmitter sub-assembly.

(h) Some functions may be limited depending on the model of smart device.

(i) Make sure to refer to the compatible smart device list before using a smart device as a digital key.

(j) It is possible to use the digital key system when the OS is Android 6.0 or iOS 9.0 or higher, and the "Bluetooth" version is 4.2 or higher.

(k) Make sure to use the latest version of the application on the smart device.

(l) The application may stop working due to maintenance of the digital key system.

(m) The digital key application may sometimes connect to the center. It is necessary for the customer to incur the fees for all data communication performed.

(n) If a smart device has been lost, the risk of vehicle theft increases significantly. Contact your dealer immediately with the electrical key transmitter sub-assembly.

(o) When the keyless entry function cannot be operated using a digital key, use the following methods to lock/unlock the doors.

  • Use the wireless function of the digital key.
  • Use the electrical key transmitter sub-assembly.
  • Operate the smart device near a door pillar.
