Title: BRAKE SYSTEM (OTHER): BRAKE SYSTEM (for Gasoline Model): GENERAL; 2024 - 2026 MY Tacoma [12/2023 -        ]

GENERAL

OUTLINE

(a) A tandem plunger type brake master cylinder is used.

(b) A single type brake booster is used.

(1) A vacuum sensor assembly is used.*1

(2) A vacuum warning switch assembly is used.*2

HINT:

*1: Models with stop and start system

*2: Models without stop and start system

Type

Tandem Plunger

Diameter

23.81 mm (0.94 in.)

Brake Booster Assembly

Type

Diameter

10.5 in.
