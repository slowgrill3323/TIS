POWER TAIL GATE CONTROL

FUNCTION OF MAIN COMPONENTS

Component

Function

Power Tail Gate ECU (Multiplex Network Door ECU)
  • Controls the power tail gate system in accordance with the signals received from the switches, sensors and ECUs.
  • Receives the signals from the Hall IC to detect the tail gate position and the opening and closing speed of the tail gate.
  • Informs the user by sounding the power tail gate warning buzzer and flashing the hazard warning light when each operation switch is pressed.

Power Tail Gate Unit Assembly RH

Tail Gate Drive Motor

Opens and closes the tail gate when actuated by the power tail gate ECU (multiplex network door ECU).

Hall IC

Detects the operation conditions of the tail gate and outputs a pulse signal to the power tail gate ECU (multiplex network door ECU).

Tail Gate Lock Assembly RH/LH

Tail Gate Lock Motor

Latches and unlatches the tail gate when actuated by the power tail gate ECU (multiplex network door ECU).

Latch Switch

Detects if the tail gate is ajar.

Pawl Switch

Detects whether the ratchet condition is released or engaged.

Initial Switch

Detects the initial (neutral) position of the driven gear of the tail gate lock motor. This switch turns on when the driven gear is in the neutral position and turns off when the driven gear is in any position other than neutral.

Tail Gate Courtesy Switch

Detects whether the tail gate is open or closed. This switch turns on when the tail gate is open or ajar and turns off when the tail gate is closed.

Tail Gate Handle Switch (Tail Gate Opener Switch Assembly) or Elbow Switch (Luggage Electrical Key Switch)
  • The tail gate opens and closes when the tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) on outside of the vehicle is turn on.
  • Outputs signals to the certification ECU (smart key ECU assembly).

Power Tail Gate Control Switch (Tail Gate Opener Switch)
  • The tail gate opens and closes when the power tail gate control switch (tail gate opener switch) is turned on.
  • Outputs signals to the power tail gate ECU (multiplex network door ECU).

Touch Sensor (Power Tail Gate Sensor Assembly LH/RH)

Detects if an object is jammed in the tail gate during the closing operation of the tail gate.

Power Tail Gate Warning Buzzer

When the power tail gate system starts to operate, the power tail gate warning buzzer sound to inform the user that the system is operating.

Combination Meter Assembly
  • Receives a signal to blink the hazard warning light.
  • Outputs a customization signal to the main body ECU (multiplex network body ECU).

Electrical Key and TPMS Receiver Assembly

Receives the power tail gate button signal and transmits it to the certification ECU (smart key ECU assembly).

Certification ECU (Smart Key ECU Assembly)

Receives the power tail gate button signal and transmits it to the main body ECU (multiplex network body ECU).

Main Body ECU (Multiplex Network Body ECU)

Sends a tail gate lock/unlock control signal, vehicle speed signal and shift position P signal to the power tail gate ECU (multiplex network door ECU) via the Controller Area Network (CAN).

Electrical Key Transmitter Sub-assembly

  • Power Tail Gate Button

Sends the power tail gate button signal to the electrical key and TPMS receiver assembly.

SYSTEM CONTROL

(a) The power tail gate opener function operates when all of the following conditions are met:

(1) The power tail gate system is on.

(2) The tail gate is unlocked, or enter the detection area holding the electrical key transmitter sub-assembly.

(3) The vehicle speed is approximately 3 km/h (2 mph) or less (only when the ignition switch is ON).

(4) The brake pedal is depressed, the shift position is in P* or the parking brake is on (only when the ignition switch is ON).

HINT:

*: Models with automatic transmission

(b) The power tail gate closer function operates when all of the following conditions are met:*

HINT:

The electrical key transmitter sub-assembly does not work while ignition switch is ON.

(1) The power tail gate system is on.

(2) The vehicle speed is approximately 3 km/h (2 mph) or less (only when the ignition switch is ON).

(3) The brake pedal is depressed, the shift position is in P* or the parking brake is on (only when the ignition switch is ON).

HINT:

*: Models with automatic transmission

(c) Opener Function 1 - Unsupervised/Remote Operation (Power Tail Gate Control Switch (Tail Gate Opener Switch) or Power Tail Gate Button (Electrical Key Transmitter Sub-assembly))

(1) When the power tail gate control switch (tail gate opener switch) or the power tail gate button of the electrical key transmitter sub-assembly is operated, the opener function operates as follows:

1

After the power tail gate control switch (tail gate opener switch) or the power tail gate button of the electrical key transmitter sub-assembly is pressed for approximately 0.8 seconds, the power tail gate warning buzzer sounds for 0.8 seconds.

2

The tail gate lock assembly (tail gate lock motor) releases the latch.

3

The tail gate courtesy switch is turned on and the motor in the power tail gate unit assembly RH set starts to operate.

4

The tail gate lock motor is returned to the initial position and the latch switch is turned on.

5

After the power tail gate ECU (multiplex network door ECU) detects a tail gate stop position pulse, the motor in the power tail gate unit assembly RH set is stopped.

(d) Opener Function 2 - Supervised Opener Function (Tail Gate Handle Switch (Tail Gate Opener Switch Assembly) or Elbow Switch (Luggage Electrical Key Switch))

(1) When the tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) is operated, the opener function operates as follows:

1

After the tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) is pressed, the power tail gate warning buzzer sounds for 0.8 seconds.

2

After the power tail gate warning buzzer sounds, the tail gate lock assembly releases the latch.

3

The tail gate courtesy switch is turned on and the motor in the power tail gate unit assembly RH set starts to operate.

4

The tail gate lock motor is returned to the initial position and the latch switch is turned on.

5

After the power tail gate ECU (multiplex network door ECU) detects a tail gate stop position pulse, the motor in the power tail gate unit assembly RH set is stopped.

(e) Opener Function 3 (Operate Switch while Tail Gate is Moving)

(1) When the power tail gate control switch (tail gate opener switch), tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or power tail gate button of the electrical key transmitter sub-assembly is operated, the opener function operates as follows:

1

When the power tail gate control switch (tail gate opener switch), tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or power tail gate button of the electrical key transmitter sub-assembly is pressed while the tail gate is moving (either opening or closing), the power tail gate warning buzzer sounds for approximately 0.2 seconds and the tail gate reverses direction.

2

When the power tail gate control switch (tail gate opener switch), tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or power tail gate button of the electrical key transmitter sub-assembly is operated a second time the power tail gate warning buzzer sounds for approximately 0.2 seconds (again) and the tail gate reverses direction a second time.

3

If any of those switches is pressed a third time, the power tail gate stops (maximum number of reversal times = 2).

4

After this stop the power tail gate system has entered manual mode and to resume power operation it is necessary to reinitialize the power tail gate.

HINT:

  • Manually close power tail gate by hand to reinitialize and resume power operation.
  • Note that all switch inputs while the tail gate is moving are short hold (to 0.1 second).

(f) Closer Function 1 - Unsupervised/Remote Operation (Power Tail Gate Control Switch (Tail Gate Opener Switch) or Power Tail Gate Button (Electrical Key Transmitter Sub-assembly))

(1) When the power tail gate control switch (tail gate opener switch) or the power tail gate button of the electrical key transmitter sub-assembly is operated, the closer function operates as follows:

1

After the power tail gate control switch (tail gate opener switch) or the power tail gate button of the electrical key transmitter sub-assembly is pressed for approximately 0.8 seconds, the power tail gate warning buzzer sounds for 0.8 seconds.

2

The motor in the power tail gate unit assembly RH set starts to operate.

3

When the latch switch is turned off, the voltage of the motor is reduced.

4

The tail gate lock assembly performs the closing operation 1 second after the latch switch is turned off, and stops the operation when the tail gate courtesy switch is turned off.

(g) Closer Function 2 - Supervised Operation (Tail Gate Handle Switch (Tail Gate Opener Switch Assembly) or Elbow Switch (Luggage Electrical Key Switch))

(1) When the tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) is operated, the closer function operates as follows:

1

When the tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) is turned on, the power tail gate warning buzzer sounds for 0.8 seconds.

2

The motor in the power tail gate unit assembly RH starts to operate.

3

When the latch switch is turned off, the voltage of the motor is reduced.

4

The tail gate lock assembly performs the closing operation 1 second after the latch switch is turned off, and stops the operation when the tail gate courtesy switch is turned off.

(h) Closer Function 3 - Close Assist

(1) When the tail gate is lifted about 20 mm (0.8 in.), the closer function operates as follows:

1

When the tail gate is lifted about 20 mm (0.8 in.), the power tail gate warning buzzer sounds for approximately 1.0 second.

2

The motor in the power tail gate unit assembly RH starts to operate.

3

When the latch switch is turned off, the voltage of the motor is reduced.

4

The tail gate lock assembly performs the closing operation 1.0 second after the latch switch is turned off, and stops the operation when the tail gate courtesy switch is turned off.

HINT:

The power tail gate has 2 tail gate lock motor. Closing operation is complete after both pawl switch is off.

FUNCTION

(a) The power tail gate system has the following functions:

Function

Outline

Opener Function

Upon receiving an on signal from the power tail gate control switch (tail gate opener switch), the tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or the power tail gate button on the electrical key transmitter sub-assembly with the tail gate unlocked, the power tail gate ECU (multiplex network door ECU) actuates the tail gate lock motors and the power tail gate unit assembly RH in order to release the latch of the tail gate and open the tail gate.

Closer Function

Upon receiving an on signal from the power tail gate control switch (tail gate opener switch), the tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or the power tail gate button on the electrical key transmitter sub-assembly with the tail gate open, the power tail gate ECU (multiplex network door ECU) actuates the power tail gate unit assembly RH and the tail gate lock motors in order to close the tail gate and engage the latch of the tail gate lock.

Close Assist Power Tail Gate Close Function

When the tail gate is open, the power tail gate ECU (multiplex network door ECU) closes the tail gate and engages the tail gate latch by operating the tail gate lock motor and power tail gate unit assembly RH when the tail gate is lifted about 20 mm (0.8 in.).

Jam Protection Function

Upon receiving signals from a touch sensor (power tail gate sensor assembly) or the Hall ICs of the power tail gate unit assembly RH, when the power tail gate ECU (multiplex network door ECU) detects that an object is jammed in the tail gate, the power tail gate operation is stopped. After a jam is detected, the tail gate must be manually closed to resume powered operation. The power tail gate warning buzzer sounds twice when a jammed object is detected.

Tail gate Open/Close Speed Control Function

In accordance with the signals received from the Hall ICs, the power tail gate ECU (multiplex network door ECU) controls the opening and closing speeds of the tail gate.

Suspending Control Function

By conducting any of the following operations during the opening or closing operation of the power tail gate, the operation will be canceled:

  • The power tail gate system is turned off.
  • The power tail gate ECU (multiplex network door ECU) detects that an object is jammed in the tail gate.

Other Input During Operation

The power tail gate ECU (multiplex network door ECU) detects that an object is jammed in the tail gate.

  • The power tail gate button on the electrical key transmitter sub-assembly is pressed.
  • The tail gate handle switch (tail gate opener switch assembly) or elbow switch (luggage electrical key switch) is pressed.
  • The power tail gate control switch (tail gate opener switch) is pressed.

Close Assist Function

When the tail gate is open and then manually operated in the closing direction more than 20 mm (0.8 in.), the power tail gate ECU (multiplex network door ECU) drives the power tail gate unit assembly RH set to operate the tail gate until it is fully closed.

Customization Function

By using the multi-information display, the following items can be customized.

  • On/Off setting of the power tail gate system.

HINT:

If the power supply is stopped such as when the cable is disconnected from the negative (-) terminal of auxiliary battery, initialization is required. If you attempt to operate the power tail gate system when initialization is required, a message is sent from the power tail gate ECU (multiplex network door ECU) to the display audio.
