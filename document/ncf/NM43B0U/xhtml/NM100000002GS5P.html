STEERING LOCK CONTROL

FUNCTION OF MAIN COMPONENTS

(a) The main components of the steering lock control system are as follows:

Component

Function

Steering Lock (with ECU) Actuator Assembly

Steering Lock ECU

Receives the steering wheel unlock/lock request signal from the certification ECU (smart key ECU assembly) and activates the steering lock motor.

Certification ECU (Smart Key ECU Assembly)
  • Certifies the electrical key transmitter sub-assembly.
  • Transmits each power supply signal for the lock/unlock motor to the steering lock ECU (steering lock (with ECU) actuator assembly).

ID Code Box (Immobiliser Code ECU)

Sends each set/unset request signal to the steering lock ECU (steering lock (with ECU) actuator assembly).

Ignition Switch (Push Start Switch)

Transmits the ignition switch signal to the certification ECU (smart key ECU assembly).

Combination Meter Assembly

Buzzer

Sounds to warn the driver of a malfunction in the system.

Multi-information Display

Displays the warning message to inform the driver of a malfunction in the electrical steering lock system.

Central Gateway ECU (Network Gateway ECU)

Relays data between buses.

FUNCTION

(a) The steering lock (with ECU) actuator assembly is integrated into the steering lock ECU, and the ECU controls the lock bar operation in the steering lock (with ECU) actuator assembly through the control of the steering lock motor.

(b) The steering lock ECU (steering lock (with ECU) actuator assembly) detects the position (lock/unlock) of the lock bar and transmits this information to the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU).

(c) In this system, the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU) determine whether to lock and unlock the steering wheel. Then, the certification ECU (smart key ECU assembly) sends lock or unlock command signals to the steering lock ECU (steering lock (with ECU) actuator assembly) via LIN communication. Upon receiving the signals, the steering lock ECU (steering lock (with ECU) actuator assembly) operates the steering lock motor to lock or unlock the steering.

HINT:

It is not possible to replace only the steering lock ECU in the steering lock (with ECU) actuator assembly. Therefore, if the ECU malfunctions, the entire steering lock (with ECU) actuator assembly must be replaced.
