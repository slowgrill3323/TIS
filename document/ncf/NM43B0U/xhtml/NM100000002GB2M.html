Title: CLUTCH: CLUTCH SYSTEM: GENERAL; 2024 - 2026 MY Tacoma [12/2023 -        ]

GENERAL

OUTLINE

(a) A dry type single plate clutch, which is operated by hydraulic pressure, is used.

(b) A low-peak effort clutch type clutch cover is used.

(c) The intelligent Manual Transmission(iMT) system is used. An integrated clutch stroke sensor type master cylinder assembly is used.

(d) A clutch pedal with turnover spring is used.

(e) A clutch start system is used.
