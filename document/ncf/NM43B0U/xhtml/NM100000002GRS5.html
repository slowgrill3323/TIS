MULTI-INFORMATION DISPLAY

CONSTRUCTION

  • The information currently displayed can be switched by operating the steering pad switch assembly.
  • The following information item can be displayed on the multi-information display by switching the tabs:

    Tab

    Item
    *: Models with RSA

    Fuel Economy Information

    The following items can be displayed:

    • Fuel Consumption Graph
      • Drivable Distance
      • Current Fuel Consumption
      • Average Fuel Consumption
        1. After Start
        2. After Reset
        3. After Refuel
    • ECO Drive Indicator Zone Display
      • Eco Driving Indicator Zone Display
      • Drivable Distance

    Driving Support System Information

    The following items can be displayed:

    • Dynamic Radar Cruise Control Information
    • Lane Departure Alert (LDA) Information
    • Lane Tracing Assist (LTA) Information
    • Road Sign Assist (RSA) Information*

    Vehicle Information

    The following items can be displayed:

    • Drive Information
      • Trip (After Start)
        1. Average Vehicle Speed
        2. Travel Distance
        3. Travel Time
      • Total (After Reset)
        1. Average Vehicle Speed
        2. Travel Time
    • Tire Inflation Pressure Display

    Settings

    The following settings can be changed:

    • Clock Setting
    • Rheostat
    • Lane Departure Alert (LDA) System
    • Blind Spot Monitor (BSM) System
    • Pre-Collision System (PCS)
    • Intuitive Parking Assist System
    • Rear Cross Traffic Alert (RCTA) Function
    • Road Sign Assist (RSA) System*
    • Vehicle Settings
    • Meter Settings

    Warning Message

    Warning messages about vehicle malfunctions are displayed.

(a) Fuel Economy Information Tab

(1) Displays the following information item:

  1. Drivable Distance
    • Displays the drivable distance information.

      Drivable Distance

      Display Range

      1 to 9999 km (miles)

      Minimum Unit

      1 km (mile)

      Function
      • From the vehicle speed signal, fuel consumption signal, etc., the fuel economy is learned*1, the estimated distance to empty is calculated*2 from the current amount of fuel remaining signal and is displayed.
      • The displayed value is updated every second.
      • If the value cannot be calculated, the display will be blank.

      HINT:

      • *1: Each time refueling judgment is made, the learned fuel economy is updated based on the fuel economy before refueling, the amount used since the previous refueling and the average fuel economy after refueling.
      • *2: If refueling is performed, arithmetic processing is performed based on the value effective at the time when refueling judgment is made normally. (Refueling judgment amount: 15 liters (4 gallon) or more)
  2. Fuel Consumption Graph
    • Displays the Fuel Consumption information.

      Current Fuel Consumption

      Display Range

      0.0 to 30.0 km/L (0.0 to 30.0 L/100 km) (0.0 to 60.0 MPG)

      Minimum Unit

      0.1 km/L (L/100 km) (MPG)

      Function
      • Displays the instantaneous fuel consumption calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal.
      • The displayed value is updated every 10 seconds.
      • When the value cannot be calculated, "－－.－" is displayed.

      Average Fuel Consumption (After Start)

      Display Range

      0.0 to 99.9 km/L (L/100 km) (MPG)

      Minimum Unit

      0.1 km/L (L/100 km) (MPG)

      Function
      • Displays the average fuel consumption calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal after the engine starts.
      • The displayed value is updated every 10 seconds.
      • When the value cannot be calculated, the display changes to blank.

      Average Fuel Consumption (After Reset)

      Display Range

      0.0 to 99.9 km/L (L/100 km) (MPG)

      Minimum Unit

      0.1 km/L (L/100 km) (MPG)

      Function
      • Displays the average fuel consumption calculated from the driving distance (calculated based on the vehicle speed signal) and fuel consumption signal.
      • The displayed value is updated every 10 seconds.
      • When the value cannot be calculated, the display changes to blank.

      Average Fuel Consumption (After Refuel)

      Display Range

      0.0 to 99.9 km/L (L/100 km) (MPG)

      Minimum Unit

      0.1 km/L (L/100 km) (MPG)

      Function
      • Displays the average fuel consumption calculated from the driving distance from the previous refueling (calculated based on the vehicle speed signal) and fuel consumption signal.
      • The displayed value is updated every 10 seconds.
      • When the value cannot be calculated, the display changes to blank.
  3. ECO Driving Indicator Zone Display
    • When the ECM, using information (such as vehicle speed, accelerator pedal depressed volume, engine speed signal (engine r/min data), and so on), calculates that the vehicle is driven in an Eco-friendly manner, it turns on Eco driving indicator and shows the bar graph of ECO driving indicator zone display within zone of Eco driving.
    • The following table shows the operation of Eco driving indicator and ECO driving indicator zone display in different driving conditions:

      Driving Condition

      Eco Driving Indicator Light

      Eco Driving Indicator Zone Display

      While Stopped

      Turns off

      While Eco Driving

      Turns on

      While Non-eco Driving

      Turns off

      When any of the following conditions is met, operation is stopped.

      • Engine switch off
      • Engine stopped
      • Shift lever is not in D.
      • SPORT mode is selected.
      • The vehicle is being driven at a speed of approximately 130 km/h or higher.

      Turns off

(b) Driving Support System Information Tab

(1) Displays the operating status of the following systems:

  • Dynamic Radar Cruise Control
  • Lane Departure Alert (LDA)
  • Lane Tracing Assist System (LTA)
  • Road Sign Assist (RSA) System*
    • *: Models with RSA

(c) Vehicle Information Tab

(1) Displays the following information item:

  1. Drive Information
    • Displays the average vehicle speed and travel time after reset.

      Average Vehicle Speed (After Start)

      Display Range

      0 to 999 km/h (MPH)

      Minimum Unit

      1 km/h (MPH)

      Function
      • The average vehicle speed after starting the engine is calculated from vehicle speed signals and displayed.
      • When the value cannot be calculated, the display changes to blank.

      Travel Distance (After Start)

      Display Range

      0.0 to 6553.3 km (miles)

      Minimum Unit

      0.1 km (miles)

      Function
      • The travel distance is calculated from vehicle speed signals, and the distance traveled after starting the engine is displayed.
      • When the value cannot be calculated, the display changes to blank.

      Travel Time (After Start)

      Display Range

      00:00 to 99:59

      Function
      • The time elapsed since starting the engine is measured, and then integrated and displayed.
      • When the value cannot be calculated, the display changes to blank.

      Average Vehicle Speed (After Reset)

      Display Range

      0 to 999 km/h (MPH)

      Minimum Unit

      1 km/h (MPH)

      Function
      • The average vehicle speed after reset is calculated from vehicle speed signals and displayed.
      • When the value cannot be calculated, the display changes to blank.

      Travel Time (After Reset)

      Display Range

      00:00 to 99:59

      Function
      • The time elapsed after reset is measured and displayed.
      • When the value cannot be calculated, the display changes to blank.

(d) Settings Tab

(1) The items displayed on the multi-information display can be customized.

(e) Warning Message Tab

(1) Warning messages about vehicle malfunctions are displayed.
