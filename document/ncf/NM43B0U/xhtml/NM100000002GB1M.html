Title: DOOR LOCK: POWER DOOR LOCK CONTROL SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [12/2023 -        ]

GENERAL

OUTLINE

(a) A power door lock control system, which uses a door lock motor to lock and unlock the doors, is used.

(b) This system is controlled by the main body ECU.

(c) The main body ECU outputs signals to the respective door lock motors.

(d) A 1-motion open operation, which simultaneously unlocks and opens the door when the inside handle is pulled, is provided for the driver door and front passenger door for when the doors are opened from inside the vehicle.

(e) The customization function of the GTS or radio and display receiver assembly makes it possible to adjust and change the settings for each function.

(f) For the components and functions described in this section, remote operation of remote services are available. For details, see the TELEMATICS SYSTEM section.
