STEREO JACK ADAPTER ASSEMBLY

CONSTRUCTION

(a) A USB (type-C) port that can be used as a power supply or connection of an external audio device is used for the No. 1 stereo jack adapter assembly.

HINT:

When audio files have been stored on an USB Memory using a computer, hidden files may also be stored.

If hidden files are stored, the system may not be able to correctly play back an audio file.
