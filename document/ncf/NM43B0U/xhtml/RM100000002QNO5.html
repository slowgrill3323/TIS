FUNCTION

(a) The power door lock control system has the following functions:

Function*1

Outline

Manual Unlock

This function unlocks all the doors when the door control switch is operated to the unlock side.

Manual Lock

This function locks all the doors when the door control switch is operated to the lock side.

Key-linked Unlock*2

This function, which is linked with the door lock key cylinder, can unlock all the doors when an unlock operation is performed.

Key-linked Lock

This function, which is linked with the door lock key cylinder, can lock all the doors when a lock operation is performed.

Key-linked 2-step Unlock*2

This function is provided to unlock the driver door when the mechanical key is turned in the door lock key cylinder the first time and to unlock the remaining doors when it is turned the second time within 5 seconds of the first time.

Collision Door Lock Release

If the SRS airbag system has been activated, and the main body ECU has detected the activation signal output by the airbag ECU assembly, the main body ECU unlocks all the doors.

Manual Unlock Prohibition

When the doors are all closed and are locked using a key-linked lock operation, wireless door lock operation or entry lock operation, unlocking the vehicle using the door control switch is prohibited.

Driver Door Opening-linked Automatic Door Unlock

When the driver door is opened within 45seconds of the ignition switch being turned from ON to ACC or off, all of the doors will be automatically unlocked.

Shift-linked Automatic Door Unlock*3

When the ignition switch is ON and the shift lever is moved to P position from a position other than P, all of the doors will be automatically unlocked.

Shift-linked Automatic Door Lock

When the conditions listed below are met, this function causes all the doors to be automatically locked:

  • The ignition switch is changed from ACC or off to ON.
  • All the doors are closed.
  • The shift lever is moved out of P position.*3
  • The shift position is moved out of N position.*4
  • Any of the doors are in an unlocked state.

Speed Sensitive Automatic Door Lock

When the conditions listed below are met, this function causes all the doors to be automatically locked:

  • Vehicle speed is higher than approximately 20 km/h (12 mph).
  • All the doors are closed.
  • Any of the doors is unlocked.

Key Confinement Prevention

When the electrical key transmitter sub-assembly is inserted in the interior actuation area and the door lock operation is performed while the driver door is open, all the doors will be unlocked.

Customization

Turning on or off of some functions or settings can be performed using the GTS or radio and display receiver assembly. In addition, some functions are not available depending on the specification. For details, refer to the Repair Manual.

HINT:

*1: The default setting differs depending on the model.

*2: The function can be set to unlock all the doors after 1 or 2 key-linked unlock operations for the driver side by using the customization function.

*3: Models with automatic transmission

*4: Models with manual transmission

DIAGNOSIS

(a) If there is a malfunction in the system, the main body ECU stores the Diagnostic Trouble Codes (DTCs) in its memory. For details, refer to the Repair Manual.
