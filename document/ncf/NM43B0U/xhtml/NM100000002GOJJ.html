Title: BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS: TRAILER BRAKE CONTROL SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [12/2023 -        ]

GENERAL

GENERAL

(a) A trailer brake control system is used.

(b) The trailer brake control system uses a trailer brake control ECU to control the trailer brake from the vehicle.

(c) The trailer brake control system performs optimal control by using the appropriate gain (amount of control) and trailer brake type settings.

(d) The state of the system settings is displayed on the multi-information display in the combination meter assembly.

PRECAUTION

(a) When towing a trailer, turn the intuitive parking assist system off.

(b) Before using the trailer brake control system, check that the trailer brake operates normally.

(c) Before using the trailer brake control system, check the compatibility between the trailer brake control system and the trailer brake.

(d) Set the appropriate gain setting to optimally control the trailer brake. When the gain is set to "0", the trailer brake is not controlled. Also, it is necessary to adjust the gain setting when the vehicle load, trailer load, road condition or weather changes.

(e) The trailer brake control amount may decrease in the following condition:

(1) When the Anti-lock Brake System (ABS) function operates and the vehicle decelerates, the trailer brake control amount is reduced as there is a possibility that the trailer may sway and the wheels may lock.
