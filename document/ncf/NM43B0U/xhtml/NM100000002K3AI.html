SPEED SENSOR

CONSTRUCTION

(a) An active type speed sensor is used. This sensor contains a sensor IC.

(b) The magnet type sensor rotor, which consists of N and S poles that are arranged in a circle, is integrated with the hub bearing inner race.

(c) The active type speed sensor uses a sensor IC to detect magnetic field changes caused when the sensor rotor rotates, and the sensor outputs the detected information to the No. 2 skid control ECU as digital pulses (vehicle speed signal).

(d) To detect the vehicle speed, the frequency of the output pulses is used. Because the active type sensor outputs digital pulses, it can detect vehicle speeds even when the vehicle is nearly stationary.
