SOFTWARE UPDATE

FUNCTION

(a) Software Update Function

(1) Toyota Safety Sense is updated to the latest software via wireless communication module (DCM). Wireless software update is called OTA (Over The Air) reprogramming.

(2) The target component for the software update of Toyota Safety Sense is the forward recognition camera.

(3) If a software update is available, a notification will be displayed on OneApp. Perform a software update in accordance with the instructions shown on the screen.

(4) The vehicle can be driven normally and the Toyota Safety Sense can be used during a software update.

(5) Download of software should be performed with the ignition switch ON (IG). The download is suspended if the ignition switch is turned off. When the ignition switch is turned to ON (IG) again, the suspended download will restart from where it stopped. New updated functions can be used after the ignition switch is turned to ON (IG) again after completion of the download.

NOTICE:

  • Depending on the software data volume, an update such as a major update may take several hours.
  • Once the software is updated, it cannot be reverted to a previous version. If a used car is purchased, the software state updated by the previous owner is taken over.
  • When the software is updated, the way functions are used may change and functions may be added. For details about updates or additions, check the Online Owner's Manual at Toyota.com.
  • If a connected service subscription contract has not been entered or has expired, software updates will not be able to be performed wirelessly.
