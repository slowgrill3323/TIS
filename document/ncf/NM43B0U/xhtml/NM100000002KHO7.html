(d) Remote Engine Start Function

(1) The remote function is not made available in some countries, areas, or on some models.

(2) The remote engine start function is operated in the following 2 ways: key operation via an electrical key transmitter sub-assembly or the exclusive application on a user's cellular phone. For key operation, 2 short presses followed by 1 long press of the lock button on the electrical key transmitter sub-assembly starts the hybrid system. When the hybrid system is successfully started, the hazard warning lights flash to inform the driver. The user can also confirm the hybrid system start using the application on the user's cellular phone.

HINT:

The hybrid system can be started by the remote engine start function when all of the following conditions are met:

  • All doors are closed.
  • The engine hood is closed.
  • The power source mode is off.
  • The electrical key transmitter sub-assembly is not in the vehicle.
  • The auxiliary battery voltage is 11 V or higher.
  • The cumulative hybrid system operating time after being started by remote engine start is less than 15 minutes.
  • All doors are locked.
  • The hybrid system has not already been started by remote engine start.

(e) Remote Engine Stop Function

(1) The remote function is not made available in some countries, areas, or on some models.

(2) The remote engine stop function is operated in the following 2 ways: key operation via an electrical key or the exclusive application on a user's cellular phone. For key operation, 1 long press of the lock button or 1 short press of the unlock button on the electrical key stops the hybrid system. The user can also confirm hybrid system stop using the application on the user's cellular phone.

HINT:

The hybrid system will stop when any of the following conditions is met:

  • Any door is opened.
  • The engine hood is opened.
  • The power switch is pressed.
  • A stop request is received from the electrical key transmitter sub-assembly.
  • The theft deterrent system is in the alarm sounding state.
  • The auxiliary battery voltage drops to less than 11 V.
  • 10 minutes elapse after the hybrid system is started.
  • The cumulative hybrid system operating time after being started by remote engine start is 20 minutes or more.
  • The shift lever is moved to a position other than P.
  • The vehicle speed is 3 mph (5 km/h) or more.
  • The stop lights illuminate or their conditions cannot be determined.
  • Any door is unlocked.
  • The hybrid system stops for 3 seconds.
  • The hybrid system is malfunctioning.
  • The fuel level warning is displayed on the combination meter assembly.
  • The CAN communication is malfunctioning.

FUNCTION

(a) The smart key system consists of the following functions:

Function

Outline

Wireless Door Lock Control

This function is a convenient system for locking and unlocking all the doors at a distance. The operation of this function in the smart key system is the same as that of the wireless door lock control system. However, the receiver in the certification ECU uses an electrical key and TPMS receiver assembly to perform the control.

Entry Illumination

When an electrical key transmitter sub-assembly enters the exterior actuation area of the door electrical key antenna or outside electrical key antenna, each interior light will illuminate.

Entry Unlock

The doors are unlocked by grabbing the door outside handle or pressing the tail gate opener switch*1 while carrying the electrical key transmitter sub-assembly.

  • The driver door outside handle unlocks driver door. (Driver Door Unlock Mode)
  • The driver door outside handle unlocks all doors. (All Door Unlock Mode)
  • The front passenger door outside handle unlocks all doors.
  • The tail gate opener switch unlocks all doors.*1

Entry Tail Gate Open*1

The tail gate is opened by pressing the elbow switch (luggage electrical key switch) and the tail gate opener switch while carrying the electrical key transmitter sub-assembly.

Entry Lock

All doors are locked by touching the lock sensor of the driver or front passenger door outside handle or by pressing the lock switch of the tail gate*1 while carrying the electrical key transmitter sub-assembly.

Answer Back

The wireless door lock buzzer sounds and the hazard warning lights blink once while locking and twice while unlocking when the doors are being locked or unlocked. Also, the answer back function operates when the doors are locked by the automatic relock function.

Automatic Relock

If none of the doors are opened within 60 seconds after being unlocked using the entry unlock operation, all the doors will be locked again automatically.

Power Switch Lettering Illumination Indication

Indicates the power source change condition and that the hybrid system start conditions are met by flashing and illuminating the power switch lettering.

Door Open Operation Prohibition

Prohibits the use of entry lock operations to lock the door when any door is ajar or fully open.

Prevention of Key Confinement

Prevents the vehicle from being locked with an electrical key transmitter sub-assembly left inside it. This function operates in 2 different situations: when the lock switch on the door control switch is used (keyless lock) or when an entry lock operation is performed to lock the vehicle. (Does not operate when the electrical key transmitter sub-assembly is in the sleep state.)

Start

Changes the power source or starts the hybrid system.

Automatic Hybrid System Shut Off*2

Automatically turns the ignition switch off and stops the hybrid system after a certain amount of time elapses after the vehicle is parked with the hybrid system operating.

Immobiliser

Only permits electrical key transmitter sub-assembly registered with the vehicle to start the hybrid system, preventing unauthorized starting of the system.

Warning

The smart key system causes the certification ECU to sound the buzzer in the combination meter assembly and uses the multi-information display in order to alert the driver.

Battery Saving
  • Suppresses auxiliary battery level drops when the vehicle is not used for a long period of time by extending the interval between or stopping each periodic transmission of the request signal and restricting the entry function.
  • Puts the electrical key transmitter sub-assembly in the sleep condition to suppress key battery level drops.*3
  • When the unlock button is pressed 2 times while pressing the lock button of the electrical key transmitter sub-assembly, the electrical key transmitter sub-assembly stops waiting on standby to receive electrical waves, suppressing key battery level drops.

Entry Unlock Mode Switching

Allows switching between 2 modes that can be operated with the entry unlock function.

  • Driver Door Unlock Mode
  • All Door Unlock Mode

Key Cancel

The following key functions can be canceled by following certain procedures:

  • Entry unlock/lock
  • Prevention of key confinement
  • Start
  • Warning

Key Code Registration

A total of 7 keys can be registered. Key registration enables the registering (writing and storing) of transmitter recognition codes in the EEPROM that is contained in the certification ECU.

Customization

Allows the user to change settings as desired. Refer to the Repair Manual for details.

SYSTEM CONTROL

(a) Entry Unlock Function

(1) When the electrical key transmitter sub-assembly is in the actuation area formed by the door electrical key antenna, holding the door outside handle of the driver seat and touching the built-in touch sensor unlocks the driver door only. Continuing to touch the built-in touch sensor for 2 seconds or more unlocks all the doors. And holding any of the front passenger door outside handles and touching the built-in touch sensor unlocks all the doors. (Driver door unlock mode)

(2) When the electrical key transmitter sub-assembly is in the actuation area formed by the door electrical key antenna, holding driver door or front passenger door outside handle. and touching the built-in touch sensor unlocks all the doors. (All door unlock mode)

(3) After the doors have been unlocked, the wireless door unlock buzzer sounds twice as an answer back and the hazard lights flash twice at the same time.

NOTICE:

  • When the user quickly performs an unlock operation or abruptly enters the exterior actuation area to perform an unlock operation and is unable to unlock the doors, an unlock operation needs to be performed again. When an unlock operation is performed while wearing gloves, the door may not unlock.
  • If another electrical key transmitter sub-assembly is in the exterior actuation area, the time it takes to unlock the door(s) after the unlock operation may be slightly longer than usual.
  • After the doors are locked, the doors cannot be unlocked for approximately 3 seconds.
  • When a wireless door lock, manual door lock or key-linked door lock operation is performed with an electrical key transmitter sub-assembly in the interior actuation area or near the vehicle, the entry unlock function may not operate. In this case, it is possible to unlock the doors by performing a wireless door unlock operation.

HINT:

If the doors are locked by wireless door lock operation, the smart key system or key linked power door lock operation, it may not be possible to unlock the doors using the smart key system. In this situation, use the wireless door lock function of the electrical key transmitter sub-assembly to unlock the doors.

(d) Start Function

(1) With the electrical key transmitter sub-assembly in the cabin internal detection area, power source mode switching and hybrid system start can be performed by pressing the power switch while depressing the brake pedal.

(2) If the power switch is pressed without depressing the brake pedal when the power source mode is off and the vehicle is entered while carrying the electrical key transmitter sub-assembly, the power source mode changes.

HINT:

  • The change varies depending on the value set by the ACC customization function.
  • When ACC customize setting is set to ON, ACC mode is available, and when it is set to OFF, ACC mode is not available.
  • When ACC customize setting is set to ON:

    When the power switch is pressed, the power source mode changes to ACC and "ACCESSORY" is displayed on the multi-information display in the combination meter assembly. If the power switch is pressed again while "ACCESSORY" is displayed, the power source mode changes to ON and "POWER ON" is displayed on the multi-information display in the combination meter assembly. Each time the power switch is pressed, the power source mode changes from off to ACC to ON to off (repeating).

  • When the ACC customization function is set to OFF:

    If the power switch is pressed, the power source mode changes to ON and "POWER ON" is displayed on the multi-information display in the combination meter assembly. Each time the power switch is pressed, the power source mode changes from off to ON to off (repeating).

(3) If the brake pedal is depressed with park (P) selected when the power source mode is off and the vehicle is entered while carrying the electrical key transmitter sub-assembly, the smart key indicator is displayed on the multi-information display in the combination meter assembly. If the power switch is pressed while the smart indicator is displayed, the hybrid system starts.

(4) When ACC customize setting is set to ON:

When the vehicle is stopped, if the power switch is pressed with park (P) selected while the hybrid system is activated, the power source mode changes to off. If the power switch is pressed while park (P) is not selected, the power source mode changes to off.

NOTICE:

  • Normally, the operation of the power switch is disabled while the vehicle is being driven. However, in an emergency, by pushing the power switch for approximately 2 seconds or more or pushing the power switch 3 times or more in a row, the driver can stop the hybrid system while the vehicle is in motion.
  • If no signals are transmitted to the certification ECU (smart key ECU assembly) due to malfunctions in the stop light switch assembly, the hybrid system may not start when the power switch is pushed with the brake pedal depressed. The hybrid system may start by setting the shift state to P, engaging the parking brake, turning the ignition switch to ACC, depressing the brake pedal and pressing and holding the power switch for 15 seconds or more.
  • The above 2 operations must be applied only in emergency situations. Under normal conditions, the hybrid system must not be stopped by pushing the power switch while driving. Also, it should not be started without depressing the brake pedal or when the shift lever is in any position other than P or N.

Power source mode change when the electrical key transmitter sub-assembly does not operate correctly due to a depleted key battery or radio wave interference:

  • Unlock the door with the mechanical key built into the electrical key transmitter sub-assembly, and while carrying the electrical key transmitter sub-assembly, enter the vehicle.
  • While depressing the brake pedal, touch the ornament side of the electrical key transmitter sub-assembly to the front of the power switch.
  • The buzzer of the combination meter assembly sounds and the power source mode changes from off to ON. If the entry unlock function is disabled, the power source mode changes from off to ACC.
  • If the power switch is pressed without depressing the brake pedal, the power source mode changes from ON to off. If the entry unlock function is disabled, the power source mode changes from ACC to ON to off.

Starting the hybrid system when the electrical key transmitter sub-assembly does not operate correctly due to a depleted key battery or radio wave interference:

  • Unlock the door with the mechanical key built into the electrical key transmitter sub-assembly, and while carrying the electrical key transmitter sub-assembly, enter the vehicle.
  • With park (P) or neutral (N) selected and while depressing the brake pedal, touch the ornament side of the electrical key transmitter sub-assembly to the front of the power switch.

(h) Battery Saving Function

(1) Battery saving preserves the vehicle and electrical key transmitter sub-assembly batteries. The battery saving function is activated when the vehicle remains parked for a long time and when the electrical key transmitter sub-assembly is left in the exterior actuation area.

Condition

Condition

Outline

When the vehicle remains parked for a long time.

Condition 1:

No response from the electrical key transmitter sub-assembly for more than 5 days.

  • Door electrical key antennas and outside electrical key antenna stop emitting signals for creating the actuation area. In addition, the lock sensors or unlock sensors of all the seats are available, so touching the sensor can lock or unlock.

When the power source is turned ON, the control is canceled and the system is returned to normal.

Condition 2:

No response from the electrical key transmitter sub-assembly for more than 14 days.

  • Door electrical key antennas and outside electrical key antenna stop emitting signals for creating the actuation area.
  • The lock sensors and unlock sensors except for the driver door are disabled.

When the power source is turned ON, the control is canceled and the system is returned to normal. When one of the following conditions is met, the control is returned to condition 1:

  • The doors are locked or unlocked using the wireless door lock control function.
  • The doors are locked or unlocked by touching the lock/unlock sensor on the driver door.
  • A door is locked or unlocked using the mechanical key.
  • A Door is opened closed.

When the electrical key transmitter sub-assembly is left in the exterior actuation area.

This state continues longer than 10 minutes.

  • Electrical key antenna that detects the electrical key transmitter sub-assembly stops emitting signals for creating the actuation area.

  • The doors are locked or unlocked using the wireless door lock control function.
  • The doors are locked or unlocked by touching the lock/unlock sensor.
  • A door is locked or unlocked using the mechanical key.

(2) In addition to the above functions, the auto power off function helps to reduce drain on the battery by turning the power source mode to off when approximately 20 minutes have elapsed since the shift lever was moved to P with the power source ACC, or the hybrid system was turned off with the shift lever in P, the power source ON and the vehicle stationary.

(i) Key Battery Saving Function

(1) This function cancels the signal reception waiting state of the electrical key transmitter sub-assembly, reducing the consumption of current from its battery.

(2) The electrical key transmitter sub-assembly can be placed into and taken out of battery saving mode. When the electrical key transmitter sub-assembly is in battery saving mode, the smart key system will not be able to detect the electrical key transmitter sub-assembly, as the electrical key transmitter sub-assembly will not respond to signals from the vehicle. The smart key system will not operate if the electrical key is in battery saving mode.

(3) Pressing the unlock button of the electrical key transmitter sub-assembly twice while pressing the lock button (the indicator light of the electrical key transmitter sub-assembly blinks 4 times) selects battery saving mode for the electrical key transmitter sub-assembly.

(4) Battery saving mode can be canceled by pressing any button on the electrical key transmitter sub-assembly.

(j) Entry Unlock Mode Switching Function

(1) The following entry unlock modes can be selected when using the entry unlock mode switching function.

  • All door unlock mode: When the sensor of a door outside handle is touched, an entry unlock operation is performed simultaneously for all doors.

(2) The system functions can be deactivated by using the following key cancel procedure when the power source is off and the driver door is closed and unlocked:

Key Cancel Procedure
  • The procedure is as follows:
    • Press the unlock button of the electrical key transmitter sub-assembly once.
    • Within 5 seconds of the step above, open the driver door (driver door: close open).
    • Within 5 seconds of the step above, press the unlock button of the electrical key transmitter sub-assembly twice.
    • Repeat close open for the driver door twice* (driver door: open close open close open).
    • Press the unlock button of the electrical key transmitter sub-assembly twice.*
    • Repeat close open for the driver door once* (driver door: open close open).
    • Within 5 seconds of the step above, close the driver door.
  • When the cancel procedure is complete, the buzzer sounds twice. Meanwhile, when the procedure is performed again to resume the system, the buzzer sounds once.

HINT:

*: All these 3 steps are performed within 30 seconds of the previous step.

DIAGNOSIS

(a) The certification ECU can detect malfunctions in the smart key system when the power source is ON.

(b) When the ECU detects a malfunction, a message is displayed on the multi-information display to warn the driver. At the same time, the ECU stores a 5-digit Diagnostic Trouble Code (DTC) in its memory.

(c) The indicator light warning continues for 15 seconds even after the power source is turned off.

(d) The 5-digit DTCs can be read after connecting the GTS to the DLC3.

(e) The smart key system may not operate successfully if a malfunction occurs.
