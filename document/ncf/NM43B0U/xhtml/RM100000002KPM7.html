  • Controls operation of the meter buzzer, and sends a display request signal, etc. to the parking assist ECU and combination meter assembly, based on static object detection data and abnormality judgment.

Combination Meter Assembly

  • Receives static object detection signals from the clearance warning ECU assembly, and displays intuitive parking assist system detection on the multi-information display.
  • When the intuitive parking assist system is off, turns the intuitive parking assist system OFF indicator on.
  • Sends a customization signal to the clearance warning ECU assembly.
  • When a malfunction is detected in the intuitive parking assist system or the ultrasonic sensors determine that control is not possible, the intuitive parking assist OFF indicator illuminates, buzzer sounds and a message is displayed on the multi-information display to warn the driver.

SYSTEM CONTROL

(a) Intuitive Parking Assist System Detection Area

(1) Static objects are detected within approximately 600 mm (23.6 in.) of the front left and right corner sensors and rear left and right corner sensors, within approximately 1000 mm (39.4 in.) of the front left and right center sensors, and within approximately 1500 mm (59.1 in.) of the rear left and right center sensors. Note that there are some areas where static objects cannot be detected if they are too close to the bumper. The detection range may also vary depending on the shape of the static object.

HINT:

Furthermore, the detection range may differ depending on the following conditions.

  • Difference of shapes (stationary object)
  • The number of occupants in the vehicle
  • Luggage in the vehicle

FUNCTION

(a) Intuitive Parking Assist System Operation

(1) The combination meter assembly customization setting can be used to turn the intuitive parking assist system on and off.

(2) An intuitive parking assist system off indicator is located in the combination meter assembly to make it possible to check that the intuitive parking assist system is off.

(b) Warning Buzzer Mute Function

(1) When the warning buzzer is sounding, it is automatically muted if any of the following conditions are established.

  • There is no change to the distance from the static object over a certain period of time.
  • The distance between the vehicle and static object increases.
  • The static object is removed from the path.

(2) When the warning buzzer has been automatically muted, it once again begins sounding if any of the following conditions are established.

  • A static object is detected within short-range.
  • The vehicle approaches a static object.
  • A static object that was out of the path of the vehicle is now on the path of the vehicle.

(3) When the warning buzzer is sounding, the warning buzzer is temporarily muted if the steering pad switch assembly is used to do so.

(4) When the warning buzzer has been temporarily muted, temporary muting is canceled if any of the following conditions are established.

  • The shift status is changed.
  • The vehicle speed reaches a certain value or higher.
  • There is a sensor abnormality (open/frozen).
  • It cannot be used (CAN communication abnormality).
  • The intuitive parking assist system was turned off.
  • The ignition switch was turned off.

(c) Abnormality Judgment

(1) When the intuitive parking assist system is turned on, abnormality judgment is performed on all sensors regardless of the shift position.

(2) If there is an open or frozen sensor, or if there is a communication malfunction in the clearance warning ECU, an intuitive parking assist system off indicator illuminates and a warning is displayed on the multi-information display.

(d) Intuitive Parking Assist System Settings

(1) The combination meter assembly customization setting can be used to change the volume of the buzzer (3 levels).

(2) The Global TechStream (GTS) can be connected to change settings for some functions as desired by the user. Refer to the Repair Manual for details.

DIAGNOSIS

(a) In order to make system inspections easier to perform, a diagnosis function is used in consideration of serviceability. The DTCs of malfunctioning parts can be read by connecting the Global TechStream (GTS). Refer to the Repair Manual for details.
