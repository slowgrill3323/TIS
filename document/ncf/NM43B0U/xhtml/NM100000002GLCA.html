Learning Mode

(New Code Registration)

When an untrained switch is pressed and released, the mode changes to learning mode in which a transmitter code can be registered.

If no codes are registered within approximately 60 seconds after entering learning mode, the mode changes to low power mode.
  • During Learning Mode: Orange (Slow Flashing)
  • Registration Completed: Green (Quick Flashing)
  • Quick flashing is for rolling code.*
  • Steady Green is for fixed code.*

Learning Mode

(Overwrite)

When a trained switch is pressed and held for 20 seconds, the mode changes to learning mode in which an existing transmitter code can be overwritten.

If no codes are registered within approximately 60 seconds after entering learning mode, the mode changes to low power mode.

All Delete Mode

When the 2 outer garage door opener switches are pressed simultaneously for 10 seconds, all the transmitter codes that are registered to the 3 switches are cleared. By releasing these switches within 10 seconds after clearing the codes, the mode will automatically change to learning mode the next time the switches are pressed. By keeping the switches pressed for longer than 10 seconds after clearing the codes, all the switches will be registered with a code for operation verification.

Code Clearing Completed: Green (Quick Flashing)

Low Power Mode

If a switch remains pressed longer than 120 seconds, such as in the case in which the pressed switch does not release itself, the mode will automatically be changed to low power mode to reduce power consumption.

Listen Mode*

When a garage door is operated in transmission mode, the garage door operation indicator lights show the operation status.
  • Currently Opening/Closing: Orange (Flashing)
  • Opening/Closing Completed: Green
  • Signals Not Received: Red (Flashing)
