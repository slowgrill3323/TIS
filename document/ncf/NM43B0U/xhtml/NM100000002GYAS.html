SPARK PLUG

CONSTRUCTION

(a) Long-reach, thin-electrode type iridium tipped spark plugs are used. This type of spark plug allows the area of the cylinder head sub-assembly which receives the spark plugs to be made thicker. Thus, the cylinder head water jacket can be extended near the combustion chamber, which contributes to improved cooling performance.

(b) Iridium-tipped spark plugs improve ignition performance while maintaining the same durability as platinum-tipped spark plugs.
