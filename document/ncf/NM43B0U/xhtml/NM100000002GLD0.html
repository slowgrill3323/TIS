MULTI-DISPLAY

OPERATION

(a) While the multi-terrain monitor system*1, panoramic view monitor system*2 or parking assist monitor system*3 is operating, if the intuitive parking assist system detects a static object, the panoramic view cooperative display*4 or the intuitive parking assist system icon display*5 is used to display the detection location and the approximate distance to the static object on the radio and display receiver assembly.

  • *1: Models with multi-terrain monitor system
  • *2: Models with panoramic view monitor system
  • *3: Models with parking assist monitor system
  • *4: When panoramic view vehicle icon is displayed
  • *5: When panoramic view vehicle icon is not displayed or models with parking assist monitor system

(b) Intuitive Parking Assist Detection Display (Panoramic View Cooperative Display)

(1) When the panoramic view vehicle icon is displayed, if the front center sensor and rear center sensor* detect static objects, the detection location and the approximate distance to static objects are displayed on the multi-display assembly at 5 levels (longest range, long-range, medium-range, short-range and shortest range).

HINT:

*: When driving backward with a trailer connected, the rear center sensor and rear corner sensor turn off and only the front corner sensor operates. Therefore, the intuitive parking assist detection display for the rear sensor area when driving backward is not displayed.

Center Sensors (The illustrations are examples only. Actual screens may vary.)

(2) When the panoramic view vehicle icon is displayed, if the front corner sensor and rear corner sensor* detect a static object, the detection location and the approximate distance to the static object are displayed on the multi-display assembly at 4 levels (long-range, medium-range, short-range and shortest range).

HINT:

*: When driving backward with a trailer connected, the rear center sensor and rear corner sensor turn off and only the front corner sensor operates. Therefore, the intuitive parking assist detection display for the rear sensor area when driving backward is not displayed.

HINT:
