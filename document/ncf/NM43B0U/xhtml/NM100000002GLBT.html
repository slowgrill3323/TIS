VOICE RECOGNITION

FUNCTION

(a) When a wake word is spoken*1 and the microphone icon button in the screen is selected, or the voice recognition button on the steering pad switch assembly is pressed, the system transitions to the voice recognition mode screen and a search for functions and information, etc. can be performed using voice commands or a keyboard.

(b) The standard onboard assistant function and intelligent assistant function strengthens the existing voice command system. As well as including all assistant functions (telephone, radio, navigation, streaming media), the intelligent assistant function*2 provides excellent voice command.

(c) To make it easy to grasp speaking timing or the voice control conditions when detection processing is being performed etc., a voice recognition indicator is displayed on the screen to visually display the voice control status.

(d) When the user utters "help", a voice command example list is displayed.

(e) By performing a keyboard search using the voice recognition mode screen, a location / music / contact address / settings / etc. can easily be searched for even without using the intelligent assistant function.

HINT:

  • *1: By uttering "Hey / Hi / Hello / OK Toyota" from a front seat, a hands-free voice dialogue can start by using the wake word. The use of the wake word can be turned on/off using the settings menu.
  • *2: Offboard voice recognition can be used when there is a telematics service contract and DCM communication is possible.
  • When offline, a POI search via the navigation function cannot be performed.
