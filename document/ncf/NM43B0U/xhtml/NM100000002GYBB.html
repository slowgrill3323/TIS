STARTER CONTROL (CRANKING HOLD FUNCTION)

FUNCTION

(a) When first starting the engine, if the certification ECU (smart key ECU assembly) receives a start signal from the engine switch, the certification ECU (smart key ECU assembly) sends an operation signal to the ST relay to operate the starter assembly. At this time, the certification ECU (smart key ECU assembly) turns the ACC relay operation signal (ACCD) off to prevent flickering of the meter, clock and audio systems.

(b) When starting the engine during idling stop control, the engine stop and start ECU* sends operation signals to the ST relay according to the vehicle condition to operate the starter assembly.

HINT:

*: Models with Stop and Start System

(c) While the engine is cranking, the certification ECU (smart key ECU assembly) and/or engine stop and start ECU* continues to send an operation signal to the ST relay until it has been detected that the engine has started successfully. Whether the engine has started successfully is determined by the certification ECU (smart key ECU assembly) and ECM, and once it has been determined that the engine has started successfully, the signal output from the certification ECU (smart key ECU assembly) and/or engine stop and start ECU* to the ST relay is stopped.

HINT:

*: Models with Stop and Start System

(d) The crank hold time and engine speed when it has been determined that the engine has started successfully are determined according to the engine coolant temperature.
