TOUCH SENSOR

CONSTRUCTION

(a) The touch sensor (power tail gate sensor assembly) consists of conductive rubber and a core wire. When the conductive rubber is compressed, its resistance value goes down and electric current flows through the core wire. This signal is input to the power tail gate ECU (multiplex network door ECU), which determines the tail gate condition.

(b) The power tail gate ECU (multiplex network door ECU) has a built-in resistor for checking the operation of the touch sensor (power tail gate sensor assembly).

(c) When the resistance reaches 0 Î©, the power tail gate ECU (multiplex network door ECU) disables the power-close function and displays an error message on the display audio.
