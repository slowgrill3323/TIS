GENERAL

OUTLINE

(a) Rear Window Defogger System (Models with Rear Window Defogger System)

  • The rear window defogger system uses the heater wires on the back window glass to defog the back window glass.
  • This system is controlled by the air conditioning amplifier assembly.
  • The rear window defogger system operates together with the mirror heater system.

(b) Windshield Deicer System (Models with Windshield Deicer System)

  • The windshield deicer system uses a heater element in the windshield glass to help reduce the formation of ice where the front wipers park.
  • This system is controlled by the air conditioning amplifier assembly.
