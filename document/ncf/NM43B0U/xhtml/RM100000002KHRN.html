Title: THEFT DETERRENT / KEYLESS ENTRY: IMMOBILISER SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [03/2024 -        ]

GENERAL

OUTLINE

(a) Gasoline Models

(1) The immobiliser system compares the ID code that is registered in the certification ECU (smart key ECU assembly) with the ID code of the transponder chip that is embedded in the electrical key transmitter sub-assembly. The system unsets if these ID codes match. Thus, the certification ECU (smart key ECU assembly) and the ECM communicate with each other to authorize fuel injection and ignition, enabling the engine to start.

HINT:

When replacing the certification ECU (smart key ECU assembly) or making a new electrical key transmitter, key's recognition code(s) must be registered. For details, refer to the Repair Manual.

(b) HEV Models

(1) The immobiliser system compares the ID code that is registered in the certification ECU (smart key ECU assembly) and ID code box (immobiliser code ECU) with the ID code of the transponder chip that is embedded in the electrical key transmitter sub-assembly. The system unsets if these ID codes match. Thus, the certification ECU (smart key ECU assembly) or ID code box (immobiliser code ECU) and the hybrid vehicle control ECU communicate with each other to enable the hybrid system to start.

HINT:

When replacing the certification ECU (smart key ECU assembly) or making a new electrical key transmitter, key recognition code(s) must be registered. For details, refer to the Repair Manual.
