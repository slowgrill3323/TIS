SYSTEM CONTROL

(a) The skid control ECU locks or releases the parking brake by controlling the rotation direction and drive current of the motor built into the parking brake actuator assembly based on signals from the electric parking brake switch and each ECU.

(b) When the electric parking brake is locked, the skid control ECU operates the motor in the parking brake actuator assembly to lock the parking brake.

(c) When the electric parking brake is released, the skid control ECU reverse operates the motor in the parking brake actuator assembly to release the parking brake.

(d) The skid control ECU detects the motor operation condition via its built-in current sensor.

(e) The skid control ECU generates braking force according to the road surface while the vehicle is stopped by calculating the road grade based on the signal from the deceleration sensor to control the motor drive current.

FUNCTION

(a) Manual Function

(1) When the electric parking brake switch is pulled with the vehicle stopped, the parking brake locks.

(2) When the brake pedal is depressed, and then the electric parking brake switch is pushed, the parking brake releases.

(3) Braking force is only generated while the electric parking brake switch is pulled up when the vehicle is in motion. The braking force is generated by hydraulic pressure, and while the braking force is being generated, the brake light illuminates. When the switch is released, the braking force decreases.

(4) The parking brake lock operation can be operated regardless of whether the ignition switch is turned ON or off.

(5) The parking brake can be released when the ignition switch is ON.

(b) Auto Function (Automatic Transmission)

(1) The auto function can be switched to on and off via operation of the electric parking brake switch.

(2) When the electric parking brake switch is pulled and held for 3 seconds or more with the parking brake locked, the auto function turns on. Then a message is displayed on the multi-information display indicating that the auto function is on.

HINT:

When switching from the parking brake released condition, it is necessary to pull it for 5 seconds or more.

(3) When the electric parking brake switch is pushed and held for 3 seconds or more with the parking brake released, the auto function turns off. Then a message is displayed on the multi-information display indicating that the auto function is off.

HINT:

When switching from the parking brake locked condition, it is necessary to press it for 5 seconds or more.

(4) When the driver changes the shift position to P while the auto function is on, the parking brake is locked. Also, when the shift position is changed from P to any other position with the brake pedal depressed, the parking brake is released.

(c) Auto Function (Manual Transmission)

(1) When the auto function is on, if the driver turns the ignition switch off, the skid control ECU controls the parking brake actuator assembly motor operation and locks the parking brake.

(d) Parking Brake Automatic Release Function

(1) When all of the following conditions are met and the driver depresses the accelerator pedal, the parking brake is released.

*1: Automatic Transmission

*2: Manual Transmission
  • Driver side door is closed.
  • Shift position is not in P or N.*1
  • Shift position is not in N.*2
  • Clutch pedal is depressed.*2
  • Driver side seat belt is fastened.

FAIL-SAFE

(a) When a malfunction occurs in the system, control using the manual function is maintained as much as possible and control using the auto function and parking brake automatic release function is prohibited.

HINT:

When a malfunction occurs in the electric parking brake switch, the system automatically switches to the auto function to ensure operability.

(b) When the parking brake cannot be released due to the fail-safe state or a malfunction in the power source system, the parking brake can be released by supplying power to the electric release connector. For details, refer to the Repair Manual.

(c) When the parking brake cannot be released due to a mechanical malfunction, the parking brake can be released by removing the parking brake actuator assembly from the brake caliper. For details, refer to the Repair Manual.

DIAGNOSIS

(a) If a malfunction is detected in the system, the skid control ECU will alert the driver by illuminating the brake warning light / red (minor malfunction) in the combination meter and outputting a warning message on the multi-information display.

(b) At the same time, Diagnosis Trouble Codes (DTCs) are stored in the memory. The DTCs stored in the skid control ECU are output to the Global TechStream (GTS) connected to the DLC3. For details, refer to the Repair Manual.
