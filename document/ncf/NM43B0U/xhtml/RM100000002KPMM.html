Parking Assist ECU
  • Combines video signals from the front, rear, left and right television camera to create a seamless overhead image.
  • Calculates each guide line based on information signals from each ECU and sensor, combines the lines with the panoramic view and images from each television camera assembly, and outputs a video signal to the multi-display.
  • Turns each television camera assembly on and off according to the input shift position signal and each switch signal.
  • Combines detection information with the currently displayed panoramic view monitor system screen according to signals from the clearance warning ECU assembly, blind spot monitor sensor RH and outputs a video signal to the multi-display.

(b) In addition to manual display mode using a panoramic view monitor switch operation (when automatic display mode is off), automatic display mode (when automatic display mode is on) that displays the "panoramic view and front view" automatically according to vehicle speed is provided as a method for displaying the panoramic view monitor system screen.

(1) Manual Display Mode

  • Manual display mode is a method of displaying the panoramic view monitor system screen according to the needs of the driver via operation of the panoramic view monitor switch and touch.
  • The panoramic view monitor system is displayed by operating the panoramic view monitor switch when vehicle speed is approximately 16 km/h (10 mph) or less.
  • When the vehicle speed exceeds approximately 16 km/h (10 mph), the panoramic view monitor system screen ends and switches to display the navigation screen or information screen.

(2) Automatic Display Mode

  • When starting off and stopping the vehicle in automatic display mode, it is possible to display the "panoramic view and front view" automatically instead of by pressing the panoramic view monitor switch.
  • The automatic display mode can be set with the auto mode button displayed on the screen.

(3) Display Function Linked to Intuitive Parking Assist System

  • When the intuitive parking assist detects a stationary object while driving at a low speed, the monitor screen automatically changes from the current screen (navigation screen, DVD screen, etc.) to the clearance information screen with the "panoramic view and front view" screen.
  • When all of the following conditions are met, the panoramic view and front view will be displayed.
    • The shift position is not P or R.
    • The intuitive parking assist system has detected a stationary object.
  • When either of the following conditions is met, the panoramic view and front view will be canceled and return to its original screen.
    • The intuitive parking assist system does not detect a stationary object.

FUNCTION

(a) Panoramic View Monitor System Screen and Operation

  • Each screen of the panoramic view monitor system displays guide lines, estimated course lines, etc., to inform the driver of the tire positions and trajectories which cannot be directly seen from the driver seat, according to the selected screen display mode.
  • When the intuitive parking assist system or RCTA function or RCD function detects an operation target, intuitive parking assist system information or RCTA function information or RCD function information is superimposed on the screen according to the selected screen display mode.

(1) Panoramic View and Front View

  • The panoramic view created by combining images from each television camera assembly and the image from the front television camera assembly are displayed simultaneously. The images assist the driver to confirm the safety of the surroundings and areas to the left and right of the vehicle at locations with poor visibility, such as intersections and T-junctions.
  • The panoramic view and front view are displayed by selecting the panoramic view and front view mode button.

(8) Moving View

  • This view combines images from the front, rear, left and right side television camera assemblies to display an image of the vehicle from a viewpoint diagonally above the vehicle. The displayed image rotates 360 degrees around the vehicle displayed on the screen to assist the driver to confirm the safety of the surroundings. Furthermore, after the image of the vehicle from a viewpoint diagonally above the vehicle rotates a full 360 degrees, an overhead view of the vehicle is displayed.
  • Using the rotation pause button, the rotation of the moving view can be paused and unpaused at will.
  • When all of the following conditions are met, the moving view will be displayed.
    • The shift position is P.*1
    • The parking brake is applied.*2
    • The shift position is not R.*2
    • The vehicle is stopped.*2

    • Panoramic view monitor switch is pressed.
