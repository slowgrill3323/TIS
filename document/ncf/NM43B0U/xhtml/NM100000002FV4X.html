POWER RUNNING BOARD CONTROL

FUNCTION OF MAIN COMPONENTS

(a) The power running board system is comprised of the following components.

Function

Outline

Side Auto Step Controller ECU Assembly

Controls all power running board system functions based on signals from the switches and sensors.

Automatic Running Board Sub-assembly

  • Side Auto Step Motor Assembly

Deploys or stows the step in response to input signals from the side auto step controller ECU assembly. The internal hall IC detects foreign objects and send signals to the side auto step controller ECU assembly.

Side Auto Step Switch Assembly

Sends switch signals to the side auto step controller ECU assembly.

  • When the side auto step switch assembly is operated to the OUT position, the automatic running board sub-assembly move to the deployed position.
  • When the side auto step switch assembly is operated to the OFF position, the automatic running board sub-assembly move to the stowed position.
  • When the side auto step switch assembly is operated to the AUTO position, the automatic running board sub-assembly operate automatically according to the vehicle condition.

Combination Meter Assembly

In the following situations, the operation mode or a message is displayed on the multi-information display.

  • The operation mode of the power running board system is switched by operating the side auto step switch assembly.
  • The automatic running board sub-assembly is not in the proper position and the fail-safe function is activated.
  • There is a malfunction in the power running board system.

Main Body ECU

(Multiplex Network Body ECU)

Sends power running board system operation request signals according to the vehicle condition to the side auto step controller ECU assembly.

SYSTEM CONTROL

(a) The power running board system performs the following controls.

Function

Outline

Automatic Deployment

When all of the following conditions have been satisfied, the automatic running board sub-assembly deploy automatically.

Automatic Running Board Sub-assembly

  • The side auto step switch assembly is in the AUTO position.
  • Vehicle speed is below 3 mph (5 km/h).
  • From a situation in which the front and rear doors are shut, either the front or rear doors become "open."*1

Automatic Stowing

When all of the following conditions have been satisfied, the automatic running board sub-assembly stow automatically.

Automatic Running Board Sub-assembly

  • The side auto step switch assembly is in the AUTO position.
  • With both the front and rear doors (or either) open, both the front and rear doors are closed.*1

Vehicle movement-induced stowing

When all of the following conditions have been satisfied, the automatic running board sub-assembly stow automatically.*2, *3

  • Vehicle speed is 3 mph (5 km/h) or above.

Preventing Deployment

Preventing deployment of the step can be accomplished by switching the side auto step switch assembly from AUTO to OFF.

Jam Protection

If foreign objects or jams are detected during deployment (stowing), the motor is reversed, returning the step to its original position.

HINT:

  • *1: The right side and left side operate independently.
  • *2: After vehicle movement-induced stowing, the step will not return to its deployed position even when vehicle speed drops below 3 mph (5 km/h) when waiting for traffic signals, etc.
  • *3: Additionally, when the side auto step controller ECU assembly detects a foreign object during vehicle movement-induced stowing, a reverse movement is induced once, deploying the step, and then the step is re-stowed if vehicle speed continues at 3 mph (5 km/h) or above. If the foreign object is not removed at this point, the step halts when it makes contact with the foreign object.

(b) Jam Protection Function

(1) If a foreign object is detected in a defined area when during the automatic deployment (stowing) operation, the side auto step controller ECU assembly reverses the motor and returns the step to its original position*1.

(2) If the jam protection function activates three times in succession, the side auto step controller ECU assembly halts the step in the possession where the foreign object was detected*2.

HINT:

  • *1: To prevent erroneous activation, the side auto step controller ECU assembly does not detect foreign objects when the step is outside a defined area.
  • *2: If the foreign object is removed before jam detection operates three times, normal functioning will resume after the power running board system operates once in response to door opening or closing.

(3) The side auto step controller ECU assembly evaluates whether there is a jam based on changes in the width of the pulse signal from the hall IC inside the motor.
