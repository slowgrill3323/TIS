MIRROR CONTROL

FUNCTION OF MAIN COMPONENTS

(a) The main components in the mirror control system have the following functions:

Component

Function

Outer Rear View Mirror Assembly LH/RH

Moves the mirror surface of the outer rear view mirror assembly LH/RH based on the mirror surface operation signal output by the outer mirror switch assembly.

Outer Mirror Switch (Multiplex Network Master Switch Assembly)

Mirror Adjust Switch

Outputs the mirror surface angle adjustment signal to the outer rear view mirror assembly LH/RH.

Mirror Select Switch

Selects the outer rear view mirror assembly whose mirror surface angle is to be adjusted.
