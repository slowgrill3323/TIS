Title: PARKING BRAKE: ELECTRIC PARKING BRAKE SYSTEM (for HEV Model): GENERAL; 2024 - 2026 MY Tacoma HV [03/2024 -        ]

GENERAL

OUTLINE

(a) An electric parking brake system is used.

(b) The electric parking brake uses actuators to operate the parking brake, making it possible to perform automatic linked operations by using information from other ECUs and improving usability.

(c) The electric parking brake has the following functions.

Manual Function

Locks and releases the parking brake according to driver switch operations.

Auto Function

Locks and releases the parking brake according to the shift position when this function is on.

Parking Brake Automatic Release Function

Detects accelerator pedal depression and releases the parking brake.

Brake Hold Function

Operates together with the brake hold function and locks the parking brake. For details, refer to the Brake Control System section.
