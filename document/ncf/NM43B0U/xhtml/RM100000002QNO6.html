Electrical Key Transmitter Sub-assembly

When a driver pushes the lock button, unlock button, tail gate button*1 or panic button on the key, the key outputs a request signal.

Electrical Key and TPMS Receiver Assembly

Receives a lock/unlock, tail gate button*1 or panic button operation signal from the electrical key transmitter sub-assembly and sends this information to the certification ECU.

Sounds to inform the driver when warning conditions are met.

ALL DOOR LOCK Relay

Inputs door lock request signals from the main body ECU and operates the door lock motor of all doors.

P/RL/RR DOOR UNLOCK Relay

Inputs front passenger, rear passenger RH and rear passenger LH door unlock request signals from the main body ECU and operates the door lock motor of front passenger, rear passenger RH and rear passenger LH doors.

D DOOR UNLOCK Relay

Inputs driver door unlock request signals from the main body ECU and operates the driver door lock motor.

  • Detects the vehicle condition using ECU and switch signals to control the entire door lock control system.

  • Outputs normal and reverse rotation signals to the tail gate lock motor.

FUNCTION

(a) The wireless door lock control system has the following functions:

Function*1

Outline

All Doors Lock

Pressing the lock button of the electrical key transmitter sub-assembly locks all doors and the tail gate.

All Doors Unlock (1-step Unlock)*2

Pressing the unlock button of the electrical key transmitter sub-assembly unlocks all doors and the tail gate.

All Doors Unlock (2-step Unlock)*2

Pressing the unlock button on the electrical key transmitter sub-assembly once unlocks the driver door, and pressing it again within 5 seconds unlocks all the doors.

Answer Back

When the doors are being locked or unlocked using the electrical key transmitter sub-assembly, the wireless door lock buzzer sounds and the hazard warning lights blink once while locking and twice while unlocking.

Panic Alarm

Pressing and holding the panic button of the electrical key transmitter sub-assembly for more than approximately 0.8 seconds causes the following to activate as an alarm:

  • Sounds the vehicle horn and security horn.
  • Flashes the hazard warning lights and exterior lights.
  • The map lights are illuminated (if the door-linked interior light switch is on).

Door Ajar Warning

If any door is open or ajar, pressing the lock button of the electrical key transmitter sub-assembly will cause the wireless door lock buzzer to sound for approximately 5 seconds.

Automatic Relock

If none of the doors are opened for 60 seconds after being unlocked by the wireless door lock control, all the doors are locked again automatically.

Repeat

If a door is not locked in response to the locking operation of the electrical key transmitter sub-assembly, the main body ECU will output a lock signal again after 1 second elapses.

Security

Sends a door lock/unlock operation request signal as a rolling code.

Customization

The on or off setting of some functions and settings can be performed using the GTS. In addition, some functions are not available depending on specification. For details, refer to the Repair Manual.

HINT:

*1: The default setting differs depending on the model.

*2: The function can be set to unlock all the doors after 1 or 2 wireless door lock control operations by using the customization function.

DIAGNOSIS

(a) If there is a malfunction in the system, the main body ECU stores the Diagnostic Trouble Codes (DTCs) in its memory. For details, refer to the Repair Manual.
