Title: DOOR LOCK: WIRELESS DOOR LOCK CONTROL SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [12/2023 -        ]

GENERAL

OUTLINE

(a) The wireless door lock control system is a convenient system for locking and unlocking all the doors from a distance.

(b) In this system, the electrical key and TPMS receiver assembly performs the code identification process and the main body ECU performs door lock control.

(c) An electrical key transmitter sub-assembly is used, and it contains up to a maximum of 4 buttons: lock button, unlock button, panic button and tail gate button*.

  • *: Models with power tail gate system

(d) When preparing a new electrical key transmitter sub-assembly to replace one that has been lost, it is necessary to register the recognition code. For details, refer to the Repair Manual.

(e) The customization function of the GTS or radio and display receiver assembly makes it possible to adjust and change the settings for each function.
