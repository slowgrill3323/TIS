COMBINATION METER

CONSTRUCTION

(a) "POWER ON" or "ACCESSORY" is displayed on the combination meter assembly depending on the current power source change condition. In addition, when the hybrid system can be started, "Key Certification Indicator" is displayed on the multi-information display. This enables, the user can check the current power source change condition and hybrid system startability condition.

(b) When a malfunction occurs in the smart key system, a message is displayed on the multi-information display in the combination meter assembly to inform the driver of a smart key system malfunction.
