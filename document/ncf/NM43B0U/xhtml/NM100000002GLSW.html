Title: ADVANCED DRIVER ASSISTANCE SYSTEM: DYNAMIC RADAR CRUISE CONTROL SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [12/2023 -        ]

GENERAL

OUTLINE

(a) In addition to constant speed control, dynamic radar cruise control is used which detects and judges the proceeding vehicle using the millimeter wave radar sensor assembly and forward recognition camera.

(b) The distance (time) setting used when following a preceding vehicle is set using the vehicle-to-vehicle distance control switch on the steering pad switch assembly and can be set to 4 levels.

(c) The dynamic radar cruise control operation status is displayed in the headup display* and multi-information display in the combination meter assembly.

  • *: Models with headup display

(d) Automatic brake control is performed when too close to the preceding vehicle.

(e) Instead of operating switches, a voice-operated function* allows the driver to specify a set vehicle speed and set distance using the voice control function.

HINT:

*May not be available depending on the vehicle.
