MULTI-DISPLAY

OPERATION

(a) RCTA Function Display

(1) A parking assist monitor system*1, multi-terrain monitor system*2 or panoramic view monitor system*3 cooperative display, which is linked with the RCTA function is used to display the RCTA icon while video is displayed by the parking assist monitor system*1, multi-terrain monitor system*2 or panoramic view monitor system*3.

(2) The direction of the approaching vehicle is displayed on the radio and display receiver assembly.

  • *1: Models with parking assist monitor system
  • *2: Models with multi-terrain monitor system
  • *3: Models with panoramic view monitor system
