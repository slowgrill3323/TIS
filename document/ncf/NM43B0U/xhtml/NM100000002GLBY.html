EXTERNAL DEVICE CONNECTION FUNCTION

FUNCTION

(a) "Apple CarPlay" Function (Models with "Apple CarPlay" Function)

(1) To utilize the "Apple CarPlay" function, it is necessary to connect a compatible iPhone to the audio head unit using a USB cable or "Wi-Fi" wireless connection. When the connection is detected and certified using the screen prompts, "Apple CarPlay" is displayed on the multi-display.

(2) Users can operate the iPhone on the multi-display, such as by displaying the iPhone's map on the multi-display by operating the "Apple CarPlay" screen and outputting registered music files from the vehicle speakers. For details, refer to https://www.apple.com/ios/carplay/

HINT:

  • "Apple CarPlay" are trademarks of Apple Inc., registered in the U.S. and other countries.
  • The "Apple CarPlay" function can be used by connecting an iPhone (iOS 13.3 or higher) supporting "Apple CarPlay" via USB cable. For details on supported iPhones models, refer to https://www.apple.com/ios/carplay/
  • For an audio head unit that supports 5.0 GHz "Wi-Fi", if an iPhone is connected via "Wi-Fi" in order to use "Apple CarPlay", the Bluetooth function of the connected iPhone will turn off. However, the Bluetooth functions of other Bluetooth devices will remain enabled. For an audio head unit that does not support 5.0 GHz "Wi-Fi", if an iPhone is connected via "Wi-Fi" in order to use "Apple CarPlay", all Bluetooth functions inside the vehicle off will turn off.
  • If an iPhone is connected to the audio head unit via USB cable in order to use "Apple CarPlay", all Bluetooth functions of the connected iPhone will turn off.
  • If a Bluetooth device is connected via Bluetooth and "Apple CarPlay" is used from another iPhone via USB cable connection, some Bluetooth functions of the Bluetooth device may be disabled. (What will be disabled will vary depending on whether the driver is registered)
  • As the "Apple CarPlay" screen displays the language setting of the iPhone, use the iPhone to change the set language.
  • Depending on the destination, the "Apple CarPlay" function may not be able to be used via the 2.4 GHz "Wi-Fi" wireless connection.

(b) "Android Auto" Function (Models with "Android Auto" Function)

(1) To utilize the "Android Auto" function, it is necessary to connect a compatible smartphone to the audio head unit using a USB cable or "Wi-Fi" wireless connection. When the connection is detected and certified using the screen prompts, "Android Auto" is displayed on the multi-display.

(2) Users can operate the smartphone on the multi-display, such as by displaying the smartphone's map on the multi-display by operating the "Android Auto" screen and outputting registered music files from the vehicle speakers. For details, refer to https://www.android.com/auto/

HINT:

  • "Android Auto" are trademarks of Google LLC., registered in the U.S. and other countries.
  • The "Android Auto" function can be used by connecting a smartphone (Android 6.0 (Lollipop) or higher) supporting "Android Auto" via USB cable. For details on supported smartphones models, refer to https://www.android.com/auto/
  • If a smartphone is connected via "Wi-Fi" in order to use "Android Auto", the Bluetooth function of the connected smartphone will turn off. However, the Bluetooth functions of other Bluetooth devices will remain enabled.
  • If a smartphone is connected to the audio head unit via USB cable in order to use "Android Auto", all Bluetooth functions of the connected smartphone will turn off.
  • If a Bluetooth device is connected via Bluetooth and "Android Auto" is used from another smartphone via USB cable connection, some Bluetooth functions of the Bluetooth device may be disabled. (What will be disabled will vary depending on whether the driver is registered)
  • As the "Android Auto" screen displays the language setting of the smartphone, use the smartphone to change the set language.
  • When a "Wi-Fi" wireless connection is used, the "Android Auto" function can be used via the 5.0 GHz connection only.
