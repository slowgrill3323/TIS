WIPER-LINKED LOW BEAM HEADLIGHT CONTROL

FUNCTION OF MAIN COMPONENTS

Component

Function

LED Driver Module RH (Headlight LED ECU RH)

Supplies power to the headlights.

LED Driver Module LH (Headlight LED ECU LH)

Power Distribution Box Assembly

Supplies power to the parking lights, front side marker lights, taillights, rear side marker lights and license plate lights.

Main Body ECU (Multiplex Network Body ECU)

Receives various signals and sends headlights, parking lights, side marker lights, taillights and license plate lights illumination signals.

Steering Sensor
  • Sends the light control switch position signals to the main body ECU.
  • Sends the front wiper switch position signals to the main body ECU.

Turn Signal Switch

Light Control Switch

Sends the light control switch position signal to the steering sensor.

Certification ECU (Smart Key ECU Assembly)

Sends ignition switch condition signals to the main body ECU.

Central Gateway ECU (Network Gateway ECU)

Relays the signal between the CAN communication lines.

Windshield Wiper Switch Assembly

Sends the front wiper switch position signals to the steering sensor.

SYSTEM CONTROL

(a) When all of the following conditions are met for 10 seconds or more, the wiper-linked low beam headlight control will turn the low beam headlights on:

  • The ignition switch is ON.
  • The light control switch is in the AUTO position.
  • The front wipers are operating. (Except MIST and washer-linked wiper operation)

FUNCTION

(a) Wiper-linked low beam headlight control system is designed to automatically illuminate the low beam headlights when it is raining to make the vehicle more visible to other vehicles.

(b) The wiper-linked low beam headlight control system can be turned on and off using the customization function. For details, refer to the Repair Manual.
