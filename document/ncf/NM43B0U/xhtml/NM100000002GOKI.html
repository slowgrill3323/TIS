MULTI-TERRAIN SELECT

SYSTEM CONTROL

(a) Multi-terrain select is a function that switches to the optimal control for the road conditions, including motive force control, brake control and steering control, when the driver performs a switch operation to select an appropriate mode for the road conditions. Optimal control for the road conditions is automatically performed, maintaining driving performance and achieving superior off-road driveability.

(b) In multi-terrain select, the mode can be selected from 6 modes: "MUD", "DIRT", "AUTO", "DEEP SNOW", "SAND" and "ROCK".

(c) A signal for the selected multi-terrain select mode is input into the ECM. The ECM sends the multi-terrain select mode signal to the skid control ECU and the ECUs of parts such as the combination meter via CAN communication.

(d) According to the multi-terrain select mode signal, the skid control ECU optimally controls the brake amount of the 4 wheels, the power steering ECU optimally performs steering control and the ECM optimally controls the motive force.

HINT:

The mode of multi-terrain select that can be selected varies according to the transfer drive mode.
