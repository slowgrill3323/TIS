CLUTCH START SYSTEM

SYSTEM CONTROL

(a) The clutch start system prevents the starter from cranking unless the clutch pedal is depressed. When the engine is started with the shift lever in a position other than neutral, it helps ensure safety by preventing the vehicle from moving unintentionally.

(b) A momentary type clutch start cancel switch assembly is provided in this system.

(c) When the clutch start cancel switch is turned on the clutch start system is canceled to allow engine cranking without depressing the clutch pedal.

(d) When all of the following conditions are met, the clutch start system is canceled.

  • The ignition switch is ON
  • The clutch start cancel switch and brake pedal are pressed simultaneously for 1.5 seconds

HINT:

  • When the clutch start system is canceled, the clutch start cancel switch indicator is illuminated and multi-information display displays graphics.
  • When the clutch start system is canceled, hill-start assist control brake function is disabled.

(e) When start the engine cranking in the clutch start system is canceled.

  • The vehicle must be stationary with brake pedal is pressed
  • The parking brake and brake hold system must be tured off
  • Simulationeously depress the ignition switch and brake pedal for 1.5 seconds to begin the engine clanking
  • When clanking starts, release the brake pedal
  • Continue to press ignition switch until the engine completely starts

(f) When any of the following condition is met, the clutch start system is activated.

  • When 1 minute has been elapsed after the clutch start cancel switch was pressed
  • When the clutch start cancel switch assembly is pressed again after activation
  • When the vehicle is turned off by the ignition switch
