BRAKE BOOSTER

CONSTRUCTION

(a) A single diaphragm type brake booster is used to achieve weight reduction.

(b) Tie-rods are used inside the brake booster to ensure rigidity during operation.

(c) A vacuum sensor assembly, which detects the brake booster vacuum pressure, are installed to the brake booster.*1

(d) A vacuum warning switch assemmbly, which detects the brake booster vacuum pressure, are installed to the brake booster.*2

HINT:

*1: Models with stop and start system

*2: Models without stop and start system

*A

Models with stop and start system

*B

Models without stop and start system
