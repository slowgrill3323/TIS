KEY

CONSTRUCTION

(a) Electrical Key Transmitter Sub-assembly

(1) The electrical key transmitter sub-assembly consists of a mechanical key, transmitter for the wireless door lock control, transceiver for the entry function, and transponder chip for the immobiliser function.

(2) This mechanical key works for the driver's door, but cannot be used to start the hybrid system.

(3) The transmitter for the wireless door lock control has a lock button, unlock button, tail gate button* and panic button.

  • *: Models with power tail gate system

(4) The transceiver for the entry function receives the signals from the antennas and returns the ID code of the electrical key transmitter sub-assembly to the electrical key and TPMS receiver assembly.

(5) The transponder chip for the immobiliser function returns a signal to the power switch as a response to the radio wave signals received from the power switch.

(6) An indicator light that can be used to check the operation condition and battery condition is provided.

(b) Electrical Key (Card Key) Transmitter Sub-assembly (Models with Card Key)

(1) The electrical key (card key) transmitter sub-assembly consists of a mechanical key and a transceiver for the smart key system.

(2) The transceiver for the smart key system receives the signals from the electrical key antennas and returns the ID code to the electrical key and TPMS receiver assembly.

(3) The transponder chip for the immobiliser system returns a signal to the power switch as a response to the radio waves received from the power switch.

(4) This mechanical key works for the driver door, but cannot be used to start the hybrid system.

OPERATION

(a) Electrical Key Transmitter Sub-assembly

(1) Auto Power Off

  1. When any button of the electrical key transmitter sub-assembly is pressed and held, it stops sending electrical waves after approximately 20 seconds to prevent battery depletion.

(2) Indicator Light

  1. The indicator light of the electrical key transmitter sub-assembly illuminates while communication is in progress during ID certification and during button operations so that both the operation condition and battery condition can be checked.
