Title: THEFT DETERRENT / KEYLESS ENTRY: SMART KEY SYSTEM (for Gasoline Model): GENERAL; 2024 - 2026 MY Tacoma [12/2023 -        ]

GENERAL

OUTLINE

(a) The smart key system has the following main functions, making it a very user-friendly system:

Function

Outline

Entry Function

Entry Unlock

The doors are unlocked by grabbing the door outside handle or pressing the tail gate opener switch*1 while carrying the electrical key transmitter sub-assembly.

  • The driver door outside handle unlocks the driver door. (Driver Door Unlock Mode)
  • The driver door outside handle unlocks all doors. (All Door Unlock Mode)
  • The front passenger door outside handle unlocks all doors.*2
  • The tail gate opener switch unlocks all doors.*1

Entry Lock

All doors are locked by touching the lock sensor of the door outside handle or pressing the lock switch of the tail gate*1 while carrying the electrical key transmitter sub-assembly.

Entry Tail Gate Open*1

The tail gate is opened by pressing the elbow switch (luggage electrical key switch) and the tail gate opener switch while carrying the electrical key transmitter sub-assembly.

Wireless Door Lock Control

The wireless door lock control function is a convenient system for locking and unlocking all the doors or opening the tail gate button*1 from a distance. The basic operation is the same as that of the wireless door lock control system.

Start Function

The start function is operated by simply pressing the engine switch while carrying the electrical key transmitter sub-assembly. When the engine switch is pressed, the certification ECU turns on/off the ACC relay, IGP relay and IGR relay to switch the power source pattern.*3

  • The power source mode changes depending on the depression of the clutch pedal*4 or brake pedal*5 and the shift lever position*5.

Immobiliser

The immobiliser function compares the ID code registered in the ID code box with the ID code of the transponder chip embedded in the electrical key transmitter sub-assembly.

Steering Lock*4

The steering lock function is operated to lock and unlock the steering wheel by using a lock/unlock motor in the steering column assembly.

HINT:

*1: Models with power tail gate system

*2: Models with front passenger side door electrical key antenna

*3: In this chapter, the expression "power source" has been used in some locations to allow precise explanations. The power source condition is also expressed using conventional expressions such as "ignition switch off", "ignition switch on (ACC)" and "ignition switch on (IG)".

*4: Models with manual transmission

*5: Models with automatic transmission
