(a) Component Operation Output Function

(1) This function turns the corresponding component on and off in accordance with operation signals or switch input signals from the ECU.

(b) Semiconductor Fuse Function

(1) When an abnormal electric current occurs during operation of components, the components are turned off by shutting off the current before electrical wire emits smoke.

(2) The electrical current is maintained shut off while operation signals or switch signals are on.

(3) When operation signals or switch signals are turned off, the shut off condition is canceled.

(c) Multiplex Communication Function

(1) Operation Signal Communication

  • This communication uses the multiplex communication line (CXPI communication line) to input operation signals or switch signals from the ECU.

(2) Diagnosis Communication

  • Checking Diagnostic Trouble Codes (DTCs) regarding ECU and other malfunctions is possible by connecting the GTS.
  • The semiconductor fuse condition (shut off or not) can be confirmed by connecting the GTS.

(3) Reprogramming

  • Reprogramming can be performed by connecting the GTS.

HINT:

Refer to the Repair Manual for the details of the diagnostic communication and reprogramming.

DIAGNOSIS

(a) In order to make system inspections easier to perform, a diagnosis function is used in consideration of serviceability. The Diagnostic Trouble Codes (DTCs) of malfunctions can be read by connecting the GTS. For details, refer to the Repair Manual.
