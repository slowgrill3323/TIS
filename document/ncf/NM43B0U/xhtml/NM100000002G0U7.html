Title: DOOR / HATCH: POWER TAIL GATE SYSTEM: GENERAL; 2024 - 2026 MY Tacoma Tacoma HV [12/2023 -        ]

GENERAL

OUTLINE

(a) The power tail gate system enables the tail gate to be opened and closed automatically by a motor in accordance with signals from the power tail gate control switch (tail gate opener switch), tail gate handle switch (tail gate opener switch assembly), elbow switch (luggage electrical key switch) or power tail gate button on an electrical key transmitter sub-assembly.

(b) The hands free power tail gate function which enables the tail gate to be closed by lifting it about 20 mm (0.8 in.).

(c) This system is controlled by the power tail gate ECU (multiplex network door ECU).

PRECAUTION

(a) Battery saver

  • When power tail gate is left open more than 8 hours, power tail gate ECU (multiplex network door ECU) goes to sleep.
  • If the power tail gate ECU (multiplex network door ECU) is asleep, then when you enter the detection area holding the electrical key transmitter sub-assembly, release the tail gate lock or open one of the doors, the power tail gate ECU (multiplex network door ECU) restarts.
  • Close assist does not operate when the power tail gate ECU (multiplex network door ECU) is asleep.

(b) Before operating the power tail gate system, check your surrounding for anything that could contact the tail gate.

(c) A load of 45 kg (99.2 lb) or more on the tail gate may cause the tail gate to open faster than the normal operation.

(d) If luggage is placed on or is pressing against the tail gate, then the tail gate may open faster than normal operation.
