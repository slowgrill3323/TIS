SECURITY INDICATOR LIGHT

CONSTRUCTION

(a) An LED is used as a security indicator light, and it is positioned in the roof console box assembly.

(b) The security indicator light is controlled by the certification ECU and displays the theft deterrent system control status by turning off, illuminating and blinking.

OPERATION

(a) When the theft deterrent system or immobiliser system is set, the security indicator light starts blinking at approximately 2-second intervals (approximately 0.125 seconds on/approximately 1.875 seconds off). Furthermore, when the theft deterrent system is in the armed state or alarm sounding state, the security indicator light flashes.
