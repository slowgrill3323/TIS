PRECAUTIONS FOR BLIND SPOT MONITOR

INSPECTION PROCEDURES FOR THE BLIND SPOT MONITOR SENSOR BRACKET AND INSTALLATION AREA

(a) Refer to the TOYOTA Repair Manual for blind spot monitor sensor adjustment procedure.

HINT:

Audio / Visual / Telematics > Park assist / Monitoring > Blind Spot Monitor Sensor Adjustment

(b) If you have replaced the bracket installation panel or blind spot monitor bracket, let alone repaired the bracket installation panel, the sensor installation angle must be checked since it may become misaligned.

(c) Be sure to always perform a functional inspection after any body repair. (This is performed as a comprehensive inspection with the bumper installed.)
